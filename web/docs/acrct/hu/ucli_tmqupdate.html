A WebSphere MQ frissítése Java üzenetszolgáltatás szolgáltatóként és a JMS erőforrás beállítások alkalmazásügyfelekhez

Frissítheti a WebSphere MQ egy meglévő beállítását, mint Java üzenetszolgáltatás (JMS) szolgáltatót, és a WebSphere MQ kapcsolatgyárak vagy WebSphere MQ címzettek beállítását is.

A feladatról

Ezzel a feladattal frissítheti a WebSphere MQ egy meglévő beállítását, mint Java üzenetszolgáltatás (JMS) szolgáltatót, és a WebSphere MQ kapcsolatgyárak vagy WebSphere MQ címzettek beállítását is.

Eljárás

  1. Indítsa el az Alkalmazásügyfél erőforrás beállítási eszközt (ACRCT).
  2. Nyissa meg azt a vállalati archívum (EAR) fájlt, ami tartalmazza a WebSphere MQ JMS szolgáltatót, WebSphere MQ kapcsolatgyárat, vagy a WebSphere MQ címzettet. Az EAR fájl tartalma megjelenik a navigációs fa nézetben látható.
  3. Válassza ki a Java archívum (JAR) fájlt, ami tartalmazza a JMS szolgáltatót, kapcsolatgyárat, vagy a frissítendő címzettet.
  4. Bontsa ki a JAR fájlt a tartalmának megtekintéséhez, amíg nem találja meg az adott JMS szolgáltatót, kapcsolatgyárat, vagy a frissíteni kívánt címzettet. Vigye véghez az alábbi műveletek egyikét:
    • Kattintson a jobb egérgombbal a megfelelő objektumra, majd kattintson a Tulajdonságok menüpontjára.
    • A menüsoron kattintson a Szerkesztés > Tulajdonságok elemre.
  5. A tulajdonságok frissítése a megjelenített mezőkben. Részletes mező súgóhoz lássa:
    • JMS szolgáltató tulajdonságok
    • MQ sor kapcsolatgyár tulajdonságok
    • MQ témakör kapcsolatgyár tulajdonságok
    • MQ sor címzett tulajdonságok
    • MQ témakör címzett tulajdonságok
  6. Kattintson az OK gombra.
  7. Kattintson a Fájl > Mentés menüpontra a módosítások mentéséhez.


A jelzett hivatkozások (online) Internet-hozzáférést igényelnek.
