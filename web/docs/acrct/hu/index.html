A JMS felület egy közös módszert biztosít Java programoknak (ügyfeleknek és Java Enterprise Edition (Java EE) alkalmazásoknak) aszinkron kérések JMS üzenetként létrehozásához, küldéséhez, fogadásához és olvasásához.
