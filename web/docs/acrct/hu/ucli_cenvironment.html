Erőforrás környezet szolgáltatók és erőforrás környezet bejegyzések

Egy erőforrás környezet hivatkozás egy logikai nevet képez le az ügyfélalkalmazás által az objektum fizikai nevére.

Nem minden kiszolgáló JNDI névtartományhoz kötött objektum lett az alkalmazásügyfél használatára tervezve. Például, a WebSphere Application Server ügyfél futási környezet nem támogatja a Java 2 kapcsolódó (J2C) objektumokat az ügyfélnél. Az objektumnak távolról vezérelhetőnek kell lennie, és az ügyfél oldali megvalósításoknak elérhetőknek kell lenniük az alkalmazás futási környezet osztályútvonalán.

Az erőforrás környezeti hivatkozások eltérnek az erőforrás hivatkozásoktól. Erőforrás környezeti hivatkozások lehetővé teszik, hogy az alkalmazásügyfél logikai nevet használjon a kiszolgáló JNDI névtartományhoz kötött erőforrás kikereséséhez. Egy erőforrás hivatkozás lehetővé teszi, hogy az alkalmazás egy logikai nevet használjon egy helyi J2EE erőforrás kikereséséhez. A J2EE specifikáció nem határozza meg egy erőforrás adott megvalósítását.



A jelzett hivatkozások (online) Internet-hozzáférést igényelnek.
