Alapértelmezett szolgáltató témakör címzett beállításai

A panelen megtekintheti vagy módosíthatja a kiválasztott JMS témakörcél beállítási tulajdonságait, amit a belső termék Java üzenetkezelési szolgáltatás (JMS) szolgáltatóval lehet használni, ami WebSphere alkalmazáskiszolgálóval kerül telepítésre.

Az alábbi Alkalmazásügyfél erőforrás beállítási eszköz (ACRCT) oldal megtekintéséhez kattintson a Fájl > Megnyitás elemre. Miután keresett egy EAR fájlt, kattintson a Megnyitás gombra. Bontsa ki a kiválasztott JAR fájl > Üzenetkezelési szolgáltatók > Alapértelmezett szolgáltató ágat. Kattintson a jobb egérgombbal a Témakör címzettekre, majd válassza az előugró menü Új menüpontját. A következő mezők jelennek meg az Általános lapon.
