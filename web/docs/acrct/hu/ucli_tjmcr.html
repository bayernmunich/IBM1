Java üzenetkezelési ügyfél erőforrások beállítása

Java üzenetkezelési ügyfél erőforrások konfigurálásához, új JMS szolgáltató beállításokat hoz létre az alkalmazásügyfélhez. Az alkalmazásügyfél használhat egy üzenetkezelési szolgáltatást a Java üzenetszolgáltatási alkalmazás programozási felületeken keresztül. A JMS szolgáltató kétfajta J2EE gyárat biztosít. Az egyik egy JMS kapcsolatgyár, és a másik egy JMS címzettgyár.

Kezdeti lépések

Egy külön adminisztrációs feladatban, telepítse a Java üzenetszolgáltatás (JMS) ügyfelet azon az ügyfél számítógépen, ahol az alkalmazásügyfél található. Az üzenetkezelési termék szállítójának biztosítania kell a JMS ügyfél megvalósítását. További információkért tekintse meg az üzenetkezelési termék dokumentációját.

Megjegyzés: Amikor befejezi ezt a feladatot létrehozhat egy új üzenetkezelési szolgáltatót vagy használhat egy már létezőt.

Eljárás

  1. Indítsa el az Alkalmazásügyfél erőforrás beállítási eszközt (ACRCT).
  2. Nyissa ki azt az EAR fájlt, amelyhez új JMS szolgáltatót szeretne beállítani. Az EAR fájl tartalma a megjelenő fa nézetben látható.
  3. Válassza ki a fából azt a JAR fájlt, amiben be szeretné állítani az új JMS szolgáltatót.
  4. Bontsa ki a JAR fájlt a tartalma megtekintéséhez.
  5. (Nem kötelező) Ha új üzenetkezelési szolgáltatót kíván létrehozni, kattintson a jobb egérgombbal az Üzenetkezelési szolgáltatók elemre, majd válassza az előugró menü Új menüpontját.
  6. Állítsa be a JMS szolgáltató tulajdonságait az eredményül kapott tulajdonság párbeszédablakban.
  7. Kattintson az OK gombra.
  8. Kattintson a Fájl > Mentés menüpontra.
