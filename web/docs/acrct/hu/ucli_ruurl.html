URL beállítások alkalmazásügyfelekhez

Ezen oldal használatával egy bizonyos URL protokoll funkcióját valósítja meg, mint a Hiperszöveg átviteli protokoll (HTTP).

Az alábbi Alkalmazásügyfél erőforrás beállítási eszköz (ACRCT) oldal megtekintéséhez kattintson a Fájl > Megnyitás elemre. Miután keresett egy EAR fájlt, kattintson a Megnyitás gombra. Bontsa ki a kiválasztott JAR fájl > URL szolgáltatók > URL szolgáltató példány ágat. Kattintson a jobb egérgombbal az URL címekre, majd válassza az előugró menü Új menüpontját. A következő mezők jelennek meg az Általános lapon.

A szolgáltató osztályokból áll és a java.net.URLStreamHandler és java.net.URLConnection osztályokat terjeszti ki.

Név

Adminisztrációs név az URL címhez.

Leírás

Megadja az URL elhagyható leírását az adminisztrációs rekordok számára.

JNDI név

Az alkalmazásügyfél futás közben használja ezt a mezőt beállítási információk lekéréséhez.

URL

Egy egységes erőforrás-mutató (URL) név, amely egy Internet vagy intranet erőforrásra mutat. Például: http://www.ibm.com.

Egyéni tulajdonságok

Név-érték párokat határoz meg további tulajdonságok beállításához azon az objektumon, ami futás közben jön létre ehhez az erőforráshoz.

Meg kell adnia egy nevet, ami nyilvános tulajdonsága az objektumnak és egy értéket, ami átváltható karaktersorozatból a tulajdonság beállító metódusához szükséges típusba. Az elfogadható tulajdonságok és értékek a létrehozott objektumtól függnek. Forduljon az objektum dokumentációjához az érvényes tulajdonságok és értékek listájához.
