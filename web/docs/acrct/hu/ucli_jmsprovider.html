Java üzenetszolgáltatás szolgáltatók ügyeleknek

Ez a témakör bemutatja, hogy milyen módon használhatják az ügyfélalkalmazások a Java üzenetszolgáltatás (JMS) szolgáltatókat a WebSphere Application Server programmal.

Az IBM WebSphere Application Server támogatja az aszinkron üzenetkezelést egy JMS szolgáltató és a hozzá kapcsolódó üzenetkezelési rendszer használatán keresztül. JMS szolgáltatóknak meg kell felelniük a JMS 1.1 változat specifikációnak. Az üzenetvezérelt komponensek használatához a JMS szolgáltatónak támogatnia kell a specifikációban meghatározott, választható Alkalmazáskiszolgáló szolgáltatás (ASF) funkciót, vagy a JCA 1.5 változatú specifikációban meghatározott bejövő erőforrás csatolót.

Az IBM WebSphere Application Server szolgáltatás integrációs technológiái működhetnek üzenetkezelési rendszerként, ha beállított egy szolgáltatás integrációs buszt, ami az alapértelmezett üzenetkezelési szolgáltatón keresztül érhető el. Ez a támogatás a WebSphere Application Server részeként kerül telepítésre és az adminisztrációs konzolon keresztül felügyelhető, valamint teljesen integrált a WebSphere Application Server futási környezetbe.

A WebSphere Application Server a következő JMS szolgáltatókat is támogatja:
WebSphere MQ
A WebSphere MQ támogatott változataival használható.
Általános
A 3. fél üzenetkezelési rendszer használatához, ami támogatja az ASF szolgáltatást.

A korábbi kiadásokkal visszamenőleges kompatibilitás miatt a WebSphere Application Server szintén tartalmazza a V5 alapértelmezett üzenetkezelési szolgáltató támogatását is, ami lehetővé teszi, hogy erőforrásokat állítson be a WebSphere Application Server 5. változat beépített üzenetkezelési rendszerrel. A V5 alapértelmezett üzenetkezelési szolgáltató használható a szolgáltatás integrációs busszal.

A WebSphere alkalmazások használhatják a JMS szolgáltatók által biztosított üzenetkezelési erőforrásokat. Mindemellett a szolgáltató választását leggyakrabban a meglévő üzenetkezelési rendszer használata vagy az integráció szükséglete írja elő. Például lehet hogy már rendelkezik egy WebSphere MQ alapú üzenetkezelési infrastruktúrával. Ebben az esetben vagy közvetlenül csatlakozik a tartalmazott WebSphere MQ (mint JMS szolgáltató) támogatással, vagy állítson be egy szolgáltatás integrációs buszt hivatkozásokkal egy WebSphere MQ hálózathoz, majd érje el a buszt az alapértelmezett üzenetkezelési szolgáltatóval.

A szolgáltatás integráció busz hozzáférést biztosít egy alapértelmezett üzenetkezelési szolgáltatóhoz is. Ez egy J2EE 1.4 szabványnak megfelelő üzenetkezelési szolgáltató, amely teljesen integrált a WebSphere Application Server termékben. Ezt az üzenetkezelési együttműködésre használhatja egy WebSphere MQ hálózattal több kiszolgálóból álló konfigurációkban.



A jelzett hivatkozások (online) Internet-hozzáférést igényelnek.
