管理対象オブジェクトの設定

このパネルを使用して、選択した管理対象オブジェクトの構成プロパティーを表示または変更します。

この「アプリケーション・クライアント・リソース構成ツール (ACRCT)」ページを 表示するには、「ファイル」>「オープン」とクリックします。 EAR ファイルのブラウズ後、「オープン」をクリックします。 選択した JAR ファイル >「リソース・アダプター」>「resource_adapter_instance」と展開します。「管理対象オブジェクト」を右マウス・ボタン・クリックして、「新規」をクリックします。 「一般」タブ上に以下のフィールドが表示されます。

管理オブジェクトの設定は接続ファクトリーと同様に処理します。 管理対象オブジェクトを更新する場合は、管理対象オブジェクトの作成時に使用したパネルを使用します。

名前

IBM® WebSphere® Application Server 内でこの管理対象オブジェクトを管理目的で使用する際の名前。この名前は、製品管理可能ドメイン全体のリソース・アダプター管理対象オブジェクト内で固有である必要があります。

データ型 ストリング
説明

IBM WebSphere Application Server 内で管理目的で使用する、この接続ファクトリーのオプションの説明。

データ型 ストリング
JNDI 名

この項目は、resource-env-ref 名、message-destination-ref 名 (message-destination-ref がリンクを持たない場合)、または message-destination リンクです。

データ型 ストリング
タイプ

リソース・アダプター・アーカイブ (RAR) ファイル内の管理対象オブジェクトに対して定義されている、 すべての管理対象オブジェクト・クラスとインターフェースのペアのドロップダウン・リスト。

各「タイプ」には、「プロパティー」ボックスで指定されている一連のプロパティーがあります。 この一連のプロパティーは、各管理対象オブジェクト定義からプロパティーを取り出して構成されています。 更新用に表示されている既存の管理対象オブジェクトの場合、このプロパティーのリストは、オブジェクトに対して指定されているプロパティーでオーバーレイされます。 「タイプ」フィールドを変更すると、そのタイプの適切なプロパティーに合わせてプロパティーも変更されます。
