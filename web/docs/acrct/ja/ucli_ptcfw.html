アプリケーション・クライアントの V5 デフォルト・メッセージング・プロバイダーのトピック接続ファクトリー設定

このパネルを使用して、内部製品 Java™ Message Service (JMS) プロバイダーで使用するために選択されたトピック接続ファクトリーの構成プロパティーを表示または変更します。この構成プロパティーは、JMS プロバイダーとその JMS プロバイダーが使用するメッセージング・システム間で接続が作成される方法を制御します。

この「アプリケーション・クライアント・リソース構成ツール (ACRCT)」ページを 表示するには、「ファイル」>「オープン」とクリックします。 EAR ファイルのブラウズ後、「オープン」をクリックします。 選択した JAR ファイル >「Messaging Providers」>「Version 5 Default Provider」と展開します。「Topic Connection Factories」を右マウス・ボタン・クリックして、「新規」をクリックします。 「一般」タブ上に以下のフィールドが表示されます。

V5 デフォルト・メッセージング・プロバイダーのトピック接続ファクトリーには、以下のプロパティーがあります。

名前

WebSphere® Application Server 内で、このキュー接続ファクトリーを管理目的で識別する際の名前。 この名前は、WebSphere Application Server 管理ドメイン全体の JMS 接続ファクトリー内で固有である必要があります。

データ型 ストリング
説明

WebSphere Application Server 内で管理目的で使用する、このトピック接続ファクトリーの説明。

データ型 ストリング
JNDI 名

アプリケーション・クライアント・ランタイムは、 このフィールドを使用して構成情報を検索します。

ユーザー ID

呼び出し側のアプリケーションがユーザー ID とパスワードを明示的に指定していない場合に、認証のために「パスワード」プロパティーとともに使用されるユーザー ID です。

「ユーザー ID」プロパティーに値を指定する場合は、「パスワード」プロパティーにも値を指定する必要があります。

接続ファクトリーの「ユーザー ID」プロパティーおよび「パスワード」プロパティーは、 呼び出し側のアプリケーションにユーザー ID およびパスワード が明示的に指定されていない場合 (呼び出し側のアプリケーションがメソッド createTopicConnection() を使用する場合など) に使用されます。 JMS クライアントは、ユーザー ID とパスワードを JMS サーバーに渡します。

データ型 ストリング
パスワード

呼び出し側のアプリケーションがユーザー ID とパスワードを明示的に提供していない場合、 認証のために「ユーザー ID」プロパティーとともにこのパスワードを使用します。

「ユーザー ID」プロパティーに値を指定する場合は、「パスワード」プロパティーにも値を指定する必要があります。

データ型 ストリング
再入力パスワード

パスワードを確認します。

Node

JMS サーバーがこの接続ファクトリー用に実行される管理ノードの WebSphere Application Server ノード名。このファクトリーによって作成された接続は、その JMS サーバーに接続します。

データ型 列挙型
範囲 WebSphere Application Server 管理可能ドメイン内のノードのプルダウン・リスト。
アプリケーション・サーバー

アプリケーション・サーバーの名前を入力します。 この名前は、マシンのホスト名ではなく構成済みアプリケーション・サーバーの名前です。

ポート

JMS サーバーに接続するために接続が使用する、2 つのポートのうちのいずれかを指定します。QUEUED ポートは、全機能 JMS Publish/Subscribe サポートに使用し、DIRECT ポートは非持続的かつ非トランザクションで、非永続的サブスクリプション専用です。

注: メッセージ駆動型 Bean は、Publish または Subscribe サポートに直接リスナー・ポートを使用できません。 したがって、ポートを「Direct」に設定して構成した トピック接続ファクトリーは、メッセージ駆動型 Bean で使用することはできません。
データ型 列挙型
デフォルト QUEUED
範囲
QUEUED
全機能 JMS 準拠の Publish または Subscribe サポートに使用するリスナー・ポート。
DIRECT
Publish または Subscribe サポートの直接 TCP/IP 接続 (非トランザクション、 非持続的、および非永続的サブスクリプション専用) に使用するリスナー・ポート。

これらのポートの TCP/IP ポート番号は、製品内部の JMS サーバーに定義されています。
クライアント ID

MQSeries® キュー・マネージャーへの接続に使用される JMS クライアント ID。

データ型 ストリング
カスタム・プロパティー

このリソースの実行時に作成されるオブジェクトに関する追加のプロパティーを設定するための、 名前と値のペアを指定します。

このオブジェクトのパブリック・プロパティーの名前、およびストリングから このプロパティーのセット・メソッドが必要とするタイプに変換できる値を入力する必要が あります。受け入れ可能なプロパティーと値は作成されるオブジェクトによって 異なります。有効なプロパティーと値のリストについては、 オブジェクトの資料を参照してください。
