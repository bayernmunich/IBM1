Konfigurowanie nowych dostawców środowiska zasobów dla klientów aplikacji

Za pomocą narzędzia ACRCT (Application Client Resource Configuration Tool) można tworzyć nowe konfiguracje dostawy środowiska zasobów dla klienta aplikacji.

Czynności wstępne

To zadanie umożliwia utworzenie nowych konfiguracji dostawcy środowiska zasobów dla klienta aplikacji.

Informacje dotyczące tego tematu

Aby skonfigurować nowego dostawcę środowiska zasobów, wykonaj następujące kroki:

Procedura

  1. Uruchom Narzędzie konfiguracji zasobów aplikacji i otwórz plik EAR, dla którego ma zostać skonfigurowany nowy dostawca JMS (Java Message Service). Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  2. Wybierz z drzewa plik JAR, w którym zostanie skonfigurowany nowy dostawca JMS.
  3. Rozwiń plik JAR, aby wyświetlić jego zawartość.
  4. Kliknij folder Dostawcy środowiska zasobów. Wykonaj jedną z poniższych czynności:
    • Kliknij prawym przyciskiem myszy folder dostawcy, a następnie kliknij opcję Nowy.
    • Kliknij opcję Edytuj > Nowy na pasku menu.
  5. Skonfiguruj właściwości dostawcy JMS w wyświetlonych polach.
  6. Po zakończeniu kliknij przycisk OK.
  7. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.
