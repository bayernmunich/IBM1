Konfigurowanie dostawców poczty i sesji dla klientów aplikacji

Za pomocą narzędzia ACRCT (Application Client Resource Configuration Tool) można edytować konfiguracje sesji poczty i dostawców dla klientów aplikacji.

Informacje dotyczące tego tematu

Narzędzie ACRCT (Application Client Resource Configuration Tool) służy do edytowania konfiguracji sesji poczty i dostawców dla klientów aplikacji.

Procedura

  1. Uruchom narzędzie ACRCT.
  2. Otwórz plik EAR.
  3. Znajdź obiekty poczty w drzewie wyświetlanym dla pliku EAR. Jeśli na przykład plik zawiera sesje poczty, rozwiń menu Zasoby > aplikacja.jar > Dostawcy poczty > instancja_dostawcy_poczty_java > Sesje poczty.

    W tym przykładzie instancja_dostawcy_poczty_java jest konkretnym dostawcą poczty.

Wyniki

Instancje sesji poczty elektronicznej znajdują się w folderze Sesje JavaMail.
