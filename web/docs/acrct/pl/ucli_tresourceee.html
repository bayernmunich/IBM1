Konfigurowanie nowych pozycji środowiska zasobów dla klientów aplikacji

Za pomocą narzędzia ACRCT (Application Client Resource Configuration Tool) można utworzyć nowe pozycje środowiska zasobów dla aplikacji klienckiej.

Czynności wstępne

To zadanie umożliwia utworzenie nowych pozycji środowiska zasobów dla aplikacji klienta.

Informacje dotyczące tego tematu

Procedura

  1. Uruchom narzędzie ACRCT (Application Client Resource Configuration Tool).
  2. Otwórz plik EAR, dla którego zostanie skonfigurowana nowa pozycja środowiska zasobów. Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  3. Kliknij żądanego dostawcę środowiska zasobów i wykonaj następującą operację, aby skonfigurować nowych dostawców:
    • Skonfiguruj nowego dostawcę środowiska zasobów.
  4. Rozwiń dostawcę środowiska zasobów, aby wyświetlić folder Pozycje środowiska zasobów.
  5. Kliknij folder pozycji środowiska zasobów i wykonaj jedną z następujących czynności:
    • Kliknij folder prawym przyciskiem myszy, a następnie wybierz opcję Nowy.
    • Kliknij opcję Edytuj > Nowy na pasku menu.
  6. Skonfiguruj właściwości pozycji środowiska zasobów w wyświetlonych polach.
  7. Kliknij przycisk OK.
  8. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.
