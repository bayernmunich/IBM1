Ustawienia miejsca docelowego kolejki domyślnego dostawcy

Ten panel umożliwia wyświetlanie i zmianę właściwości konfiguracyjnych wybranego miejsca docelowego kolejki JMS, które ma być używane z wewnętrznym dostawcą JMS (Java Message Service) produktu, instalowanym wraz z serwerem WebSphere Application Server.

Aby wyświetlić tę stronę narzędzia ACRCT (Application Client Resource Configuration Tool), należy kliknąć opcję Plik > Otwórz. Po wyszukaniu pliku EAR należy kliknąć przycisk Otwórz. Rozwiń wybrany plik JAR > Dostawcy przesyłania komunikatów > Domyślny dostawca. Kliknij prawym przyciskiem myszy folder Miejsca docelowe kolejki. Kliknij przycisk Nowy. Na karcie Ogólne zostaną wyświetlone następujące pola.

Nazwa

Nazwa fabryki miejsc docelowych kolejki. Wypełnienie tego pola jest obowiązkowe.

Typ danych String
Opis

Opis tego miejsca docelowego kolejki w celach administracyjnych na serwerze IBM WebSphere Application Server.

Typ danych String
Nazwa JNDI

Nazwa JNDI umożliwiająca dopasowanie tej definicji do nazwy resource-env-ref deskryptora wdrażania.

Typ danych String
Nazwa kolejki

Nazwa kolejki.

Typ danych String
Tryb dostarczania

Tryb dostarczania komunikatów wysłanych do tego miejsca docelowego.

Typ danych String
Zakres Application, Trwałe lub Nietrwałe
Wartość domyślna Aplikacja
Czas życia

Domyślny czas przechowywania przez system komunikatu wysłanego do tego miejsca docelowego (liczony od momentu jego wysłania), gdzie wartość 0 oznacza nieskończony czas życia. Jeśli pole Czas życia nie jest wypełnione, użyta zostaje wartość producenta.

Typ danych Integer
Jednostki Milisekundy
Priorytet

Priorytet wysyłania komunikatów do tego miejsca docelowego. Jeśli pole to nie jest wypełnione, użyta zostaje wartość producenta.

Typ danych Integer
Zakres Od 0 do 9, gdzie 0 oznacza najniższy priorytet, a 9 najwyższy
Odczyt z wyprzedzeniem

Służy do sterowania optymalizacją odczytu z wyprzedzeniem podczas dostarczania komunikatu.

Typ danych String
Zakres Jako połączenie, Zawsze włączony i Zawsze wyłączony
Wartość domyślna Jako połączenie
