Ustawienia dostawcy źródeł danych dla klientów aplikacji

Ta strona umożliwia utworzenie źródła danych w ramach dostawcy JDBC, który dostarcza określoną klasę implementacji sterownika JDBC.

Aby wyświetlić tę stronę narzędzia ACRCT (Application Client Resource Configuration Tool), należy kliknąć opcję Plik > Otwórz. Po wyszukaniu pliku EAR należy kliknąć przycisk Otwórz. Rozwiń plik JAR. Kliknij prawym przyciskiem myszy folder Dostawcy źródeł danych >, a następnie kliknij polecenie Nowy. Na karcie Ogólne pojawią się poniższe pola:

Nazwa

Określa wyświetlaną nazwę źródła danych.

W polu tym można na przykład wprowadzić nazwę Testowe źródło danych.

Typ danych String
Opis

Określa opis tekstowy zasobu.

Typ danych String
Ścieżka klasy

Lista ścieżek lub nazw plików .jar, które łącznie tworzą położenie klas dostawcy zasobów.

Klasa implementacji

To ustawienie umożliwia wykonywanie funkcji specyficznych dla bazy danych.

Typ danych String
Wartość domyślna W zależności od klasy implementacji sterownika JDBC
Właściwości niestandardowe

Pary nazwa-wartość do ustawiania dodatkowych właściwości obiektu, który jest tworzony w czasie wykonywania dla tego zasobu.

Należy wprowadzić nazwę będącą właściwością publiczną obiektu oraz wartość, która może być przekształcona z łańcucha tekstowego w typ wymagany przez metodę ustawioną dla tej właściwości. Dopuszczalne właściwości oraz wartości zależą od obiektu, który jest tworzony. Wykaz poprawnych wartości można znaleźć w dokumentacji obiektu.
