Konfigurowanie nowych fabryk połączeń dla adapterów zasobów

Do konfigurowania nowych fabryk połączeń dla adapterów zasobów służy narzędzie Application Client Resource Configuration Tool (ACRCT).

Informacje dotyczące tego tematu

Wykonaj poniższe czynności, aby skonfigurować nowe fabryki połączeń dla adapterów zasobów.

Procedura

  1. Uruchom narzędzie ACRCT (Application Client Resource Configuration Tool).
  2. Otwórz plik EAR, dla którego chcesz skonfigurować nowe fabryki połączeń. Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  3. Zaznacz w widoku drzewa plik JAR, w którym chcesz skonfigurować nowe fabryki połączeń.
  4. Rozwiń plik JAR, aby wyświetlić jego zawartość.
  5. Kliknij folder Adaptery zasobów.
  6. Rozwiń adapter zasobów, dla którego chcesz utworzyć fabryki połączeń.
  7. Kliknij prawym przyciskiem myszy folder Fabryki połączeń, a następnie kliknij polecenie Nowa.
  8. Skonfiguruj właściwości fabryki połączeń w wyświetlonym oknie dialogowym właściwości.
  9. Kliknij przycisk OK.
  10. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.


Zaznaczone odsyłacze (online) wymagają dostępu do Internetu.
