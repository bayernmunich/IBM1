Konfigurowanie dostawców źródeł danych (dostawców JDBC) dla klientów aplikacji

Za pomocą narzędzia ACRCT (Application Client Resource Configuration Tool) można tworzyć nowych dostawców źródła danych, nazywanych też dostawcami JDBC, dla klienta aplikacji.

Czynności wstępne

To zadanie umożliwia utworzenie nowych dostawców źródeł danych, nazywanych również dostawcami JDBC, dla klienta aplikacji. W osobnym zadaniu administracyjnym należy zainstalować kod źródłowy języka Java dla wymaganego dostawcy źródła danych na kliencie, na którym zainstalowana jest aplikacja kliencka.

Informacje dotyczące tego tematu

To zadanie umożliwia połączenie klientów aplikacji z relacyjnymi bazami danych.

Procedura

  1. Uruchom narzędzie ACRCT (Application Client Resource Configuration Tool) i otwórz plik EAR, dla którego zostanie skonfigurowany nowy dostawca źródła danych. Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  2. Wybierz z drzewa plik JAR, w którym zostanie skonfigurowany nowy dostawca źródła danych.
  3. Rozwiń plik JAR, aby wyświetlić jego zawartość.
  4. Kliknij folder Dostawcy źródeł danych. Wykonaj jedną z następujących czynności:
    • Kliknij folder prawym przyciskiem myszy, a następnie kliknij opcję Nowy dostawca.
    • Kliknij opcję Edytuj > Nowy na pasku menu.
  5. Skonfiguruj właściwości dostawcy źródła danych w wyświetlonym oknie dialogowym właściwości.
  6. Po zakończeniu kliknij przycisk OK.
  7. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.
