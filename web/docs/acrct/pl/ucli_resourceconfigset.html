Ustawienia adaptera zasobów

W tym panelu można wyświetlić lub zmienić właściwości konfiguracyjne adaptera zasobów. Właściwości te kontrolują sposób, w jaki tworzone są adaptery zasobów.

Aby wyświetlić tę stronę narzędzia ACRCT (Application Client Resource Configuration Tool), należy kliknąć opcję Plik > Otwórz. Po wyszukaniu pliku EAR należy kliknąć przycisk Otwórz. Należy rozwinąć wybrany plik JAR > Adapter zasobów. Należy kliknąć prawym przyciskiem myszy opcję Adapter zasobów, a następnie kliknąć opcję Nowy. Na karcie Ogólne zostaną wyświetlone następujące pola.

Nazwa

Nazwa służąca do identyfikowania tego adaptera zasobów w celach administracyjnych na serwerze IBM WebSphere Application Server. Nazwa musi być unikalna dla Adapterów zasobów wewnątrz domeny administracyjnej produktu.

Typ danych String
Opis

Opis adaptera zasobów używany w celach administracyjnych na serwerze IBM WebSphere Application Server.

Typ danych String
Ścieżka klasy

Dowolna dodatkowa ścieżka klasy. Ścieżka do katalogu adaptera zasobów zostanie dodana automatycznie.

Typ danych String
Wartość domyślna Ścieżka do katalogu Adaptera zasobów.
Ścieżka rodzima

Ścieżka rodzima do lokalizacji Adaptera zasobów. W tym polu należy wprowadzić dowolną dodatkową rodzimą ścieżkę klasy.

Typ danych String
Nazwa adaptera zasobów

Obowiązkowe pole wskazujące na zainstalowany podkatalog adaptera zasobów. Wpis nie przedstawia pełnej nazwy katalogu dla adaptera zasobów. Pełna nazwa katalogu jest ścieżką do zainstalowanego adaptera zasobów połączoną z nazwą adaptera zasobów.

Typ danych String
Ścieżka do zainstalowanego archiwum adaptera zasobów

Katalog, w którym zainstalowane są adaptery zasobów. Jeśli to pole nie zostanie wypełnione, system użyje wartości domyślnej.

W przypadku podania wartości ${CONNECTOR_INSTALL_ROOT}, zastąpi ona zmienną CLIENT_CONNECTOR_INSTALL_ROOT na komputerze, na którym uruchomiono aplikację klienta. Ta operacja umożliwia łatwe uruchomienie aplikacji na różnych komputerach, jeśli instalacja klienta może znajdować się w różnych lokalizacjach.

Typ danych String
Wartość domyślna ${CONNECTOR_INSTALL_ROOT}
