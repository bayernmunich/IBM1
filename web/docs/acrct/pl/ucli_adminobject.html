Konfigurowanie obiektów administrowanych

Ta sekcja ułatwia konfigurowanie nowych obiektów administrowanych.

Czynności wstępne

Przed przystąpieniem do konfigurowania nowych obiektów administrowanych należy spełnić następujące wymagania wstępne:
  1. Zainstaluj plik RAR (Resource Adapter Archive) za pomocą narzędzia clientRAR.
  2. Skonfiguruj adapter zasobów dla pliku .ear za pomocą narzędzia Application Client Resource Configuration Tool (ACRCT).

Informacje dotyczące tego tematu

Wykonaj poniższe czynności, aby skonfigurować nowe obiekty administrowane dla zainstalowanych adapterów zasobów.

Procedura

  1. Uruchom narzędzie Application Client Resource Configuration Tool (ACRCT).
  2. Otwórz plik EAR, dla którego chcesz skonfigurować nowe obiekty administrowane. Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  3. Zaznacz w widoku drzewa plik JAR, w którym chcesz skonfigurować nowe obiekty administrowane.
  4. Rozwiń plik JAR, aby wyświetlić jego zawartość.
  5. Kliknij folder Adaptery zasobów.
  6. Rozwiń adapter zasobów, dla którego chcesz utworzyć obiekty administrowane.
  7. Kliknij prawym przyciskiem myszy folder Administrowane obiekty i kliknij polecenie Nowy.
  8. Skonfiguruj właściwości obiektów administrowanych w wyświetlonym oknie dialogowym właściwości.
  9. Kliknij przycisk OK.
  10. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.


Zaznaczone odsyłacze (online) wymagają dostępu do Internetu.
