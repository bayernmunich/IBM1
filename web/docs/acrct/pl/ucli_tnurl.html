Konfigurowanie nowych dostawców URL dla klientów aplikacji

Za pomocą narzędzia ACRCT (Application Client Resource Configuration Tool) można tworzyć dostawców URL i adresy URL dla aplikacji klienckiej.

Czynności wstępne

To zadanie umożliwia utworzenie dostawców URL oraz adresów URL dla klientów aplikacji. W osobnym zadaniu administracyjnym należy zainstalować kod źródłowy języka Java dla wymaganego dostawcy URL na kliencie, na którym zainstalowana jest aplikacja kliencka.

Informacje dotyczące tego tematu

Procedura

  1. Uruchom narzędzie ACRCT (Application Client Resource Configuration Tool).
  2. Otwórz plik EAR, dla którego zostanie skonfigurowany nowy dostawca URL. Zawartość pliku EAR zostanie wyświetlona w widoku drzewa.
  3. Wybierz z drzewa JAR, w którym zostanie skonfigurowany nowy dostawca URL.
  4. Rozwiń plik JAR, aby wyświetlić jego zawartość.
  5. Kliknij folder o nazwie Dostawcy URL. Wykonaj jedną z następujących czynności:
    • Kliknij prawym przyciskiem myszy folder i wybierz opcję Nowy.
    • Kliknij opcję Edytuj > Nowy na pasku menu.
  6. Skonfiguruj właściwości dostawcy URL w wyświetlonym oknie dialogowym właściwości.
  7. Kliknij przycisk OK.
  8. Kliknij opcję Plik > Zapisz na pasku menu, aby zapisać zmiany.
