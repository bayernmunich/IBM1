애플리케이션 클라이언트에 대한 WebSphere MQ 제공자 주제 대상 설정

이 패널에서 WebSphere® MQ 제품 JMS(Java™ Message Service) 제공자에 사용하려고 선택한 주제 대상의 구성 특성을 보거나 변경할 수 있습니다.

이 애플리케이션 클라이언트 자원 구성 도구(ACRCT) 페이지를 보려면 파일 > 열기를 클릭하십시오. EAR 파일을 찾은 후 열기를 클릭하십시오. 선택된 JAR 파일 > 메시징 제공자 > WebSphere MQ 제공자를 펼치십시오. 주제 대상을 마우스 오른쪽 단추로 클릭한 다음 새로 작성을 클릭하십시오. 일반 탭에 다음 필드가 표시됩니다.

참고:
  • 사용자가 지정하는 특성 값은 WebSphere MQ용 JMS 자원 구성 시 지정한 값과 일치해야 합니다. WebSphere MQ용 JMS 자원 구성에 대한 자세한 정보는 WebSphere MQ Information Center의 Java 사용을 참조하십시오.
  • WebSphere MQ에서 이름은 최대 48자를 갖습니다.

주제 대상은 연관된 일반 JMS 제공자에 대한 JMS 주제 특성을 구성하는 데 사용됩니다. WebSphere MQ 제품 JMS 제공자에서 사용할 주제는 다음 특성을 갖습니다.
