應用程式用戶端的資源環境項目設定

請利用這個頁面來指定資源環境項目內容。

若要檢視這個「應用程式用戶端資源配置工具」(ACRCT) 頁面，請按一下檔案 > 開啟。瀏覽 EAR 檔之後，請按一下開啟。展開選取的 JAR 檔 > 資源環境提供者 > 資源環境實例。 用滑鼠右鍵按一下資源環境項目，再按一下新建。這時一般標籤會出現下列欄位：

名稱

指定資源環境項目的管理名稱。

說明

指定您管理記錄的 URL 的說明。

JNDI 名稱

指定資源的 Java™ 命名和目錄介面 (JNDI) 名稱，其中包括任何命名子環境定義。

請利用這個名稱來鏈結至平台的連結資訊。連結會將模組部署描述子所定義的資源關聯到由平台連結至 JNDI 內的實際（或實體）資源。

自訂內容

請指定名稱/值配對，用來設定在這項資源的執行時期建立的物件的其他內容。

您必須輸入名稱和值，名稱必須是物件的公開內容，值必須能夠從字串轉換成內容的 set 方法所需要的類型。可接受的內容和值會隨著所建立的物件而不同。請參閱物件文件，以取得有效內容和值的清單。
