管理物件設定

請利用這個畫面來檢視或變更所選之受管理物件的配置內容。

若要檢視這個「應用程式用戶端資源配置工具」(ACRCT) 頁面，請按一下檔案 > 開啟。瀏覽 EAR 檔之後，請按一下開啟。展開選取的 JAR 檔 > 資源配接器 > resource_adapter_instance。 用滑鼠右鍵按一下受管理物件，然後按一下新建。這時一般標籤會出現下列欄位。

受管理物件之設定的處理方式和 Connection Factory 類似。如果您要更新受管理物件，請使用建立受管理物件的同一畫面。

名稱

用來識別這個受管理物件的名稱，供 IBM® WebSphere® Application Server 內管理作業使用。 在產品管理網域之間的資源配接器受管理物件內，該名稱必須是唯一的。

資料類型 字串
說明

這個 Connection Factory 的選用說明，供 IBM WebSphere Application Server 內管理作業使用。

資料類型 字串
JNDI 名稱

這個項目是一個 resource-env-ref 名稱、message-destination-ref 名稱（如果 message-destination-ref 沒有鏈結）或 message-destination 鏈結。

資料類型 字串
類型

依照資源配接器保存檔 (RAR) 中的管理物件所定義，所有受管理物件的類別介面配對下拉清單。

每一種類型在「內容」框中都指定有一組內容。這組內容係透過從每一個受管理物件的定義擷取內容所建構而成。 對於顯示供更新的所有現有的受管理物件而言，這個內容清單會被物件所指定的內容覆蓋。當類型欄位變更時，這些內容也會變更，以反映該類型正確的內容。

資料類型 字串

檔名： ucli_adminobj.html
