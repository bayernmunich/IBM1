資源環境提供者和資源環境項目

資源環境參照會將用戶端應用程式所用的邏輯名稱，對映至物件的實體名稱。

不是連結至伺服器 JNDI 名稱空間中的所有物件都是要給應用程式用戶端使用的。比方說，WebSphere® Application Server 用戶端執行時期不支援在用戶端使用 Java™ 2 Connector (J2C) 物件。 這個物件必須能夠放在遠端，且在應用程式用戶端執行時期類別路徑中，必須能夠使用用戶端實作。

資源環境參照不同於資源參照。 資源環境參照可讓您的應用程式用戶端使用邏輯名稱來查閱連結至伺服器 JNDI 名稱空間中的資源。資源參照可讓您的應用程式使用邏輯名稱來查閱本端 J2EE 資源。J2EE 規格不指定資源的特定實作。
