針對應用程式用戶端，更新作為 Java 訊息服務提供者的 WebSphere MQ 及其 JMS 資源配置

您可以更新作為 Java™ 訊息服務 (JMS) 提供者之 WebSphere® MQ 的現有配置，以及更新 WebSphere MQ Connection Factory 或 WebSphere MQ 目的地的配置。

關於這項作業

請利用這個作業來更新作為 Java 訊息服務 (JMS) 提供者之 WebSphere MQ 的現有配置，以及更新 WebSphere MQ Connection Factory 或 WebSphere MQ 目的地的配置。

程序

  1. 啟動應用程式用戶端資源配置工具 (ACRCT)。
  2. 開啟內含 WebSphere MQ JMS 提供者、WebSphere MQ Connection Factory 或 WebSphere MQ 目的地的企業保存檔 (EAR)。 這時會在導覽樹狀視圖中顯示 EAR 檔內容。
  3. 選取內含要更新的 JMS 提供者、Connection Factory 或目的地的 Java 保存檔 (JAR)。
  4. 展開 JAR 檔來檢視它的內容，直到您找到要更新的特定 JMS 提供者、Connection Factory 或目的地為止。請完成下列其中一個動作：
    • 用滑鼠右鍵按一下適當的物件，然後按一下內容。
    • 按一下功能表列中的編輯 > 內容。
  5. 在出現的欄位中更新內容。如果需要詳細的欄位說明，請參閱：
    • JMS 提供者內容
    • MQ Queue Connection Factory 內容
    • MQ Topic Connection Factory 內容
    • MQ Queue Destination 內容
    • MQ Topic Destination 內容
  6. 按一下確定。
  7. 按一下檔案 > 儲存來儲存您的變更。


標示（線上）的鏈結表示需要存取網際網路。
