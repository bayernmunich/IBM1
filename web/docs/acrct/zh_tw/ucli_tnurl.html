配置應用程式用戶端的新 URL 提供者

您可以利用應用程式用戶端資源配置工具 (ACRCT)，來建立用戶端應用程式的 URL 提供者和 URL。

開始之前

在這項作業期間，您要建立用戶端應用程式的 URL 提供者和 URL。 在個別的管理作業中，您必須在用戶端應用程式所在的用戶端機器上，安裝必要 URL 提供者的 Java™ 程式碼。

關於這項作業

程序

  1. 啟動應用程式用戶端資源配置工具 (ACRCT)。
  2. 開啟要配置新 URL 提供者的 EAR 檔。 這時會在樹狀視圖中顯示 EAR 檔內容。
  3. 從樹狀結構中，選取您要配置新 URL 提供者的 JAR 檔。
  4. 展開 JAR 檔來檢視內容。
  5. 按一下稱為 URL 提供者的資料夾。請完成下列其中一個動作：
    • 用滑鼠右鍵按一下資料夾並選取新建。
    • 按一下功能表列中的編輯 > 新建。
  6. 在出現的內容對話框中，配置 URL 提供者內容。
  7. 按一下確定。
  8. 按一下功能表列中的檔案 > 儲存來儲存您的變更。


標示（線上）的鏈結表示需要存取網際網路。
