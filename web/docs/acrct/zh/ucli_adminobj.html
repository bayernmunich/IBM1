受管对象设置

使用此面板查看或更改所选受管对象的配置属性。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件 > 资源适配器 > resource_adapter_instance。 右键单击受管对象，并单击新建。以下字段出现在常规选项卡上。

对受管对象的设置的处理类似于连接工厂。当更新受管对象时，使用您用于创建受管对象的相同的面板。

名称

在 IBM® WebSphere® Application Server 中，为了进行管理而提供的用于识别此受管对象的名称。此名称在跨产品管理域的资源适配器受管对象中必须唯一。

数据类型 字符串
描述

在 IBM WebSphere Application Server 中，为了进行管理而提供的对此连接工厂的可选描述。

数据类型 字符串
JNDI 名称

此条目是 resource-env-ref 名称、message-destination-ref 名称（如果 message-destination-ref 没有链接）或 message-destination 链接。

数据类型 字符串
类型

为资源适配器归档（RAR）文件中的管理对象定义的所有受管对象类接口对的下拉列表。

对于每种类型，都有在“属性”框中指定的一组属性。通过从每个受管对象定义获取属性来构造该组属性。对于任何为更新而显示的现有受管对象，用为对象指定的属性覆盖此属性列表。当类型字段已更改时，属性也更改，以反映该类型的正确属性。

数据类型 字符串

文件名： ucli_adminobj.html
