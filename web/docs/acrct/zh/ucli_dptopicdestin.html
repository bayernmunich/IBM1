缺省提供程序主题目标设置

使用此面板查看或更改所选 JMS 主题目标的配置属性，以供随 WebSphere® Application Server 安装的内部产品 Java™ 消息服务（JMS）提供程序使用。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件 > 消息传递提供程序 > 缺省提供程序。右键单击主题目标，然后单击新建。以下字段出现在常规选项卡上。

名称

主题目标条目的名称。

数据类型 字符串
描述

条目的描述。

数据类型 字符串
JNDI 名称

用于将此定义与部署描述符 resource-env-ref 名称匹配的 JNDI 名称。

数据类型 字符串
主题空间

主题空间的名称。此字段是必填字段。

数据类型 字符串
缺省值 DEFAULT_TOPIC_SPACE
主题名称

主题的名称。 此字段是必填字段。

数据类型 字符串
交付方式

发送到此目标的消息的缺省方式。

数据类型 字符串
范围 Application、Persistent 或 NonPersistent
缺省值 应用程序
生存时间

消息发送到此目标的分派时间的缺省时间长度应该由系统保留，其中 0 表明生存时间值不到期。如果此项未完成，那么使用来自生产者的值。

数据类型 Long
单元 毫秒
优先级

发送到此目标的消息的优先级。如果此项未完成，那么使用来自生产者的值。

数据类型 整型
范围 0 到 9，0 为最低优先级，9 为最高优先级
预读

用于控制消息交付期间的预读优化。

数据类型 字符串
范围 AsConnection、AlwaysOn 和 AlwaysOff
缺省值 AsConnection
