应用程序客户机的数据源提供程序设置

在提供特定 JDBC 驱动程序实现类的 JDBC 提供程序下使用此页面创建数据源。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件。右键单击数据源提供程序 > 并单击新建。以下字段出现在常规选项卡上：

名称

指定数据源的显示名称。

例如，您可将该字段设置为测试数据源。

数据类型 字符串
描述

指定资源的文本描述。

数据类型 字符串
类路径

路径或 .jar 文件名的列表，它们共同组成资源提供程序类的位置。

实现类

使用此设置执行数据库特定功能。

数据类型 字符串
缺省值 取决于 JDBC 驱动程序实现类
定制属性

指定用于设置运行时为此资源创建的对象上的其他属性的“名称/值”对。

您必须输入是对象上的公用属性的名称，以及可以从字符串转换到属性的 set 方法必需的类型的值。可接受的属性和值取决于创建的对象。请参阅对象文档，以获取有效属性和值的列表。
