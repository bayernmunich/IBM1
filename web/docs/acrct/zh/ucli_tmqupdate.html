为应用程序客户机更新作为 Java 消息服务提供程序的 WebSphere MQ，以及它的 JMS 资源配置

可以更新作为 Java™ 消息服务（JMS）提供程序的 WebSphere® MQ 的现有配置，并且更新 WebSphere MQ 连接工厂或 WebSphere MQ 目标的配置。

关于本任务

使用此任务来更新作为 Java 消息服务（JMS）提供程序的 WebSphere MQ 的现有配置，并且更新 WebSphere MQ 连接工厂或 WebSphere MQ 目标的配置。

过程

  1. 启动应用程序客户机资源配置工具（ACRCT）。
  2. 打开包含 WebSphere MQ JMS 提供程序、WebSphere MQ 连接工厂或 WebSphere MQ 目标的企业归档（EAR）文件。 EAR 文件内容显示在导航树形视图中。
  3. 选择要更新的包含 JMS 提供程序、连接工厂或目标的 Java 归档（JAR）文件。
  4. 展开 JAR 文件查看它的内容，直到您找到要更新的特定 JMS 提供程序、连接工厂或目标。完成以下某个操作：
    • 右键单击相应的对象，并单击属性。
    • 在菜单栏上单击编辑 > 属性。
  5. 在显示的字段中更新属性。要获取详细的字段帮助，请参阅：
    • JMS 提供程序属性
    • MQ 队列连接工厂属性
    • MQ 主题连接工厂属性
    • MQ 队列目标属性
    • MQ 主题目标属性
  6. 单击确定。
  7. 单击文件>保存以保存埂更改。


标有（在线）的链接要求访问因特网。
