应用程序客户机的 WebSphere MQ 提供程序主题目标设置

使用此面板查看或更改所选主题目标的配置属性，以供 WebSphere® MQ 产品 Java™ 消息服务 (JMS) 提供程序使用。

要查看该 Application Client Resource Configuration Tool (ACRCT) 页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件 > 消息传递提供程序 > WebSphere MQ 提供程序。右键单击主题目标，并单击新建。 以下字段出现在常规选项卡上。

注意：
  • 指定的属性值必须与为 WebSphere MQ 配置 JMS 资源时指定的值相匹配。有关为 WebSphere MQ 配置 JMS 资源的更多信息，请参阅 WebSphere MQ 信息中心中的“使用 Java”。
  • 在 WebSphere MQ 中，名称最多可以有 48 个字符。

主题目标用于配置关联 JMS 提供程序的 JMS 主题的属性。供 WebSphere MQ 产品 JMS 提供程序使用的主题具有下列属性。

名称

在 WebSphere Application Server 中，为了进行管理而提供的用于识别主题的名称。

数据类型 字符串
描述

在 WebSphere Application Server 中，为了进行管理而提供的对主题的描述。

数据类型 字符串
JNDI 名称

应用程序客户机运行时环境使用此字段来检索配置信息。

持久性

发送到目标的所有消息是持久的、非持久的，还是由应用程序定义其持久性。

数据类型 枚举
缺省值 APPLICATION_DEFINED
范围
应用程序定义
目标上的消息由将它们放入队列的应用程序定义其持久性。
队列定义
［仅用于 WebSphere MQ 目标］目标上的消息由 WebSphere MQ 队列定义属性定义其持久性。
持久
目标上的消息是持久的。
非持久
目标上的消息是不持久的。
优先级

是由应用程序还是由指定的优先级属性定义此目标的消息优先级。

数据类型 枚举
缺省值 APPLICATION_DEFINED
范围
应用程序定义
此目标上消息的优先级由将它们放入目标的应用程序定义。
队列定义
［仅用于 WebSphere MQ 目标］目标上的消息由 WebSphere MQ 队列定义属性定义其持久性。
指定的
此目标上的消息的优先级由指定的优先级属性来定义。如果您选择此选项，那么必须为指定的优先级属性定义优先级。
指定的优先级

如果优先级属性设置为 Specified，那么指定该队列的消息优先级，其范围在 0（最低）到 9（最高）之间。

如果优先级属性设置为 Specified，那么发送到此队列的消息具有此属性指定的优先级值。

数据类型 整型
单元 消息优先级级别
范围 0（最低优先级）到 9（最高优先级）
到期时间

此队列的到期超时是由应用程序定义还是由指定的到期属性定义，或者队列上的消息是否从不到期（具有无限到期超时）。

数据类型 枚举
缺省值 APPLICATION_DEFINED
范围
应用程序定义
此队列上的消息的到期超时由将它们放入队列的应用程序定义。
指定的
此队列中的消息的到期超时由指定的到期属性定义。如果您选择此选项，那么必须为指定的到期属性定义超时值。
无限
此队列上的消息无到期超时，所以这些消息从不到期。
指定的到期时间

如果到期超时属性设置为 Specified，那么输入此队列上消息到期前的毫秒数（大于 0）。

数据类型 整型
单元 毫秒
范围 大于或等于 0
  • 0 表明消息从不超时。
  • 其他值是整数的毫秒。
基本主题名称

消息将发送至的主题名。

数据类型 字符串
CCSID

要由 WebSphere MQ 队列管理器使用的编码字符集标识。

此编码字符集标识 (CCSID) 必须是 WebSphere MQ 所支持的某个 CCSID。有关更多详细信息，请参阅“WebSphere MQ 消息传递提供程序队列和高级属性设置”主题。

数据类型 字符串
单元 整型
范围 1 到 65535
整数编码

如果未启用本机编码，那么选择整型编码是正常的还是逆向的。

数据类型 枚举
缺省值 NORMAL
范围
NORMAL
使用正常整型编码。
REVERSED
使用逆向整型编码。

有关编码属性的更多信息，请参阅 WebSphere MQ 信息中心中的“使用 Java”。
十进制编码

如果未启用本机编码，那么选择十进制编码是正常的还是逆向的。

数据类型 枚举
缺省值 NORMAL
范围
NORMAL
使用正常十进制编码。
REVERSED
使用逆向十进制编码。

有关编码属性的更多信息，请参阅 WebSphere MQ 信息中心中的“使用 Java”。
浮点编码

如果未启用本机编码，那么选择浮点编码类型。

数据类型 枚举
缺省值 IEEENORMAL
范围
IEEENORMAL
使用 IEEE 正常浮点编码。
IEEEREVERSED
使用 IEEE 逆向浮点编码。
S390
使用 S/390® 浮点编码。

有关编码属性的更多信息，请参阅 WebSphere MQ 信息中心中的“使用 Java”。
本机编码

选中此复选框时表明队列目标使用本机编码（Java 平台的相应编码值）。

数据类型 枚举
缺省值 未选中
范围
未选中
未使用本机编码，因此为整型、十进制和浮点编码指定前面的属性。
所选项
使用本机编码（为 Java 平台提供相应的编码值）。

有关编码属性的更多信息，请参阅 WebSphere MQ 信息中心中的“使用 Java”。
BrokerDurSubQueue

从中检索持久预订消息的代理队列的名称。

订户在注册预订时指定队列名。

数据类型 字符串
单元 美国英语 ASCII 字符
范围 1 到 48 个 ASCII 字符
BrokerCCDurSubQueue

从中为 ConnectionConsumer 检索持久预订消息的代理队列的名称。此属性仅适用于 Web 容器使用。

数据类型 字符串
单元 美国英语 ASCII 字符
范围 1 到 48 个 ASCII 字符
目标客户机

接收应用程序是符合 JMS 还是传统的 WebSphere MQ 应用程序。

数据类型 枚举
缺省值 WebSphere MQ
范围
WebSphere MQ
目标是不支持 JMS 的传统的 WebSphere MQ 应用程序。
JMS
目标应用程序支持 JMS。
消息体

指定应用程序是否将 WebSphere MQ 消息的 RFH V2 头作为 JMS 消息体的一部分进行处理。

数据类型 下拉列表
缺省值 UNSPECIFIED
范围
UNSPECIFIED
发送消息时，根据“将 RFH V2 头追加至发送至此目标的消息”属性的值，WebSphere MQ 消息传递提供程序生成并包括或不生成并包括 RFH V2 头。接收消息时，WebSphere MQ 消息传递提供程序的行为方式按此值设置为 JMS 时进行。
JMS
发送消息时，WebSphere MQ 消息传递提供程序自动生成 RFH V2 头并将其包括在 WebSphere MQ 消息中。接收消息时，WebSphere MQ 消息传递提供程序根据 RFH V2 头中的值（如果提供了这些值）设置 JMS 消息属性；它不会在 JMS 消息体中显示 RFH V2 头。
MQ
发送消息时，WebSphere MQ 消息传递提供程序不生成 RFH V2 头。接收消息时，WebSphere MQ 消息传递提供程序在 JMS 消息体中显示 RFH V2 头。
ReplyTo 目标样式

指定 JMSReplyTo 字段的格式。

数据类型 下拉列表
缺省值 缺省值
范围
缺省值
缺省值等于 RFH V2 头中的信息。
MQMD
使用 MQMD 中提供的值。这会使用 MQMD 中的值填充应答队列管理器字段，相当于 WebSphere MQ V6.0.2.4 和 V6.0.2.5 的缺省行为。
RFH2
使用 RFH V2 头中提供的值。如果发送应用程序设置了 JMSReplyTo 值，那么会使用该值。
多点广播

此连接工厂是否使用多点广播传输。

数据类型 枚举
缺省值 AS_CF
范围
AS_CF
此连接工厂使用多点广播传输。
DISABLED
此连接工厂不使用多点广播传输。
NOT_RELIABLE
此连接工厂总是使用多点广播传输。
RELIABLE
当主题目标不可靠时，此连接工厂使用多点广播传输。
ENABLED
此连接工厂使用可靠的多点广播传输。
MQMD 读取已启用

指定应用程序能否从使用 WebSphere MQ 消息传递提供程序发送或接收的 JMS 消息中读取 MQMD 字段的值。

数据类型 复选框
缺省值 未选中
范围
未选中
应用程序不能读取 MQMD 字段的值。
所选项
应用程序可读取 MQMD 字段的值。
MQMD 写入已启用

指定应用程序能否在使用 WebSphere MQ 消息传递提供程序发送或接收的 JMS 消息中写入 MQMD 字段的值。

数据类型 复选框
缺省值 未选中
范围
未选中
应用程序不能写入 MQMD 字段的值。
所选项
应用程序可写入 MQMD 字段的值。
MQMD 消息上下文

定义将消息发送至目标时指定的消息上下文选项。

数据类型 下拉列表
缺省值 缺省值
范围
缺省值
MQOPEN API 调用和 MQPMO 结构未指定显式消息上下文选项。
SET_IDENTITY_CONTEXT
MQOPEN API 调用指定消息上下文选项 MQOO_SET_IDENTITY_CONTEXT，MQPMO 结构指定 MQPMO_SET_IDENTITY_CONTEXT。
SET_ALL_CONTEXT
MQOPEN API 调用指定消息上下文选项 MQOO_SET_ALL_CONTEXT，MQPMO 结构指定 MQPMO_SET_ALL_CONTEXT。
