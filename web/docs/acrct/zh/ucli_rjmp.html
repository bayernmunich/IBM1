应用程序客户机的邮件提供者设置

使用这个页面实现 JavaMail API 和创建邮件会话。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件。右键单击邮件提供者 > 并单击新建。以下字段出现在常规选项卡上：

名称

JavaMail 资源提供程序的名称。

描述

资源提供程序的可选描述。

类路径

指定路径或 JAR 文件名的列表，它们共同构成资源提供程序类的位置。

协议

指定协议名称。

类名

指定实现了协议的类的名称。如果您要使用缺省实现，保持该字段为空白。

类型

该菜单包含了以下两个值：TRANSPORT 或 STORE。

定制属性

指定用于设置运行时为此资源创建的对象上的其他属性的“名称/值”对。

您必须输入是对象上的公用属性的名称，以及可以从字符串转换到属性的 set 方法必需的类型的值。可接受的属性和值取决于创建的对象。请参阅对象文档，以获取有效属性和值的列表。
