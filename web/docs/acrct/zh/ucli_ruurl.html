应用程序客户机的 URL 设置

使用此页面实现特定 URL 协议的功能，如超文本传输协议（HTTP）。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件 > URL 提供程序 > URL 提供程序实例。右键单击 URL 并单击新建。以下字段出现在常规选项卡上。

此由类组成的提供程序扩展 java.net.URLStreamHandler 和 java.net.URLConnection 类。

名称

URL 的管理名称。

描述

这是对 URL 的可选描述，以供管理记录使用。

JNDI 名称

应用程序客户机运行时使用此字段检索配置信息。

URL

指向因特网或内部网资源的统一资源定位符（URL）名称。例如：http://www.ibm.com。

定制属性

指定用于设置运行时为此资源创建的对象上的其他属性的“名称/值”对。

您必须输入是对象上的公用属性的名称，以及可以从字符串转换到属性的 set 方法必需的类型的值。可接受的属性和值取决于创建的对象。请参阅对象文档，以获取有效属性和值的列表。



标有（在线）的链接要求访问因特网。
