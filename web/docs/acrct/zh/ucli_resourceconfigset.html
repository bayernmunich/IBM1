资源适配器设置

使用此面板查看或更改资源适配器的配置属性。这些配置属性控制如何创建资源适配器。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 JAR 文件 > 资源适配器。右键单击资源适配器，并单击 新建。以下字段出现在常规选项卡上。

名称

在 IBM® WebSphere® Application Server 中，为了进行管理而提供的用于识别此资源适配器的名称。此名称在跨产品管理域的资源适配器中必须唯一。

数据类型 字符串
描述

在 IBM WebSphere Application Server 中，为了进行管理而提供的对此资源适配器的描述。

数据类型 字符串
类路径

任何其他类路径。自动添加到资源适配器目录的路径。

数据类型 字符串
缺省值 到您的资源适配器目录的路径。
本机路径

资源适配器所在的本机路径。在此处输入任何其他本机类路径。

数据类型 字符串
资源适配器名称

指向安装的资源适配器子目录的必需字段。此条目不表示资源适配器的完整目录名称。完整目录名称是安装的资源适配器路径加上资源适配器名称。

数据类型 字符串
安装的资源适配器路径

安装资源适配器的目录。如果您未完成此字段，那么缺省值生效。

如果您指定值 ${CONNECTOR_INSTALL_ROOT}，那么该值替换运行客户机应用程序的机器上的 CLIENT_CONNECTOR_INSTALL_ROOT 变量的值。此操作使应用程序易于在不同的机器上运行，在这些机器上，客户机安装可能在不同的位置中。

数据类型 字符串
缺省值 ${CONNECTOR_INSTALL_ROOT}
