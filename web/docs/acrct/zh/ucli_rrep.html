应用程序客户机的资源环境提供程序设置

使用此页面来指定资源环境条目属性。

要查看该 Application Client Resource Configuration Tool（ACRCT）页面，单击文件 > 打开。在您浏览 EAR 文件后，单击打开。展开所选 Java™ 归档（JAR）文件。右键单击资源环境提供程序，并单击新建。以下字段出现在常规选项卡上：

名称

指定资源环境提供程序的管理名称。

描述

指定管理记录的资源环境提供程序的描述。

类路径

指定包含资源环境提供程序实现类的 JAR 文件的路径。

定制属性

指定用于设置运行时为此资源创建的对象上的其他属性的“名称/值”对。

您必须输入是对象上的公用属性的名称，以及可以从字符串转换到属性的 set 方法必需的类型的值。可接受的属性和值取决于创建的对象。请参阅对象文档，以获取有效属性和值的列表。
