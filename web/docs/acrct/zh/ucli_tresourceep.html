为应用程序客户机配置新资源环境提供程序

可以使用应用程序客户机资源配置工具（ACRCT）为应用程序客户机创建新的资源环境提供程序配置。

在您开始之前

执行此任务期间，为应用程序客户机创建新的资源环境提供程序配置。

关于本任务

要配置新的资源环境提供程序，执行以下步骤：

过程

  1. 启动 Application Configuration Resource Tool 并打开 EAR 文件，您要为该 EAR 文件配置新的 Java™ 消息服务（JMS）提供程序。EAR 文件内容显示在树形视图中。
  2. 从此树中选择您要在其中配置新的 JMS 提供程序的 JAR 文件。
  3. 展开 JAR 文件以查看其内容。
  4. 单击资源环境提供程序文件夹。执行以下某个操作：
    • 右键单击提供程序文件夹，然后单击新建。
    • 在菜单栏上单击编辑 > 新建。
  5. 在显示的字段中配置 JMS 提供程序属性。
  6. 完成时单击确定。
  7. 在菜单栏上单击文件 > 保存以保存您的更改。


标有（在线）的链接要求访问因特网。
