为应用程序客户机配置新的 URL 提供程序

可以使用应用程序客户机资源配置工具（ACRCT）为客户机应用程序创建 URL 提供程序和 URL。

在您开始之前

执行此任务期间，为您的客户机应用程序创建 URL 提供程序和 URL。在独立的管理任务中，必须在客户机应用程序驻留的客户机上安装必需的 URL 提供程序的 Java™ 代码。

关于本任务

过程

  1. 启动 Application Client Resource Configuration Tool（ACRCT）。
  2. 打开要为其配置新的 URL 提供程序的 EAR 文件。 EAR 文件内容显示在树形视图中。
  3. 从树中选择您要在其中配置新的 URL 提供程序的 JAR 文件。
  4. 展开 JAR 文件查看内容。
  5. 单击名为 URL 提供程序的文件夹。完成以下某个操作：
    • 右键单击文件夹并选择新建。
    • 在菜单栏上单击编辑 > 新建。
  6. 在结果属性对话框中，配置 URL 提供程序属性。
  7. 单击确定。
  8. 在菜单栏上单击文件 > 保存以保存您的更改。


标有（在线）的链接要求访问因特网。
