为应用程序客户机配置新数据源提供程序（JDBC 提供程序）

可以使用应用程序客户机资源配置工具（ACRCT）为应用程序客户机创建新的数据源提供程序（也称为 JDBC 提供程序）。

在您开始之前

执行此任务期间，您将为您的应用程序客户机创建新的数据源提供程序，也称为 JDBC 提供程序。在独立的管理任务中，在应用程序客户机驻留的客户机上安装必需的数据源提供程序的 Java™ 代码。

关于本任务

使用此任务将应用程序客户机连接到关系数据库。

过程

  1. 启动 Application Client Resource Configuration Tool（ACRCT），并打开您要配置新的数据源提供程序的 EAR 文件。EAR 文件内容显示在树形视图中。
  2. 从树中选择您要在其中配置新数据源提供程序的 JAR 文件。
  3. 展开 JAR 文件以查看其内容。
  4. 单击数据源提供程序文件夹。执行以下某个任务：
    • 右键单击文件夹并单击新建提供程序。
    • 在菜单栏上单击编辑 > 新建。
  5. 在结果属性对话框中，配置数据源提供程序属性。
  6. 完成时单击确定。
  7. 在菜单栏上单击文件 > 保存以保存您的更改。


标有（在线）的链接要求访问因特网。
