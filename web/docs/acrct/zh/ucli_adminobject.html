配置受管对象

本节帮助您配置新的受管对象。

在您开始之前

您配置新的受管对象之前，必须先完成安装以下必备软件：
  1. 使用 clientRAR 工具安装资源适配器归档文件（RAR）。
  2. 使用 Application Client Resource Configuration Tool（ACRCT）工具为 .ear 文件配置资源适配器。

关于本任务

完成此任务，为安装的资源适配器配置新的受管对象。

过程

  1. 启动 Application Client Resource Configuration Tool（ACRCT）。
  2. 打开您要配置新受管对象的 EAR 文件。 EAR 文件内容显示在树形视图中。
  3. 从树中选择您要配置新受管对象的 JAR 文件。
  4. 展开 JAR 文件以查看其内容。
  5. 单击资源适配器文件夹。
  6. 展开您要创建受管对象的资源适配器。
  7. 右键单击受管对象文件夹，并单击新建。
  8. 在产生的属性对话框中配置受管对象属性。
  9. 单击确定。
  10. 在菜单栏上单击文件 > 保存以保存您的更改。


标有（在线）的链接要求访问因特网。
