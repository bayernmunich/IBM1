资源环境提供程序和资源环境条目

一个资源环境引用将客户机应用程序使用的逻辑名映射至一个对象的物理名称。

不是所有绑定到服务器 JNDI 名称空间的对象都能供应用程序客户机使用。例如，该 WebSphere® Application Server 客户机运行时不支持客户机上 Java™ 2 连接器（J2C）对象的使用。该对象需为可远程访问，且客户端实现必须在应用程序客户机运行时类路径上可用。

资源环境引用不同于资源引用。 资源环境引用允许您的应用程序客户机使用逻辑名来查寻绑定在服务器 JNDI 名称空间的资源。资源引用允许应用程序使用逻辑名来查寻一个本地 J2EE 资源。J2EE 规范不指定资源的特殊实现。
