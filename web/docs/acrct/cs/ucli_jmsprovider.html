Poskytovatelé rozhraní JMS (Java Message Service) pro klienty

V tomto tématu jsou popsány různé způsoby, jak mohou klientské aplikace používat poskytovatele rozhraní JMS(Java Message Service) se serverem WebSphere Application Server.

Produkt IBM WebSphere Application Server podporuje asynchronní systém zpráv prostřednictvím poskytovatele rozhraní JMS a jeho souvisejícího systému zpráv. Poskytovatelé JMS musí splňovat specifikaci JMS verze 1.1. Chcete-li používat objekty bean řízené zprávami, musí poskytovatel JMS podporovat volitelnou funkci ASF (Application Server Facility) definovanou v dané specifikaci nebo podporovat adaptér příchozích prostředků definovaný ve specifikaci JCA verze 1.5.

Technologie integrace služeb serveru IBM WebSphere Application Server se mohou chovat jako systém zpráv v případě, že jste konfigurovali sběrnici pro integraci služeb, která je přístupná prostřednictvím výchozího poskytovatele systému zpráv. Tato podpora je instalována jako součást serveru WebSphere Application Server, je spravována pomocí konzoly pro zprávu a je plně integrována s běhovou komponentou serveru WebSphere Application Server.

Server WebSphere Application Server dále poskytuje podporu následujících poskytovatelů rozhraní JMS:
WebSphere MQ
Tato podpora je k dispozici pro použití s podporovanými verzemi produktu WebSphere MQ.
Generické
Poskytnuto k použití s libovolným systémem zpráv třetí strany, který podporuje funkci ASF.

Za účelem zpětné kompatibility s předchozími verzemi zahrnuje produkt WebSphere Application Server také podporu výchozího poskytovatele systému zpráv verze 5, který umožňuje konfigurovat prostředky pro použití s vestavěným systémem zpráv serveru WebSphere Application Server verze 5. Výchozí poskytovatel systému zpráv verze 5 může být dále používán se sběrnicí pro integraci služeb.

Aplikace produktu WebSphere mohou používat prostředky systému zpráv poskytované jakýmkoli z těchto poskytovatelů rozhraní JMS. Výběr poskytovatele je však ve většině případů diktován požadavky pro použití nebo integrování s existujícím systémem zpráv. Například již můžete mít infrastrukturu systému zpráv založenou na produktu WebSphere MQ. V takovém případě se můžete buď připojit přímo pomocí zahrnuté podpory produktu WebSphere MQ jako poskytovatele rozhraní JMS, nebo konfigurovat sběrnici pro integraci služeb s odkazy na síť produktu WebSphere MQ a získávat ke sběrnici přístup prostřednictvím výchozího poskytovatele systému zpráv.

Sběrnice pro integraci služeb poskytuje rovněž přístup k výchozímu poskytovateli systému zpráv. Jde o poskytovatele systému zpráv rozhraní JMS s možností použití v prostředí J2EE 1.4, který je plně integrován do produktu WebSphere Application Server. Můžete jej používat v konfiguracích s více servery pro interakce systému zpráv se sítí produktu WebSphere MQ.



Odkazy s označením (online) vyžadují přístup k Internetu.
