Poskytovatelé URL pro konfigurační nástroj prostředků aplikačního klienta

Poskytovatel URL implementuje funkci pro konkrétní protokol adres URL, například pro protokol HTTP (HyperText Transfer Protocol). Tento poskytovatel sestávající ze dvojice tříd, rozšiřuje třídy java.net.URLStreamHandler a java.net.URLConnection.



Odkazy s označením (online) vyžadují přístup k Internetu.
