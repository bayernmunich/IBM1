Nástroj pro konfiguraci prostředků aplikačního klienta

Přechod na témata nápovědy v této sekci

  • Adaptéry prostředků pro klienta
    Adaptér prostředků je softwarový ovladač na úrovni systému, který je aplikací prostředí Java používán pro připojení k podnikovému informačnímu systému (EIS). Adaptér prostředků se připojuje k aplikačnímu klientu a poskytuje připojení mezi systémem EIS a podnikovou aplikací.
  • Poskytovatelé prostředí prostředků a položky prostředí prostředků
    Odkaz na prostředí prostředku mapuje logický název používaný klientskou aplikací na fyzický název objektu.
  • Poskytovatelé rozhraní JMS (Java Message Service) pro klienty
    V tomto tématu jsou popsány různé způsoby, jak mohou klientské aplikace používat poskytovatele rozhraní JMS(Java Message Service) se serverem WebSphere Application Server.
  • Zpracování asynchronních zpráv v produktu WebSphere Application Server prostřednictvím rozhraní JMS
    Server WebSphere Application Server podporuje asynchronní systém zpráv jako metodu komunikace založenou na programovacím rozhraní JMS (Java Message Service). Rozhraní JMS poskytuje běžný způsob vytváření, odesílání, přijímání a čtení asynchronních žádostí jako zpráv rozhraní JMS pro programy prostředí Java (klienti i aplikace Java Platform, Enterprise Edition - Java EE).
  • Konfigurování nových faktorií připojení pro adaptéry prostředků
    Prostřednictvím konfiguračního nástroje pro prostředky aplikačního klienta (ACRCT) můžete konfigurovat nové faktorie připojení pro adaptéry prostředků.
  • Nastavení adaptéru prostředků
    Tento panel slouží k zobrazení nebo změně konfiguračních vlastností vybraného adaptéru prostředků. Tyto konfigurační vlastnosti řídí způsob vytváření adaptérů prostředků.
  • Nastavení adres URL pro aplikační klienty
    Prostřednictvím této stránky můžete implementovat funkci pro konkrétní protokol adres URL, například pro protokol HTTP (Hyper Text Transfer Protocol).
  • Nastavení faktorie připojení fronty poskytovatele WebSphere MQ pro aplikační klienty
    Tento panel slouží k zobrazení nebo změně konfiguračních vlastností vybrané faktorie připojení fronty, která má být použita v kombinaci s poskytovatelem rozhraní JMS (Java Message Service) produktu MQSeries. Tyto konfigurační vlastnosti řídí způsob vytváření připojení mezi poskytovatelem rozhraní JMS a produktem WebSphere MQ.
  • Nastavení faktorie připojení fronty výchozího poskytovatele
    Tento panel slouží k zobrazení nebo změně konfiguračních vlastností vybrané faktorie připojení fronty rozhraní JMS, která má být použita v kombinaci s interním poskytovatelem rozhraní JMS (Java Message Service) produktu instalovaným spolu s produktem WebSphere Application Server. Tyto konfigurační vlastnosti řídí způsob vytváření připojení mezi poskytovatelem rozhraní JMS a sběrnicí pro integraci služeb, kterou využívá.
  • Nastavení faktorie připojení fronty výchozího poskytovatele verze 5 pro aplikační klienty

Tyto konfigurační vlastnosti řídí způsob vytváření připojení mezi poskytovatelem rozhraní JMS a výchozím systémem zpráv, který využívá.
