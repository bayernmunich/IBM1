Definições de Provedor de Recursos para Aplicativos Clientes

Utilize esta página para especificar as propriedades de entrada do ambiente de recursos.

Para visualizar essa página da ACRCT (Application Client Resource Configuration Tool), clique em Arquivo > Abrir. Após procurar por um arquivo EAR, clique em Abrir. Expanda o arquivo Java Archive (JAR) selecionado. Clique com o botão direito do mouse em Provedores de Ambiente de Recursos e clique em Novo. Os campos a seguir são exibidos na guia Geral:

Nome

Especifica o nome administrativo para o provedor de ambiente de recursos.

Descrição

Especifica uma descrição do provedor de ambiente de recursos para seus registros administrativos.

Caminho da Classe

Especifica o caminho para o arquivo JAR que contém as classes de implementação para o provedor de ambiente de recursos.

Propriedades Personalizadas

Especifica pares de nome-valor para definição de propriedades adicionais no objeto criado no tempo de execução deste recurso.

Você deve inserir um nome que é uma propriedade pública no objeto e um valor que pode ser convertido de cadeia para o tipo requerido pelo método de definição da propriedade. As propriedades e valores aceitáveis dependem do objeto que é criado. Consulte a documentação do objeto para obter uma lista de propriedades e valores válidos.



Links marcados (on-line) requerem acesso à Internet.
