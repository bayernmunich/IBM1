Configurando Novos Connection Factories de Adaptadores de Recursos para o Cliente

Use o Application Client Resource Configuration Tool (ACRCT) para configurar novos connection factories de adaptadores de recursos para o cliente.

Sobre Esta Tarefa

Conclua esta tarefa para configurar novas connection factories para adaptadores de recursos.

Procedimento

  1. Inicie a ACRTCT (Application Client Resource Configuration Tool).
  2. Abra o arquivo EAR para o qual você deseja configurar novas connection factories. O conteúdo do arquivo EAR é exibido em uma exibição em árvore.
  3. Selecione o arquivo JAR no qual você deseja configurar as novas connection factories a partir da árvore.
  4. Expanda o arquivo JAR para exibir seu conteúdo.
  5. Clique na pasta Adaptadores de Recursos.
  6. Expanda o adaptador de recursos para o qual deseja criar connection factories.
  7. Clique com o botão direito do mouse na pasta Connection Factories e clique em Novo.
  8. Configure as propriedades da connection factory no diálogo de propriedade resultante.
  9. Clique em OK.
  10. Clique em Arquivo > Salvar na barra de menus para salvar suas alterações.


Links marcados (on-line) requerem acesso à Internet.
