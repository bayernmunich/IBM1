Configurando Novos Provedores de Ambiente de Recursos para Clientes Aplicativos

É possível criar novas configurações do provedor de ambiente de recursos para seu cliente aplicativo utilizando a ACRCT (Application Client Resource Configuration Tool).

Antes de Começar

Durante esta tarefa, você criará novas configurações do provedor de ambiente de recursos para seu aplicativo cliente.

Sobre Esta Tarefa

Para configurar um novo provedor de ambiente de recursos, execute as etapas a seguir:

Procedimento

  1. Inicie o Application Configuration Resource Tool e abra o arquivo EAR no qual você deseja configurar o novo provedor Java Message Service (JMS). O conteúdo do arquivo EAR é exibido em uma visualização em árvore.
  2. Na árvore, selecione o arquivo JAR no qual deseja configurar o novo provedor JMS.
  3. Expanda o arquivo JAR para exibir seu conteúdo.
  4. Clique na pasta Provedores de Ambiente de Recursos. Execute uma das ações a seguir:
    • Clique com o botão direito do mouse na pasta de provedores e clique em Novo.
    • Clique em Editar > Novo na barra de menus.
  5. Configure as propriedades do provedor JMS nos campos exibidos.
  6. Clique em OK quando você terminar.
  7. Clique em Arquivo > Salvar na barra de menus para salvar as alterações.


Links marcados (on-line) requerem acesso à Internet.
