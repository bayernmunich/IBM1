Configurações da Connection Factory do Adaptador de Recursos

Utilize esse painel para visualizar ou alterar as propriedades de configuração da connection factory do adaptador de recursos selecionada.

Para visualizar essa página da ACRCT (Application Client Resource Configuration Tool), clique em Arquivo > Abrir. Após procurar por um arquivo EAR, clique em Abrir. Expanda o Arquivo JAR selecionado > Adaptadores de Recursos. Clique com o botão direito do mouse na pasta Connection Factories e clique em Novo. Os seguintes campos aparecem na guia Geral.

Nome

Nome pelo qual esse connection factory é conhecido para propósitos administrativos no WebSphere Application Server.O nome deve ser exclusivo nas connection factories do adaptador de recursos em todo o domínio administrativo do produto.

Tipo de Dado Seqüência de Caracteres
Descrição

Uma descrição opcional deste connection factory para propósitos administrativos dentro do IBM WebSphere Application Server.

Tipo de Dado Seqüência de Caracteres
Nome JNDI

O nome JNDI utilizado para corresponder esta definição da connection factory do adaptador de recursos com o descritor de implementação. Esta entrada deve ser um nome resource-ref.

Tipo de Dado Seqüência de Caracteres
ID do Usuário

O Nome do Usuário utilizado com a propriedade Senha para autenticação se o aplicativo de chamada não fornecer ID do usuário e senha explicitamente ao obter uma conexão. Se esse campo for utilizado, então, o UserName do campo Propriedades é ignorado.

Se você especificar um valor para a propriedade Nome do Usuário, é necessário também especificar um valor para a propriedade Senha.

As propriedades Nome do Usuário e Senha da connection factory são utilizadas se o aplicativo de chamada não fornecer um ID do usuário e uma senha explicitamente ao obter uma conexão.

Tipo de Dado Seqüência de Caracteres
Senha

Especifica uma senha criptografada. Se esse campo for preenchido, o campo Senha da caixa Propriedades será ignorado.

Se você especificar um valor para a propriedade UserName, você também deve especificar um valor para a propriedade Senha.

Tipo de Dado Seqüência de Caracteres
Digitar Senha Novamente

Confirma a senha.

Tipo

Uma lista drop-down de todas connectionFactoryInterfaces, conforme definição para as fábricas no Archive do Adaptador de Recursos.

Para cada Tipo, há um conjunto de propriedades especificado na caixa Propriedades. Esse conjunto de propriedades é construído pela recuperação das propriedades de cada objeto de definição de conexão. Para as connection factories existentes exibidas para atualização, essa lista de propriedades é sobreposta com as propriedades especificadas para os objetos. Quando o campo Tipo for alterado, as propriedades também são alteradas para refletir as propriedades corretas para o tipo.
