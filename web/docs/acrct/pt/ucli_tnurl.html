Configurando Novos Provedores de URL para Clientes Aplicativos

É possível criar provedores de URL e URLs para seu aplicativo cliente utilizando a ACRCT (Application Client Resource Configuration Tool).

Antes de Começar

Durante esta tarefa, você criará provedores URL e URLs para seu aplicativo cliente. Em uma tarefa administrativa separada, você deve instalar o código Java para o provedor URL necessário no computador cliente no qual o aplicativo cliente reside.

Sobre Esta Tarefa

Procedimento

  1. Inicie a ACRCT (Application Client Resource Configuration Tool).
  2. Abra o arquivo EAR para o qual deseja configurar o novo provedor URL. O conteúdo do arquivo EAR é exibido em uma exibição em árvore.
  3. Na árvore, selecione o arquivo JAR no qual deseja configurar o novo provedor de URL.
  4. Expanda o arquivo JAR para visualizar seu conteúdo.
  5. Clique na pasta chamada Provedores de URL. Conclua uma das ações a seguir:
    • Clique com o botão direito do mouse na pasta e selecione Novo.
    • Clique em Editar > Novo na barra de menus.
  6. Configure as propriedades do provedor de URL no diálogo de propriedades resultante.
  7. Clique em OK.
  8. Clique em Arquivo > Salvar na barra de menus para salvar suas alterações.


Links marcados (on-line) requerem acesso à Internet.
