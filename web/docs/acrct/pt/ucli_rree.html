Definições de Entradas de Ambiente de Recursos para Aplicativos Clientes

Utilize esta página para especificar as propriedades de entrada do ambiente de recursos.

Para visualizar essa página da ACRCT (Application Client Resource Configuration Tool), clique em Arquivo > Abrir. Após procurar por um arquivo EAR, clique em Abrir. Expanda o arquivo JAR selecionado > Provedores de Ambiente de Recursos > resource environment instance. Clique com o botão direito do mouse em Entradas do Ambiente de Recursos e clique em Novo. Os campos a seguir aparecem na guia Geral:

Nome

Especifica o nome administrativo para a entrada de ambiente de recursos.

Descrição

Especifica uma descrição da URL para seus registros administrativos.

Nome JNDI

Especifica o nome Java Naming and Directory Interface (JNDI) para o recurso, incluindo os subcontextos de nomenclatura.

Utilize este nome para ligar às informações de ligação da plataforma. A ligação associa os recursos definidos no descritor de implementação do módulo aos recursos reais (ou físicos) ligados à JNDI pela plataforma.

Propriedades Personalizadas

Especifica pares de nome-valor para definição de propriedades adicionais no objeto criado no tempo de execução deste recurso.

Você deve inserir um nome que é uma propriedade pública no objeto e um valor que pode ser convertido de cadeia para o tipo requerido pelo método de definição da propriedade. As propriedades e valores aceitáveis dependem do objeto que é criado. Consulte a documentação do objeto para obter uma lista de propriedades e valores válidos.
