Configurando Novas Entradas de Ambiente de Recursos para Clientes Aplicativos

É possível criar novas entradas do ambiente de recursos para seu aplicativo cliente utilizando a ACRCT (Application Client Resource Configuration Tool).

Antes de Começar

Durante esta tarefa, você criará novas entradas do ambiente de recursos para seu aplicativo cliente.

Sobre Esta Tarefa

Procedimento

  1. Inicie a ACRTCT (Application Client Resource Configuration Tool).
  2. Abra o arquivo EAR para o qual deseja configurar a nova entrada de ambiente de recursos. O conteúdo do arquivo EAR é exibido na exibição em árvore.
  3. Clique no provedor de ambiente de recursos desejado e conclua a ação a seguir para configurar novos provedores:
    • Configure um novo provedor de ambiente de recursos.
  4. Expanda o provedor do ambiente de recursos para visualizar a pasta Entradas do Ambiente de Recursos.
  5. Clique na pasta de entradas do ambiente de recursos e conclua uma das seguintes ações:
    • Clique com o botão direito do mouse na pasta e selecione Novo.
    • Clique em Editar > Novo na barra de menus.
  6. Configure as propriedades da entrada do ambiente de recursos nos campos exibidos.
  7. Clique em OK.
  8. Clique em Arquivo > Salvar na barra de menus para salvar suas alterações.


Links marcados (on-line) requerem acesso à Internet.
