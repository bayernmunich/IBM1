Configurando Novas Sessões de Correio para Clientes Aplicativos

Você pode utilizar a ACRCT (Application Client Resource Configuration Tool) para configurar novas sessões de correio para seu cliente aplicativo.

Antes de Começar

Durante esta tarefa, você configurará novas sessões de correio para seu aplicativo cliente. As sessões de correio estão associadas ao provedor de correio padrão pré-configurado, fornecido pelo produto.

Procedimento

  1. Inicie a ACRCT (Application Client Resource Configuration Tool) e abra o arquivo EAR. O conteúdo do arquivo EAR é exibido em uma visualização em árvore.
  2. Selecione o arquivo JAR no qual deseja configurar a nova sessão de JavaMail.
  3. Expanda o arquivo JAR para exibir seu conteúdo.
  4. Clique em Provedores de Correio > Provedor de Correio > Sessões de Correio. Conclua uma das ações a seguir:
    • Clique com o botão direito do mouse na pasta Sessões de Correio e selecione Novo Depósito de Informações do Provedor.
    • Clique em Editar > Novo na barra de menus.
  5. Configure as propriedades da Sessão de Correio nos campos exibidos.
  6. Clique em OK.
  7. Clique em Arquivo > Salvar na barra de menus para salvar suas alterações.


Links marcados (on-line) requerem acesso à Internet.
