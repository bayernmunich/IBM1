Configurando Objetos Administrados de Adaptadores de Recursos para o Cliente

Esta seção ajuda a configurar novos objetos administrados para o cliente.

Antes de Começar

Antes de configurar novos objetos administrados, você deve preencher os seguintes pré-requisitos:
  1. Instale o arquivo RAR (Resource Adapter Archive) utilizando a ferramenta clientRAR.
  2. Configure o adaptador de recursos para o arquivo .ear utilizando a ferramenta ACRCT (Application Client Resource Configuration Tool).

Sobre Esta Tarefa

Conclua esta tarefa para configurar novos objetos administrados para adaptadores de recursos instalados.

Procedimento

  1. Inicie a ACRCT (Application Client Resource Configuration Tool).
  2. Abra o arquivo EAR para o qual você deseja configurar novos objetos administrados. O conteúdo do arquivo EAR é exibido em uma exibição em árvore.
  3. Selecione o arquivo JAR no qual você deseja configurar os novos objetos administrados a partir da árvore.
  4. Expanda o arquivo JAR para exibir seu conteúdo.
  5. Clique na pasta Adaptadores de Recursos.
  6. Expanda o adaptador de recursos para o qual deseja criar objetos administrados.
  7. Clique com o botão direito do mouse na pasta Objetos Administrados e clique em Novo.
  8. Configure as propriedades do objeto administrado no diálogo de propriedades resultante.
  9. Clique em OK.
  10. Clique em Arquivo > Salvar na barra de menus para salvar suas alterações.


Links marcados (on-line) requerem acesso à Internet.
