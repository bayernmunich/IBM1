Configurando Provedores de Correio para Aplicativos Clientes

É possível editar as configurações de sessões e provedores de correio para o aplicativo cliente usando o Application Resource Configuration Tool (ACRCT).

Sobre Esta Tarefa

Use o Application Client Resource Configuration Tool (ACRCT) para editar as configurações de sessões e provedores de correio para os aplicativos clientes a serem usados.

Procedimento

  1. Inicie a ACRCT.
  2. Abra um arquivo EAR.
  3. Localize os objetos de correio na árvore que é exibida para o arquivo EAR. Por exemplo, se o arquivo contiver sessões de correio, expanda Recursos > application.jar > Provedores de Correio > java_mail_provider_instance > Sessões de Correio.

    Neste exemplo, java_mail_provider_instance é um provedor de correio específico.

Resultados

As instâncias da sessão de correio estão localizadas na pasta Sessões de JavaMail.


Links marcados (on-line) requerem acesso à Internet.
