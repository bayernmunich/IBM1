Configuration de nouveaux fournisseurs de sources de données (fournisseurs JDBC) pour les clients d'application

Vous pouvez créer de nouveaux fournisseurs de source de données (ou fournisseurs JDBC) pour votre client d'application à l'aide de l'outil ACRCT.

Avant de commencer

Cette tâche permet de créer des fournisseurs de sources de données (également désignés par fournisseurs JDBC) pour votre client d'application. Dans une tâche d'administration distincte, installez le code Java relatif au fournisseur de sources de données requis sur le poste client sur lequel le client d'application réside.

A propos de cette tâche

Cette tâche permet de connecter des clients d'application à des bases de données relationnelles.

Procédure

  1. Lancez l'outil ACRCT et ouvrez le fichier EAR pour lequel configurer le nouveau fournisseur de sources de données. Le contenu du fichier EAR apparaît dans une vue arborescente.
  2. Dans l'arborescence, sélectionnez le fichier JAR dans lequel le nouveau fournisseur de source de données doit être configuré.
  3. Développez le fichier JAR pour en afficher le contenu.
  4. Cliquez sur le dossier Fournisseurs de sources de données. Effectuez l'une des actions suivantes :
    • Cliquez avec le bouton droit de la souris sur le dossier, puis sélectionnez Nouveau fournisseur.
    • Cliquez sur Edition > Nouveau dans la barre de menus.
  5. Configurez les propriétés du fournisseur de source de données dans la boîte de dialogue de propriétés qui s'affiche.
  6. Une fois l'opération terminée, cliquez sur OK.
  7. Cliquez sur Fichier > Sauvegarder sur la barre de menus pour enregistrer les modifications.


Les liens marqués (en ligne) requièrent un accès à Internet.
