Propriétés de la source de données pour les clients d'application

Cette page permet de créer ou de modifier les sources de données.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR sélectionné > Fournisseurs de sources de données > Instance de fournisseur de sources de données. Cliquez avec le bouton droit de la souris sur Sources de données, puis cliquez sur Nouveau. Les zones ci-dessous sont affichées dans la page Général.

Nom

Indique le nom d'affichage de la source de données.

Type de données String (chaîne)
Description

Texte décrivant la source de données.

Type de données String (chaîne)
Nom JNDI

L'environnement d'exécution du client d'application utilise cette zone pour extraire les informations de configuration.

Nom de la base de données

Nom de la base de données à laquelle vous voulez vous connecter.

Utilisateur

ID utilisateur permettant avec la propriété Mot de passe l'authentification, si l'application appelante ne fournit pas d'ID utilisateur et de mot de passe.

Si vous indiquez une valeur pour la propriété ID utilisateur, vous devez également indiquer une valeur pour la propriété Mot de passe. L'ID utilisateur et le mot de passe de la fabrique de connexions sont utilisés si l'application appelante ne fournit pas d'ID utilisateur et de mot de passe.

Password

Emploie le mot de passe avec la propriété ID utilisateur permettant l'authentification si l'application appelante ne fournit pas expressément d'ID utilisateur et de mot de passe.

Si vous indiquez une valeur pour la propriété Mot de passe, vous devez également indiquer une valeur pour la propriété ID utilisateur.

Confirmation du mot de passe

Confirme le mot de passe.

Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.



Les liens marqués (en ligne) requièrent un accès à Internet.
