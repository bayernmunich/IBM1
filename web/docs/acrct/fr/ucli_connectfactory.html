Configuration de nouvelles fabriques de connexions pour les adaptateurs de ressource pour le client

Utilisez l'outil de configuration des ressources du client d'application (ACRCT) afin de configurer les nouvelles fabriques de connexions pour les adaptateurs de ressources pour le client.

A propos de cette tâche

Effectuez cette tâche pour configurer de nouvelles fabriques de connexions pour les adaptateurs de ressource.

Procédure

  1. Démarrez l'outil de configuration des ressources du client d'application (ACRCT).
  2. Ouvrez le fichier EAR pour lequel vous souhaitez configurer de nouvelles fabriques de connexions. Le contenu du fichier EAR apparaît dans une vue arborescente.
  3. Dans l'arborescence, sélectionnez le fichier JAR dans lequel vous souhaitez configurer les nouvelles fabriques de connexions.
  4. Développez le fichier JAR pour en afficher le contenu.
  5. Cliquez sur le dossier Adaptateurs de ressources.
  6. Développez l'arborescence de l'adaptateur de ressources pour lequel vous souhaitez créer des fabriques de connexions.
  7. Cliquez avec le bouton droit de la souris sur le dossier Fabriques de connexions, puis sélectionnez Nouveau.
  8. Configurez les propriétés de la fabrique de connexions dans la boîte de dialogue des propriétés qui s'affiche.
  9. Cliquez sur OK.
  10. Cliquez sur Fichier > Sauvegarder dans la barre de menus pour sauvegarder vos modifications.


Les liens marqués (en ligne) requièrent un accès à Internet.
