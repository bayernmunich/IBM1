Paramètres d'entrée d'environnement de ressources des clients d'application

Cette page permet de spécifier les propriétés de saisie de l'environnement de ressource.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR sélectionné > Fournisseurs d'environnement de ressources > instance_environnement_ressources. Cliquez avec le bouton droit de la souris sur Entrée d'environnement de ressource et cliquez sur Nouveau. Les zones suivantes apparaissent dans l'onglet Général :

Nom

Spécifie le nom administratif de l'entrée d'environnement de la ressource.

Description

Spécifie une description de l'URL pour vos enregistrements administratifs.

Nom JNDI

Désigne le nom JNDI (Java naming and Directory Interface) qui doit être associé à la ressource, y compris les éventuels sous-contextes de nommage.

Utilisez ce nom pour établir un lien avec les informations de liaison de la plateforme. La liaison associe les ressources définies dans le descripteur de déploiement du module aux ressources réelles (ou physiques) liées dans JNDI par la plateforme.

Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.
