Paramètres du fournisseur de messagerie des clients d'application

Cette page permet d'implémenter l'API JavaMail et de créer des sessions JavaMail.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Agrandissez la fenêtre du fichier JAR. Cliquez avec le bouton droit de la souris sur Fournisseurs de messagerie > et sélectionnez Nouveau. Les zones suivantes apparaissent dans l'onglet Général :

Nom

Nom du fournisseur de ressources JavaMail.

Description

(Facultatif) Décrit le fournisseur de ressources.

Chemin d'accès aux classes

Liste des chemins d'accès ou des noms de fichiers JAR qui indiquent l'emplacement des classes du fournisseur de ressources.

Protocole

Indique le nom du protocole.

Nom de classe

Indique le nom de la classe qui implémente le protocole. Ne renseignez pas cette zone si vous voulez utiliser l'implémentation par défaut.

Type

Ce menu contient l'une des deux valeurs suivantes : TRANSPORT ou STOCKAGE.

Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.
