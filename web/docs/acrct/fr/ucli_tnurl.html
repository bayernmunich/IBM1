Configuration de nouveaux fournisseurs d'URL pour les clients d'application

Vous pouvez créer des fournisseurs d'URL et des URL pour votre client d'application à l'aide de l'outil de configuration des ressources du client d'application (ACRCT).

Avant de commencer

Cette tâche consiste à créer des fournisseurs d'URL et des URL pour l'application client. Vous devez installer, dans une tâche administrative distincte, le code Java du fournisseur d'URL sur la machine sur laquelle réside l'application client.

A propos de cette tâche

Procédure

  1. Démarrez l'outil ACRCT (Application Client Resource Configuration Tool).
  2. Ouvrez le fichier EAR pour lequel le nouveau fournisseur d'URL doit être configuré. Le contenu du fichier EAR apparaît dans une vue arborescente.
  3. Dans l'arborescence, sélectionnez le fichier JAR dans lequel configurer le nouveau fournisseur d'URL.
  4. Développez le fichier JAR pour en afficher le contenu.
  5. Cliquez sur le dossier intitulé Fournisseurs d'URL. Effectuez l'une des actions suivantes :
    • Cliquez à l'aide du bouton droit de la souris sur le dossier et sélectionnez Nouveau.
    • Cliquez sur Edition > Nouveau dans la barre de menus.
  6. Configurez les propriétés du fournisseur d'URL dans la boîte de dialogue des propriétés qui s'affiche.
  7. Cliquez sur OK.
  8. Cliquez sur Fichier > Sauvegarder dans la barre de menus pour sauvegarder vos modifications.


Les liens marqués (en ligne) requièrent un accès à Internet.
