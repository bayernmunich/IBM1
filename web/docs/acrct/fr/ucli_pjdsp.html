Paramètres des destinations JMS génériques des clients d'application

Ce panneau permet de visualiser ou de modifier les propriétés de configuration de la destination JMS sélectionnée à utiliser avec le fournisseur JMS associé.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR sélectionné > Fournisseurs de messagerie > nouvelle instance fournisseur JMS. Cliquez avec le bouton droit de la souris sur Destinations et sélectionnez Nouveau. Les zones ci-dessous sont affichées dans la page Général.

Une destination JMS permet de configurer les propriétés d'une destination JMS du fournisseur JMS générique associé. Les connexions à la destination JMS sont créées par la fabrique de connexions JMS associée. La destination JMS à utiliser avec un fournisseur JMS générique (autre que le fournisseur de messagerie par défaut ou le fournisseur JMS WebSphere MQ) possède les propriétés ci-après.

Nom

Nom sous lequel cette file d'attente est connue dans WebSphere Application Server à des fins d'administration.

Type de données String (chaîne)
Description

Description de la file d'attente en vue de son administration.

Nom JNDI

Nom JNDI de la destination JMS réelle (physique) liée dans JNDI.

Nom JNDI externe

Nom JNDI utilisé pour lier la file d'attente dans l'espace de nom du serveur d'applications.

Par convention, utilisez le nom JNDI complet, par exemple sous la forme jms/Nom, Nom correspondant au nom logique de la ressource.

Ce nom est utilisé pour lier les informations relatives à la liaison des plateformes. La liaison associe les ressources définies par le descripteur de déploiement du module aux ressources réelles (physiques) liées à JNDI par la plateforme.

Type de données String (chaîne)
Type de destination

Indique si cette destination JMS est une file d'attente (pour système point à point) ou un sujet (pour la publication ou l'abonnement).

Sélectionnez l'une des options suivantes :
File d'attente
Destination JMS de file d'attente pour système de messagerie point-à-point
Sujet
Destination de sujet JMS pour la messagerie publication/abonnement.
Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.
