Paramètres du fournisseur de sources de données pour les clients d'application

Cette page permet de créer une source de données sous un fournisseur JDBC qui fournit la classe d'implémentation du pilote JDBC spécifique.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Agrandissez la fenêtre du fichier JAR. Cliquez avec le bouton droit sur Fournisseurs de source de données > et sélectionnez Nouveau. Les zones suivantes apparaissent dans l'onglet Général :

Nom

Indique le nom d'affichage de la source de données.

Par exemple, vous pouvez y entrer la valeur Source de données test.

Type de données String (chaîne)
Description

Indique un texte décrivant la ressource.

Type de données String (chaîne)
Chemin d'accès aux classes

Liste des chemins d'accès ou des noms de fichier .jar correspondant à l'emplacement des classes du fournisseur de ressources.

Classe d'implémentation

Ce paramètre permet d'effectuer des fonctions spécifiques à la base de données.

Type de données String (chaîne)
Portée par défaut Dépend de la classe d'implémentation du pilote JDBC
Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.
