Paramètres de la fabrique de connexions de l'adaptateur de ressources

Utilisez ce panneau pour visualiser ou modifier les propriétés de configuration de la fabrique de connexions d'adaptateur de ressources sélectionnée.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR sélectionné > Adaptateurs de ressources. Cliquez avec le bouton droit de la souris sur le dossier Fabriques de connexions puis sélectionnez Nouveau. Les zones suivantes apparaissent dans l'onglet Général :

Nom

Nom sous lequel la fabrique de connexions est connue dans WebSphere Application Server en vue de son administration. Il doit être unique parmi les fabriques de connexions des adaptateurs de ressources, sur l'ensemble du domaine d'administration du produit.

Type de données String (chaîne)
Description

Description facultative de cette fabrique de connexions dans IBM WebSphere Application Server en vue de son administration.

Type de données String (chaîne)
Nom JNDI

Nom JNDI utilisé pour associer cette fabrique de connexions d'adaptateur de ressources au descripteur de déploiement. Cette entrée doit être un nom resource-ref.

Type de données String (chaîne)
Nom de l'utilisateur

Nom d'utilisateur utilisé avec la propriété Mot de passe pour l'authentification si l'application appelante ne fournit pas explicitement un id utilisateur et un mot de passe lors de l'établissement d'une connexion. Si cette zone est utilisée, la zone de propriété Nom d'utilisateur est ignorée.

Si vous indiquez une valeur pour la propriété Nom d'utilisateur, vous devez également indiquer une valeur pour la propriété Mot de passe.

Les propriétés Nom d'utilisateur et Mot de passe de la fabrique de connexions sont utilisées lorsque l'application appelante n'indique pas explicitement d'id utilisateur et de mot de passe lors de l'établissement d'une connexion.

Type de données String (chaîne)
Password

Indique un mot de passe chiffré. Si vous renseignez cette zone, la zone Mot de passe de la boîte Propriétés est ignorée.

Si vous indiquez une valeur pour la propriété Nom d'utilisateur, vous devez également indiquer une valeur pour la propriété Mot de passe.

Type de données String (chaîne)
Confirmation du mot de passe

Confirme le mot de passe.

Type

Liste déroulante répertoriant tous les objets connectionFactoryInterfaces tels que définis pour les fabriques dans le fichier archive de l'adaptateur de ressource (fichier RAR).

Pour chaque type, un jeu de propriétés est spécifié dans la zone Propriétés. Il est composé des propriétés extraites de chaque objet de définition de connexion. Si des fabriques de connexions existantes sont affichées en vue de leur mise à jour, cette liste de propriétés est remplacée par les propriétés spécifiées pour les objets. Lorsque vous sélectionnez une entrée différente dans la liste Type, ses propriétés sont immédiatement affichées à la place de celles de l'entrée précédemment sélectionnée.

Type de données String (chaîne)

Nom du fichier : ucli_connectfact.html
