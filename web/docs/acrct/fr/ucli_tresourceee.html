Configuration de nouvelles entrées d'environnement de ressources pour les clients d'application

Vous pouvez créer de nouvelles entrées d'environnement de ressources pour votre client d'application à l'aide de l'outil de configuration des ressources du client d'application (ACRCT).

Avant de commencer

Cette tâche permet de créer de nouvelles entrées d'environnement de ressource pour votre client d'application.

A propos de cette tâche

Procédure

  1. Démarrez l'outil de configuration des ressources du client d'application (ACRCT).
  2. Ouvrez le fichier EAR pour lequel la nouvelle entrée d'environnement de ressource doit être configurée. Le contenu du fichier EAR se trouve dans la vue d'arborescence affichée.
  3. Cliquez sur le fournisseur d'environnement de ressource de votre choix, puis effectuez l'action suivante qui permet de configurer les nouveaux fournisseurs :
    • Configurez un nouveau fournisseur d'environnement de ressource.
  4. Développez le fournisseur d'environnement de ressources pour afficher le dossier Entrées d'environnement de ressources.
  5. Cliquez sur le dossier des entrées d'environnement de ressources et effectuez l'une des opérations suivantes :
    • Cliquez à l'aide du bouton droit de la souris sur le dossier et sélectionnez Nouveau.
    • Cliquez sur Edition > Nouveau dans la barre de menus.
  6. Configurez les propriétés des entrées d'environnement de ressources dans les zones affichées.
  7. Cliquez sur OK.
  8. Cliquez sur Fichier > Sauvegarder dans la barre de menus pour sauvegarder vos modifications.


Les liens marqués (en ligne) requièrent un accès à Internet.
