Paramètres du fournisseur de l'environnement de ressource pour les clients d'application

Cette page permet de spécifier les propriétés de saisie de l'environnement de ressource.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR (fichier d'archive Java) sélectionné. Cliquez avec le bouton droit de la souris sur Fournisseurs d'environnement de ressources, puis cliquez sur Nouveau. Les zones suivantes sont affichées dans la page Général :

Nom

Spécifie le nom administratif du fournisseur d'environnement de ressource.

Description

Spécifie une description du fournisseur d'environnement de ressource pour vos enregistrements administratifs.

Chemin d'accès aux classes

Spécifie le chemin du fichier JAR qui contient les classes d'implémentation pour le fournisseur d'environnement de la ressource.

Propriétés personnalisées

Spécifie les paires nom-valeur permettant de définir des propriétés supplémentaires pour l'objet créé lors de l'exécution pour cette ressource.

Vous devez entrer un nom correspondant à une propriété publique dans l'objet et une valeur pouvant être convertie d'une chaîne vers le type requis par la méthode set de la propriété. Les valeurs et propriétés admises dépendent de l'objet créé. Pour obtenir la liste des valeurs et des propriétés valides, reportez-vous à la documentation de l'objet.



Les liens marqués (en ligne) requièrent un accès à Internet.
