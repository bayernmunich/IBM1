Paramètres des objets gérés

Ce panneau permet d'afficher ou de modifier les propriétés de configuration des objets gérés sélectionnés.

Pour afficher la page de l'outil de configuration des ressources du client d'application (ACRCT), cliquez sur Fichier > Ouvrir. Recherchez un fichier EAR et cliquez sur Ouvrir. Développez le fichier JAR sélectionné > Adaptateurs de ressources > instance_adaptateur_ressources. Cliquez avec le bouton droit sur Objets administrés, puis sélectionnez Nouveau. Les zones ci-après apparaissent dans l'onglet Général.

Les paramètres des objets administrés sont gérés de manière similaire à ceux des fabriques de connexions. Lorsque vous mettez à jour des objets administrés, utilisez les mêmes panneaux que ceux qui vous ont servi à les créer.

Nom

Nom sous lequel l'objet administré est connu dans IBM WebSphere Application Server en vue de son administration. Il doit être unique parmi les objets administrés des adaptateurs de ressources, sur l'ensemble du domaine d'administration du produit.

Type de données String (chaîne)
Description

Description facultative de cette fabrique de connexions dans IBM WebSphere Application Server en vue de son administration.

Type de données String (chaîne)
Nom JNDI

Cette entrée est un nom resource-env-ref, un nom message-destination-ref (si l'élément message-destination-ref n'a pas de lien) ou un lien message-destination.

Type de données String (chaîne)
Type

Liste déroulante répertoriant toutes les paires classe-interface des objets administrés, comme défini pour les objets admin du fichier RAR (archive de l'adaptateur de ressources).

Pour chaque type, un jeu de propriétés est spécifié dans la zone Propriétés. Il est composé des propriétés extraites de chaque définition d'objet administré. Si des objets administrés existants sont affichés en vue de leur mise à jour, cette liste de propriétés est remplacée par les propriétés spécifiées pour les objets. Lorsque vous sélectionnez une entrée différente dans la liste Type, ses propriétés sont immédiatement affichées à la place de celles de l'entrée précédemment sélectionnée.

Type de données String (chaîne)

Nom du fichier : ucli_adminobj.html
