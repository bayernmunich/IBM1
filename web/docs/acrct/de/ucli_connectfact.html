Einstellungen für die Verbindungs-Factorys für Ressourcenadapter

Verwenden Sie diese Anzeige, um die Konfigurationseigenschaften der ausgewählten Verbindungs-Factory für Ressourcenadapter anzuzeigen oder zu ändern.

Klicken Sie zum Anzeigen dieser ACRCT-Seite (Application Client Resource Configuration Tool) auf Datei > Öffnen. Wenn Sie die EAR-Datei gefunden haben, klicken Sie auf Öffnen. Klicken Sie auf die ausgewählte JAR-Datei und dann auf Ressourcenadapter. Klicken Sie mit der rechten Maustaste auf den Ordner Verbindungs-Factorys, und klicken Sie dann auf Neu. Die folgenden Felder erscheinen unter dem Register Allgemein.

Name

Der Name, unter dem diese Verbindungs-Factory für Verwaltungszwecke in WebSphere Application Server geführt ist. Der Name muss innerhalb der Verbindungs-Factorys für Ressourcenadapter in der Administrationsdomäne des Produkts eindeutig sein.

Datentyp String
Beschreibung

Eine optionale Beschreibung der Verbindungs-Factory für Verwaltungszwecke in IBM WebSphere Application Server.

Datentyp String
JNDI-Name

Der JNDI-Name, der verwendet wird, um diese Verbindungs-Factory für Ressourcenadapter dem Implementierungsdeskriptor zuzuordnen. Dieser Eintrag muss ein resource-ref-Name sein.

Datentyp String
Benutzername

Der Benutzername wird zusammen mit der Eigenschaft Kennwort für die Authentifizierung verwendet, wenn die aufrufende Anwendung beim Anfordern einer Verbindung nicht explizit eine Benutzer-ID und ein Kennwort angibt. Wenn Sie dieses Feld verwenden, wird das Eigenschaftsfeld Benutzername ignoriert.

Wenn Sie einen Wert für die Eigenschaft Benutzername angeben, müssen Sie auch einen Wert für die Eigenschaft Kennwort angeben.

Die Verbindungs-Factory-Eigenschaften Benutzername und Kennwort werden verwendet, wenn die aufrufende Anwendung beim Anfordern einer Verbindung nicht explizit eine Benutzer-ID und Kennwort angibt.

Datentyp String
Kennwort

Ein verschlüsseltes Kennwort. Wenn Sie dieses Feld ausfüllen, wird das Feld Kennwort im Fenster "Eigenschaften" ignoriert.

Wenn Sie einen Wert für die Eigenschaft Benutzername angeben, müssen Sie auch ein Kennwort angeben.

Datentyp String
Kennwort bestätigen

Bestätigt das Kennwort.

Typ

Eine Dropdown-Liste mit allen connectionFactoryInterfaces, die für die Factorys in der RAR-Datei (Resource Adapter Archive) definiert sind.

Für jeden Typ wird im Fenster "Eigenschaften" eine Reihe von Eigenschaften angegeben. Diese Eigenschaften werden aus den Definitionsobjekten der einzelnen Verbindungen abgerufen. Bei allen vorhandenen Verbindungs-Factorys, die zur Aktualisierung angezeigt werden, wird diese Eigenschaftsliste mit den für die Objekte definierten Werten überschrieben. Wenn das Feld Typ geändert wird, werden die entsprechenden Eigenschaften für diesen Typ angezeigt.

Datentyp String
