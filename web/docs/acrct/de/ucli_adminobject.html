Verwaltete Objekte konfigurieren

Dieser Artikel unterstützt Sie beim Konfigurieren neuer verwalteter Objekte.

Vorbereitungen

Damit Sie neue verwaltete Objekte konfigurieren können, müssen Sie die folgenden Schritte ausführen:
  1. Installieren Sie die RAR-Datei (Resource Adapter Archive) mit dem Tool clientRAR.
  2. Konfigurieren Sie den Ressourcenadapter für die Datei .ear mit Application Client Resource Configuration Tool (ACRCT).

Informationen zu dieser Task

Führen Sie diese Task aus, um neue verwaltete Objekte für installierte Ressourcenadapter zu konfigurieren.

Prozedur

  1. Starten Sie das Application Client Resource Configuration Tool (ACRCT).
  2. Öffnen Sie die EAR-Datei, für die Sie neue verwaltete Objekte konfigurieren möchten. Der Inhalt der EAR-Datei wird in einer Baumstruktursicht angezeigt.
  3. Wählen Sie in der Baumstruktur die JAR-Datei aus, in der Sie die neuen verwalteten Objekte konfigurieren möchten.
  4. Erweitern Sie die JAR-Datei, um den Dateiinhalt anzuzeigen.
  5. Klicken Sie auf den Ordner Ressourcenadapter.
  6. Klicken Sie auf den Ressourcenadapter, für den Sie verwaltete Objekte erstellen möchten.
  7. Klicken Sie mit der rechten Maustaste auf den Ordner Verwaltete Objekte, und klicken Sie dann auf Neu.
  8. Konfigurieren Sie in dem darauf angezeigten Eigenschaftendialog die Eigenschaften für das verwaltete Objekt.
  9. Klicken Sie auf OK.
  10. Klicken Sie in der Menüleiste auf Datei > Speichern, um Ihre Änderungen zu speichern.


Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
