Mail-Provider und Mail-Sitzungen für Anwendungsclients konfigurieren

Sie können die Konfigurationen von Mail-Sitzungen und Providern für Ihre Anwendungsclients mit Application Client Resource Configuration Tool (ACRCT) bearbeiten.

Informationen zu dieser Task

Verwenden Sie das ACRCT (Application Client Resource Configuration Tool), um die Konfigurationen von Mail-Sitzungen und -Providern, die von Ihren Anwendungsclients verwendet werden sollen, zu bearbeiten.

Prozedur

  1. Starten Sie ACRCT.
  2. Öffnen Sie eine EAR-Datei.
  3. Suchen Sie die Mail-Objekte in der Baumstruktur, die für die EAR-Datei angezeigt werden. Wenn Ihre Datei beispielsweise Mail-Sitzungen enthält, klicken Sie auf Ressourcen > Anwendung.jar > Mail-Provider > Instanz_des_Java-Mail-Providers > Mail-Sitzungen.

    In diesem Beispiel steht Instanz_des_Java-Mail-Providers für einen bestimmten Mail-Provider.

Ergebnisse

Die Mail-Sitzungsinstanzen befinden sich im Ordner JavaMail-Sitzungen.


Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
