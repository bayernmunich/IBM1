Einstellungen für verwaltete Objekte

Verwenden Sie diese Anzeige, um die Konfigurationseigenschaften der ausgewählten verwalteten Objekte anzuzeigen oder zu ändern.

Klicken Sie zum Anzeigen dieser ACRCT-Seite (Application Client Resource Configuration Tool) auf Datei > Öffnen. Wenn Sie die EAR-Datei gefunden haben, klicken Sie auf Öffnen. Klicken Sie auf die ausgewählte JAR-Datei und anschließend auf Ressourcenadapter > Ressourcenadapterinstanz. Klicken Sie mit der rechten Maustaste auf Verwaltete Objekte, und klicken Sie dann auf Neu. Die folgenden Felder erscheinen unter dem Register Allgemein.

Die Einstellungen für verwaltete Objekte werden ähnlich wie Verbindungs-Factorys behandelt. Für die Aktualisierung verwalteter Objekte verwenden Sie dieselben Anzeigen wie beim Erstellen verwalteter Objekte.

Name

Der Name, unter dem dieses verwaltete Objekt zu Verwaltungszwecken in IBM WebSphere Application Server bekannt ist. Der Name muss innerhalb der verwalteten Objekte für Ressourcenadapter in der Administrationsdomäne des Produkts eindeutig sein.

Datentyp String
Beschreibung

Eine optionale Beschreibung der Verbindungs-Factory für Verwaltungszwecke in IBM WebSphere Application Server.

Datentyp String
JNDI-Name

Dieser Eintrag ist ein resource-env-ref-Name, ein message-destination-ref-Name (falls die message-destination-ref keinen Link hat) oder ein message-destination-Link.

Datentyp String
Typ

Eine Dropdown-Liste mit allen verwalteten Objekt/Klassenschnittstelle-Paaren, die für die Verwaltungsobjekte in der RAR-Datei (Resource Adapter Archive) definiert sind.

Für jeden Typ wird im Fenster "Eigenschaften" eine Reihe von Eigenschaften angegeben. Diese Eigenschaftengruppe wird aus den Eigenschaften gebildet, die aus der Definition jedes verwalteten Objekts abgerufen werden. Bei allen vorhandenen verwalteten Objekten, die zur Aktualisierung angezeigt werden, wird diese Eigenschaftenliste mit den für die Objekte definierten Werten überschrieben. Wenn das Feld Typ geändert wird, werden die entsprechenden Eigenschaften für diesen Typ angezeigt.

Datentyp String

Dateiname: ucli_adminobj.html
