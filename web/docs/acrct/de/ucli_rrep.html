Einstellungen der Ressourcenumgebungsprovider für Anwendungsclients

Auf dieser Seite können Sie die Eigenschaften von Einträgen der Ressourcenumgebung angeben.

Klicken Sie zum Anzeigen dieser ACRCT-Seite (Application Client Resource Configuration Tool) auf Datei > Öffnen. Wenn Sie die EAR-Datei gefunden haben, klicken Sie auf Öffnen. Erweitern Sie die ausgewählte JAR-Datei. Klicken Sie mit der rechten Maustaste auf Ressourcenumgebungsprovider und dann auf Neu. Die folgenden Felder werden auf der Registerkarte Allgemein angezeigt:

Name

Gibt den Administrationsnamen für den Ressourcenumgebungsprovider an.

Beschreibung

Gibt eine Beschreibung des Ressourcenumgebungsproviders für die Verwaltungsdatensätze an.

Klassenpfad

Gibt den Pfad der JAR-Datei an, die die Implementierungsklasse für den Ressourcenumgebungsprovider enthält.

Angepasste Eigenschaften

Gibt Name/Wert-Paare für die Definition zusätzlicher Eigenschaften des Objekts an, das zur Laufzeit für diese Ressource erstellt wird.

Sie müssen einen Namen eingeben, der eine öffentliche Eigenschaft des Objekts ist. Außerdem müssen Sie einen Wert eingeben, der von einer Zeichenfolge in den für die Setter-Methode der Eigenschaft erforderlichen Typ konvertiert werden kann. Die gültigen Eigenschaften und Werte sind von dem zu erstellenden Objekt abhängig. Eine Liste der gültigen Eigenschaften und Eigenschaftswerte finden Sie in der Objektdokumentation.



Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
