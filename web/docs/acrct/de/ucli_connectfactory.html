Neue Verbindungs-Factorys für Ressourcenadapter konfigurieren

Verwenden Sie das Application Client Resource Configuration Tool (ACRCT), um neue Verbindungs-Factorys für Ressourcenadapter zu konfigurieren.

Informationen zu dieser Task

Führen Sie diese Task aus, um neue Verbindungs-Factorys für Ressourcenadapter zu konfigurieren.

Prozedur

  1. Starten Sie das Application Client Resource Configuration Tool (ACRCT).
  2. Öffnen Sie die EAR-Datei, für die Sie neue Verbindungs-Factorys konfigurieren möchten. Der Inhalt der EAR-Datei wird in einer Baumstruktursicht angezeigt.
  3. Wählen Sie in der Baumstruktur die JAR-Datei aus, in der Sie die neue Verbindungs-Factory konfigurieren möchten.
  4. Erweitern Sie die JAR-Datei, um den Dateiinhalt anzuzeigen.
  5. Klicken Sie auf den Ordner Ressourcenadapter.
  6. Klicken Sie auf den Ressourcenadapter, für den Sie Verbindungs-Factorys erstellen möchten.
  7. Klicken Sie mit der rechten Maustaste auf den Ordner Verbindungs-Factorys, und klicken Sie dann auf Neu.
  8. Konfigurieren Sie die Eigenschaften für die Verbindungs-Factory im daraufhin angezeigten Eigenschaftendialog.
  9. Klicken Sie auf OK.
  10. Klicken Sie in der Menüleiste auf Datei > Speichern, um Ihre Änderungen zu speichern.


Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
