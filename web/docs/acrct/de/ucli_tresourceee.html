Neue Ressourcenumgebungseinträge für Anwendungsclients konfigurieren

Sie können neue Ressourcenumgebungseinträge für Ihre Clientanwendung mit Application Client Resource Configuration Tool (ACRCT) erstellen.

Vorbereitungen

Mit dieser Task erstellen Sie neue Ressourcenumgebungseinträge für Ihre Clientanwendung.

Informationen zu dieser Task

Prozedur

  1. Starten Sie das Application Client Resource Configuration Tool (ACRCT).
  2. Öffnen Sie die EAR-Datei, für die Sie den neuen Ressourcenumgebungseintrag konfigurieren möchten. Der Inhalt der EAR-Datei wird in einer Baumstruktursicht angezeigt.
  3. Klicken Sie auf den gewünschten Ressourcenumgebungsprovider, und führen Sie die folgende Aktion aus, um die neuen Provider zu konfigurieren:
    • Konfigurieren Sie einen neuen Ressourcenumgebungsprovider.
  4. Erweitern Sie den Eintrag für den Ressourcenumgebungseintrag, um den Ordner Ressourcenumgebungseinträge anzuzeigen.
  5. Klicken Sie auf den Ordner für Ressourcenumgebungseinträge, und führen Sie eine der folgenden Aktionen aus:
    • Klicken Sie mit der rechten Maustaste auf den Ordner, und wählen Sie dann Neu aus.
    • Klicken Sie in der Menüleiste auf Bearbeiten > Neu.
  6. Konfigurieren Sie in den angezeigten Feldern die Eigenschaften für den Ressourcenumgebungseintrag.
  7. Klicken Sie auf OK.
  8. Klicken Sie in der Menüleiste auf Datei > Speichern, um Ihre Änderungen zu speichern.


Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
