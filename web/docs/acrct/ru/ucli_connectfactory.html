Настройка новых фабрик соединений для адаптеров ресурсов

Для настройки фабрик соединений для адаптеров ресурсов используйте инструмент настройки ресурсов приложения-клиента (ACRCT).

О задаче

Выполните эту задачу для настройки новых фабрик соединений для адаптеров ресурсов.

Процедура

  1. Запустите инструмент настройки ресурсов приложения-клиента (ACRCT).
  2. Откройте файл EAR для которого требуется настроить новые фабрики соединений. Содержимое файла EAR будет показано в иерархическом списке.
  3. Выберите файл JAR в котором требуется настроить новые фабрики соединений из дерева.
  4. Разверните файл JAR для просмотра его содержимого.
  5. Щелкните на папке Адаптеры ресурсов.
  6. Разверните адаптер ресурса для которого требуется создать фабрики соединений.
  7. Щелкните правой кнопкой на папке Фабрики соединений и выберите Создать.
  8. Настройте свойства фабрики соединений в открывшемся диалоговом окне свойств.
  9. Нажмите OK.
  10. Выберите в строке меню пункт Файл > Сохранить для сохранения изменений.
