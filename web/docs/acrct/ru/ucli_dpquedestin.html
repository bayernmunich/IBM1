Параметры адресата очереди провайдера по умолчанию

Используйте эту панель для просмотра или изменения свойств выбранного назначения очереди JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Провайдеры обмена сообщениями > Провайдер по умолчанию. Щелкните правой кнопкой на пункте Адресаты очереди. Нажмите Создать. На вкладке Общие появятся следующие поля.

Имя

Имя фабрики адресатов очереди. Это поле должно быть заполнено.

Тип данных Строка
Описание

Описание данного адресата очереди для администрирования в WebSphere Application Server.

Тип данных Строка
Имя JNDI

Имя JNDI, используемое для сопоставления этого адресата с именем resource-env-ref файла описания.

Тип данных Строка
Имя очереди

Имя очереди.

Тип данных Строка
Режим доставки

Режим доставки для сообщений, отправленных по данному адресату.

Тип данных Строка
Диапазон Приложение, постоянное, временное
Значение по умолчанию Приложение
Время хранения в кэше

Период времени после отправки, в течение которого сообщение, отправленное по данному адресату, должно быть получено системой, где 0 указывает на бесконечны срок хранения. Если поле Время хранения в кэше не заполнено, используется значения поставщика.

Тип данных Целое число
Единицы измерения Миллисекунды
Приоритет

Приоритет сообщений, отправленных по данному адресату. Если это поле не заполнено, используется значение поставщика.

Тип данных Целое число
Диапазон От 0 до 9, где 0 - это самый низкий приоритет, а 9 - самый высокий
Упреждающее чтение

Контролирует оптимизацию упреждающего чтения при доставке сообщений.

Тип данных Строка
Диапазон Как соединение, всегда включен, всегда выключен
Значение по умолчанию Как соединение
