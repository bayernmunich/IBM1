Параметры фабрики соединений адаптера ресурса

Используйте эту панель для просмотра или изменения свойств конфигурации выбранной фабрики соединений адаптера ресурса.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Адаптеры ресурсов. Щелкните правой кнопкой на папке Фабрики соединений и выберите Создать. На вкладке Общие появятся следующие поля.

Имя

Имя, идентифицирующее эту фабрику соединений для администрирования в WebSphere Application Server. Имя должно быть уникальным среди фабрик соединений адаптера ресурса в административном домене продукта.

Тип данных Строка
Описание

Необязательное описание данной фабрики соединений для административных целей на сервере IBM WebSphere Application Server.

Тип данных Строка
Имя JNDI

Имя JNDI, используемое для сопоставления этого определения фабрики соединений адаптера ресурса с файлом описания. Эта запись должна быть именем resource-ref.

Тип данных Строка
Имя пользователя

Имя пользователя с Паролем, используемое для идентификации если вызывающее приложение не предоставляет явно указанные ИД пользователя и пароль при установке соединения. Если это поле используется, поле свойств Имя пользователя игнорируется.

Если указать значение свойства Имя пользователя, следует также задать значение свойства Пароль.

Свойства Имя пользователя и Пароль фабрики соединений используются если вызывающее приложение не предоставляет явно указанные ИД пользователя и пароль при установке соединения.

Тип данных Строка
Пароль

Указывает зашифрованный пароль. При заполнении этого пола, поле Пароль игнорируется.

Если указать значение свойства Имя пользователя, следует также задать значение свойства Пароль.

Тип данных Строка
Повторно укажите пароль

Подтверждение пароля.

Тип

Выпадающий список со всеми connectionFactoryInterfaces, заданными для фабрик в файле RAR адаптера ресурса.

Для каждого Типа имеется набор свойств, указанных в разделе Свойства. Этот набор свойств создается с помощью извлечения свойств из каждого объекта определения соединения. Для всех имеющихся фабрик соединений, отображаемых для обновления, этот список перекрывается свойствами, указанными для объекта. При изменении поля Тип свойства также меняются для отображения корректных свойств для данного типа.

Тип данных Строка

Имя файла: ucli_connectfact.html
