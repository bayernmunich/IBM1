Параметры провайдера источника данных для приложений-клиентов

С помощью этой страницы можно создать источник данных в провайдере JDBC, предоставляющем определенный класс реализации драйвера JDBC.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR. Щелкните правой кнопкой мыши на опции Провайдеры источников данных > и выберите Создать. На вкладке Общие появятся следующие поля:

Имя

Указывает отображаемое имя этого источника данных.

Например, можно задать в этом поле значение Проверить источник данных.

Тип данных Строка
Описание

Указывает текстовое описание ресурса.

Тип данных Строка
Путь к классам

Список путей или имен файлов .jar, вместе указывающих на расположение классов провайдера ресурсов.

Класс реализации

Используйте этот параметр для выполнения связанных с базой данных функций.

Тип данных Строка
Значение по умолчанию Зависит от класса реализации драйвера JDBC
Настраиваемые свойства

Указывает пары имя-значение для настройки дополнительных свойств объекта, создаваемого при выполнении этого ресурса.

Следует ввести имя, являющееся внешним свойством объекта, и значение, которое может быть преобразовано из строки в тип, необходимый методу set свойства. Допустимые свойства и значения зависят от создаваемого объекта. Список допустимых свойств и значений приведен в документации по объекту.
