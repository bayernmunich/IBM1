Провайдеры среды ресурса и записи среды ресурса

Указатель на среду ресурса привязывает логическое имя, используемое приложением-клиентом, к физическому имени объекта.

Не все объекты, привязанные к пространству имен JNDI сервера, предназначены для применения приложением-клиентом. Например, при выполнении клиента WebSphere Application Server использование объектов коннектора Java 2 (J2C) в клиенте не поддерживается. Объект должен быть удаленным, а клиентская реализация должна быть доступной в пути к классам выполнения приложения-клиента.

Указатели на среду ресурса отличаются от указателей на ресурсы. Указатели на среду ресурса позволяют приложению-клиенту использовать логическое имя для поиска привязки ресурса к пространству имен JNDI сервера. Указатель на ресурс позволяет вашему приложению использовать логическое имя для поиска локального ресурса J2EE. Спецификация J2EE не указывает определенную реализацию ресурса.
