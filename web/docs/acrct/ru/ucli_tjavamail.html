Настройка почтовых провайдеров и сеансов для приложений-клиентов

Вы можете изменить конфигурации почтовых сеансов и провайдеров для приложений-клиентов с помощью инструмента настройки ресурсов приложения-клиента (ACRCT).

О задаче

Используйте инструмент настройки ресурсов приложения-клиента (ACRCT) для изменения конфигураций почтовых сеансов и провайдеров для приложений-клиентов.

Процедура

  1. Запустите ACRCT.
  2. Откройте файл EAR.
  3. Найдите почтовые объекты в отображаемом для файла EAR иерархическом списке. Например, если файл содержит почтовые сеансы, разверните Ресурсы > приложение.jar > Почтовые провайдеры > экземпляр_почтового_провайдера_java > Почтовые сеансы.

    В этом примере экземпляр_почтового_провайдера_java представляет определенный провайдер JavaMail.

Результаты

Экземпляры почтового сеанса расположены в папке Сеансы JavaMail.
