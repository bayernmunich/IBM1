Параметры провайдера среды ресурса для приложений-клиентов

С помощью этой страницы можно указать свойства записи среды ресурса.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл архива Java (JAR). Щелкните правой кнопкой на опции Провайдеры среды ресурсов и выберите Создать. На вкладке Общие появятся следующие поля.

Имя

Указывает административное имя для провайдера среды ресурсов.

Описание

Указывает описание провайдера среды ресурсов для администрирования.

Путь к классам

Указывает путь к файлу JAR, содержащему классы реализации провайдера среды ресурсов.

Настраиваемые свойства

Указывает пары имя-значение для настройки дополнительных свойств объекта, создаваемого при выполнении этого ресурса.

Следует ввести имя, являющееся внешним свойством объекта, и значение, которое может быть преобразовано из строки в тип, необходимый методу set свойства. Допустимые свойства и значения зависят от создаваемого объекта. Список допустимых свойств и значений приведен в документации по объекту.
