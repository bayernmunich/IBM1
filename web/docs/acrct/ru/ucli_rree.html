Параметры записи среды ресурса для приложений-клиентов

С помощью этой страницы можно указать свойства записи среды ресурса.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Провайдеры среды ресурсов > экземпляр среды ресурсов. Щелкните правой кнопкой на опции Записи среды ресурсов и выберите Создать. На вкладке Общие появятся следующие поля:

Имя

Указывает административное имя для записи среды ресурсов.

Описание

Указывает описание URL для административных записей.

Имя JNDI

Указывает имя Java Naming and Directory Interface (JNDI) для ресурса, включая подконтексты имен.

используйте это имя для создания ссылки на сведения привязки платформы. Привязка связывает ресурсы, заданные в файле описания модуля, с фактическими (физическими) ресурсами, связанными с JNDI платформой.

Настраиваемые свойства

Указывает пары имя-значение для настройки дополнительных свойств объекта, создаваемого при выполнении этого ресурса.

Следует ввести имя, являющееся внешним свойством объекта, и значение, которое может быть преобразовано из строки в тип, необходимый методу set свойства. Допустимые свойства и значения зависят от создаваемого объекта. Список допустимых свойств и значений приведен в документации по объекту.
