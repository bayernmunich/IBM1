Параметры адресата темы провайдера по умолчанию

Используйте эту панель для просмотра или изменения свойств выбранных адресатов темы JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Провайдеры обмена сообщениями > Провайдер по умолчанию. Щелкните правой кнопкой на пункте Адресаты темы и нажмите кнопку Создать. На вкладке Общие появятся следующие поля.

Имя

Имя записи адресата очереди.

Тип данных Строка
Описание

Описание записи.

Тип данных Строка
Имя JNDI

Имя JNDI, используемое для сопоставления этого адресата с именем resource-env-ref файла описания.

Тип данных Строка
Пространство темы

Имя пространства темы. Это обязательное поле.

Тип данных Строка
Значение по умолчанию DEFAULT_TOPIC_SPACE
Имя темы

Имя темы. Это обязательное поле.

Тип данных Строка
Режим доставки

Режим по умолчанию для сообщений, отправленных по данному адресату.

Тип данных Строка
Диапазон Приложение, постоянное, временное
Значение по умолчанию Приложение
Время хранения в кэше

Период времени после отправки, в течение которого сообщение, отправленное по данному адресату, должно быть получено системой, где 0 указывает на бесконечны срок хранения. Если это поле не заполнено, используется значение поставщика.

Тип данных Длинный
Единицы измерения Миллисекунды
Приоритет

Приоритет сообщений, отправленных по данному адресату. Если это поле не заполнено, используется значение поставщика.

Тип данных Целое число
Диапазон От 0 до 9, где 0 - это самый низкий приоритет, а 9 - самый высокий
Упреждающее чтение

Контролирует оптимизацию упреждающего чтения при доставке сообщений.

Тип данных Строка
Диапазон Как соединение, всегда включен, всегда выключен
Значение по умолчанию Как соединение
