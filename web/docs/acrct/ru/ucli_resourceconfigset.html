Параметры адаптера ресурса

Используйте эту панель для просмотра и изменения свойств конфигурации адаптера ресурса. Эти свойства конфигурации контролируют процесс создания адаптеров ресурсов.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Адаптер ресурсов. Щелкните правой кнопкой на опции Адаптер ресурса и выберите Создать. На вкладке Общие появятся следующие поля.

Имя

Имя Адаптера ресурсов для административных целей на сервере IBM WebSphere Application Server. Имя должно быть уникальным среди адаптеров ресурсов в административном домене продукта.

Тип данных Строка
Описание

Описание данного адаптера ресурсов для административных целей на сервере IBM WebSphere Application Server.

Тип данных Строка
Путь к классам

Любой дополнительный путь к классам. Путь к каталогу адаптера ресурса добавляется автоматически.

Тип данных Строка
Значение по умолчанию Путь к каталогу адаптера ресурса.
Внутренний путь

Внутренний путь к расположению адаптера ресурса. Укажите дополнительный внутренний путь к классам.

Тип данных Строка
Имя адаптера ресурса

Обязательной имя, указывающее на подкаталог установленного адаптера ресурса. Эта запись не представляет полное имя каталога адаптера ресурса. Полное имя каталога является путем к установленному адаптеру ресурса с именем адаптера ресурса.

Тип данных Строка
Путь к установленному адаптеру ресурса

Каталог, в котором установлены адаптеры ресурсов. Если это поле не заполнено, используется значение по умолчанию.

Если указано значение ${CONNECTOR_INSTALL_ROOT}, оно замещает значение переменной CLIENT_CONNECTOR_INSTALL_ROOT в системе, в которой выполняется приложение-клиент. Это позволяет выполнять приложение в разных системах, так как установка клиента может быть выполнена в различных расположениях.

Тип данных Строка
Значение по умолчанию ${CONNECTOR_INSTALL_ROOT}
