Настройка новых записей среды ресурсов для приложений-клиентов

С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно создать новые записи среды ресурсов для приложения-клиента.

Перед тем, как начать

В ходе выполнения этой задачи вы создадите новые записи среды ресурса для приложения-клиента.

О задаче

Процедура

  1. Запустите инструмент настройки ресурсов приложения-клиента (ACRCT).
  2. Откройте файл EAR в котором требуется настроить новую запись среды ресурса. Содержимое файла EAR будет показано в иерархическом списке навигации.
  3. Щелкните на нужном провайдере среды ресурса и выполните следующее действие для настройки новых провайдеров:
    • Настройка нового провайдера среды ресурса.
  4. Разверните провайдер среды ресурса для просмотра папки Записи среды ресурса.
  5. Щелкните на папке записей среды ресурса и выполните одно из следующих действий:
    • Щелкните правой кнопкой на папке и выберите Создать.
    • Выберите в меню пункт Изменить > Создать.
  6. Настройте свойства записи среды ресурса в отображаемых полях.
  7. Нажмите OK.
  8. Выберите в меню пункт Файл > Сохранить для сохранения изменений.
