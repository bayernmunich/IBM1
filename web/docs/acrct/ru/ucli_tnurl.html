Настройка новых провайдеров URL для приложений-клиентов

С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно создать провайдеры URL и URL для приложения-клиента.

Перед тем, как начать

В ходе выполнения этой задачи вы создадите провайдеры URL и URL для приложения-клиента. В отдельной задаче администрирования следует установить код Java для требуемого провайдера URL в клиентской системе, в которой находится приложение-клиент.

О задаче

Процедура

  1. Запустите инструмент настройки ресурсов приложения-клиента ACRCT.
  2. Откройте файл EAR для которого требуется настроить новый провайдер URL. Содержимое файла EAR будет показано в иерархическом списке.
  3. Выберите файл JAR в котором требуется настроить новый провайдер URL из иерархического списка.
  4. Разверните файл JAR для просмотра его содержимого.
  5. Щелкните на папке с именем Провайдеры URL. Выполните одно из следующих действий:
    • Щелкните на папке правой кнопкой и выберите Создать.
    • Выберите в меню пункт Правка > Создать.
  6. Настройте свойства провайдера URL в открывшемся окне свойств.
  7. Нажмите OK.
  8. Выберите в строке меню пункт Файл > Сохранить для сохранения изменений.
