Настройка администрируемых объектов

Этот раздел помогает настроить новые администрируемые объекты.

Перед тем, как начать

Перед настройкой новых администрируемых объектов следует выполнить следующие предварительные требования:
  1. Установите файл RAR адаптера ресурса с помощью утилиты clientRAR.
  2. Настройте адаптер ресурса для файла .ear с помощью инструмента настройки ресурсов приложения-клиента ACRCT.

О задаче

Выполните эту задачу для настройки новых администрируемых объектов для установленных адаптеров ресурсов.

Процедура

  1. Запустите инструмент настройки ресурсов приложения-клиента ACRCT.
  2. Откройте файл EAR, для которого требуется настроить новые администрируемые объекты. Содержимое файла EAR будет показано в иерархическом списке.
  3. Выберите файл JAR в котором требуется настроить новые администрируемые объекты из дерева.
  4. Разверните файл JAR для просмотра его содержимого.
  5. Щелкните на папке Адаптеры ресурсов.
  6. Разверните адаптер ресурса для которого требуется создать администрируемые объекты.
  7. Щелкните правой кнопкой на папке Администрируемые объекты и выберите Создать.
  8. Настройте свойства администрируемого объекта в открывшемся диалоговом окне свойств.
  9. Нажмите OK.
  10. Выберите в строке меню пункт Файл > Сохранить для сохранения изменений.
