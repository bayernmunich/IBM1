Параметры почтового провайдера для приложений-клиентов

Используйте эту страницу для реализации JavaMail API и создания почтовых сеансов.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR. Щелкните правой кнопкой мыши на опции Почтовые провайдеры > и выберите Создать. На вкладке Общие появятся следующие поля:

Имя

Имя провайдера ресурса JavaMail.

Описание

Необязательное описание провайдера ресурса.

Путь к классам

Указывает список путей или имен файлов JAR, вместе образующих расположение классов провайдера ресурса.

Протокол

Указывает имя протокола.

Имя класса

Указывает имя класса, реализующего протокол. Оставьте это поле пустым если требуется использовать реализацию по умолчанию.

Тип

Это меню содержит следующие два значение: TRANSPORT или STORE.

Настраиваемые свойства

Указывает пары имя-значение для настройки дополнительных свойств объекта, создаваемого при выполнении этого ресурса.

Следует ввести имя, являющееся внешним свойством объекта, и значение, которое может быть преобразовано из строки в тип, необходимый методу set свойства. Допустимые свойства и значения зависят от создаваемого объекта. Список допустимых свойств и значений приведен в документации по объекту.
