Инструмент настройки ресурсов приложения-клиента

Перейти к разделам справки из этой главы.

  • Настройка администрируемых объектов
    Этот раздел помогает настроить новые администрируемые объекты.
  • Настройка новых записей среды ресурсов для приложений-клиентов
    С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно создать новые записи среды ресурсов для приложения-клиента.
  • Настройка новых почтовых сеансов для приложений-клиентов
    С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно настроить новые почтовые сеансы для приложения-клиента.
  • Настройка новых провайдеров URL для приложений-клиентов
    С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно создать провайдеры URL и URL для приложения-клиента.
  • Настройка новых провайдеров источников данных (провайдеров JDBC) для приложений-клиентов
    Вы можете создать новый провайдеры источников данных, также называемых провайдерами JDBC, для ваших приложений-клиентов с помощью инструмента настройки ресурса приложения-клиента (ACRCT).
  • Настройка новых провайдеров среды ресурсов для приложений-клиентов
    С помощью инструмента настройки ресурсов приложения-клиента (ACRCT) можно создать новые конфигурации провайдера среды ресурсов.
  • Настройка почтовых провайдеров и сеансов для приложений-клиентов
    Вы можете изменить конфигурации почтовых сеансов и провайдеров для приложений-клиентов с помощью инструмента настройки ресурсов приложения-клиента (ACRCT).
  • Настройка ресурсов клиента обмена сообщениями Java
    Для настройки ресурсов клиента обмена сообщениями Java вы создадите новые конфигурации провайдера JMS для приложения-клиента. Приложение-клиент может использовать службу обмена сообщениями с помощью API Java Message Service. Провайдер JMS предоставляет два типа фабрик J2EE. Одним из них является Фабрика соединений JMS, а вторым - фабрика адресатов JMS.
  • Обновление WebSphere MQ, выполняющего роль провайдера Java Message Service, и его конфигураций ресурсов JMS для приложений-клиентов
    Вы можете обновить имеющуюся конфигурацию WebSphere MQ, выполняющего роль провайдера Java Message Service (JMS) и обновить конфигурацию фабрик соединений WebSphere MQ или адресатов WebSphere MQ.
  • Адаптеры ресурсов для клиента
    Адаптер ресурсов является драйвером программного обеспечения уровня системы, который используется приложением на Java для подключения к информационной системе предприятия (EIS). Адаптер ресурсов встраивается в приложение-клиент и обеспечивает возможность соединения между EIS и приложением J2EE.
  • Асинхронный обмен сообщениями в WebSphere Application Server с помощью JMS
    WebSphere Application Server поддерживает асинхронный обмен сообщениями в качестве метода связи на основании интерфейса программирования Java Message Service (JMS). Интерфейс JMS предоставляет программам на Java (клиентам и приложениям Java Platform, Enterprise Edition (Java EE)) стандартный способ создания, отправки, получения и чтения асинхронных запросов, таких как JMS.
  • Провайдеры Java Message Service (JMS) для клиентов
    Этот раздел содержит описание различных способов применения приложением-клиентом провайдеров Java Message Service (JMS) с WebSphere Application Server.
  • Провайдеры среды ресурса и записи среды ресурса
    Указатель на среду ресурса привязывает логическое имя, используемое приложением-клиентом, к физическому имени объекта.
  • URLs for application clients
    A Uniform Resource Locator (URL) is an identifier that points to an electronically accessible resource, such as a directory file on a machine in a network, or a document stored in a database.
  • Настройка новых фабрик соединений для адаптеров ресурсов
    Для настройки фабрик соединений для адаптеров ресурсов используйте инструмент настройки ресурсов приложения-клиента (ACRCT).
  • Параметры URL для приложений-клиентов
    С помощью этой страницы можно реализовать функцию для определенного протокола URL, например для протокола передачи гипертекстовой информации (HTTP).
  • Параметры адаптера ресурса
    Используйте эту панель для просмотра и изменения свойств конфигурации адаптера ресурса. Эти свойства конфигурации контролируют процесс создания адаптеров ресурсов.
  • Параметры администрируемых объектов
    Используйте эту панель для просмотра или изменения свойств конфигурации выбранных администрируемых объектов.
  • Параметры адресата очереди провайдера по умолчанию
    Используйте эту панель для просмотра или изменения свойств выбранного назначения очереди JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server.
  • Параметры адресата очереди провайдера по умолчанию версии 5 для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранного адресата очереди для применения с провайдером JMS (Java Message Service) для продукта.
  • Параметры адресата темы провайдера WebSphere MQ для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранных адресатов тем для применения с провайдером Java Message Service (JMS) продукта WebSphere MQ.
  • Параметры адресата темы провайдера по умолчанию
    Используйте эту панель для просмотра или изменения свойств выбранных адресатов темы JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server.
  • Параметры адресата темы провайдера по умолчанию версии 5 для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранных адресатов тем для применения с внутренним провайдером JMS (Java Message Service).
  • Параметры записи среды ресурса для приложений-клиентов
    С помощью этой страницы можно указать свойства записи среды ресурса.
  • Параметры назначения очереди провайдера WebSphere MQ для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранного назначения очереди для применения с провайдером Java Message Service (JMS) продукта WebSphere MQ.
  • Параметры общего адресата JMS для приложений клиентов
    Используйте эту панель для просмотра или изменения свойств конфигурации выбранного адресата JMS для применения со связанным провайдером JMS.
  • Параметры общей фабрики соединений JMS для приложений клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранной фабрики соединений Java Message Service (JMS) для применения со связанным провайдером JMS. Эти свойства конфигурации контролируют способ создания соединений между провайдером JMS и используемой им системой обмена сообщениями.
  • Параметры почтового провайдера для приложений-клиентов
    Используйте эту страницу для реализации JavaMail API и создания почтовых сеансов.
  • Параметры почтового сеанса для приложений-клиентов
    С помощью этой страницы можно настроить свойства почтового сеанса.
  • Параметры провайдера JMS для приложений-клиентов
    С помощью этой страницы можно настроить свойства провайдера Java Message Service (JMS) при необходимости использовать провайдер JMS, отличный от провайдера обмена сообщениями по умолчанию или WebSphere MQ, выполняющего роль провайдера JMS.
  • Параметры провайдера URL для приложений-клиентов
    Используйте эту страницу для создания новых провайдеров URL.
  • Параметры провайдера источника данных для приложений-клиентов
    С помощью этой страницы можно создать источник данных в провайдере JDBC, предоставляющем определенный класс реализации драйвера JDBC.
  • Параметры провайдера среды ресурса для приложений-клиентов
    С помощью этой страницы можно указать свойства записи среды ресурса.
  • Параметры фабрики соединений адаптера ресурса
    Используйте эту панель для просмотра или изменения свойств конфигурации выбранной фабрики соединений адаптера ресурса.
  • Параметры фабрики соединений очереди провайдера WebSphere MQ для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранной фабрики соединений очереди для применения с провайдером Java Message Service (JMS) продукта MQSeries. Эти свойства конфигурации контролируют способ создания соединений между провайдером JMS и WebSphere MQ.
  • Параметры фабрики соединений очереди провайдера по умолчанию
    Используйте эту панель для просмотра или изменения свойств выбранной фабрики соединений очереди JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server. Эти свойства конфигурации контролируют создание соединений между провайдером JMS и использующей его шиной интеграции служб
  • Параметры фабрики соединений очереди провайдера по умолчанию версии 5 для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить параметры конфигурации выбранной фабрики соединений очередей JMS для двухточечного обмена сообщениями для работы с приложениями WebSphere Application Server версии 5. Эти свойства конфигурации управляют способом создания соединений между модулем JMS и системой обмена сообщениями по умолчанию, которая его использует.
  • Параметры фабрики соединений провайдера WebSphere MQ для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранной фабрики соединений для применения с провайдером Java Message Service (JMS) продукта WebSphere MQ. Эти свойства конфигурации контролируют способ создания соединений между провайдером JMS и WebSphere MQ.
  • Параметры фабрики соединений провайдера по умолчанию
    Используйте эту панель для просмотра или изменения свойств выбранной фабрики соединений JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server. Эти свойства конфигурации контролируют создание соединений между провайдером JMS и использующей его шиной интеграции служб
  • Параметры фабрики соединений темы провайдера по умолчанию
    Используйте эту панель для просмотра или изменения свойств выбранной фабрики соединений темы JMS для применения с внутренним провайдером Java Message Service (JMS) продукта, устанавливаемым вместе с WebSphere Application Server. Эти свойства конфигурации контролируют создание соединений между провайдером JMS и использующей его шиной интеграции служб.
  • Параметры фабрики соединений темы провайдера по умолчанию версии 5 для приложений-клиентов
    С помощью этой панели можно просмотреть или изменить свойства конфигурации выбранной фабрики соединений темы для применения с внутренним провайдером JMS (Java Message Service) продукта. Эти свойства конфигурации контролируют способ создания соединений между провайдером JMS и используемой им системой обмена сообщениями.
  • Провайдеры URL для инструмента настройки ресурсов приложения-клиента
    Провайдер URL реализует функцию для определенного протокола URL, например для протокола передачи гипертекстовой информации (HTTP). Этот провайдер, состоящий из пар классов, расширяет классы java.net.URLStreamHandler и java.net.URLConnection.
  • Свойства источника данных для приложений-клиентов
    На этой странице можно создать или изменить источники данных.
