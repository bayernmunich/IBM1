Параметры администрируемых объектов

Используйте эту панель для просмотра или изменения свойств конфигурации выбранных администрируемых объектов.

Для того чтобы открыть эту страницу Application Client Resource Configuration Tool (ACRCT), выберите Файл > Открыть. Выберите файл EAR и нажмите кнопку Открыть. Разверните выбранный файл JAR > Адаптеры ресурсов > экземпляр_адаптера_ресурсов. Щелкните правой кнопкой на пункте Администрируемые объекты и нажмите кнопку Создать. На вкладке Общие появятся следующие поля.

Параметры администрируемых объектов обрабатываются также, как и фабрики соединений. При обновлении администрируемых объектов используйте те же панели, что и для создания администрируемых объектов.

Имя

Имя этого администрируемого объекта для административных целей на сервере IBM WebSphere Application Server. Имя должно быть уникальным среди администрируемых объектов адаптера ресурса в административном домене продукта.

Тип данных Строка
Описание

Необязательное описание данной фабрики соединений для административных целей на сервере IBM WebSphere Application Server.

Тип данных Строка
Имя JNDI

Эта запись является именем resource-env-ref, именем message-destination-ref (если message-destination-ref не содержит ссылки) или ссылкой message-destination.

Тип данных Строка
Тип

Выпадающий список со всеми парами класс-интерфейс администрируемого объекта, заданными для администрируемых объектов в файле RAR адаптера ресурса.

Для каждого Типа имеется набор свойств, указанных в разделе Свойства. Этот набор свойств составляется с помощью извлечения свойств из каждого адресата администрируемого объекта. Для всех имеющихся администрируемых объектов, отображаемых для обновления, этот список перекрывается свойствами, указанными для объекта. При изменении поля Тип свойства также меняются для отображения корректных свойств для данного типа.
