Valores de URL para clientes de aplicaciones

Utilice esta página para implementar la función de un protocolo de URL determinado, como por ejemplo HTTP (Hyper Text Transfer Protocol).

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Proveedores de URL > instancia de proveedor de URL. Pulse con el botón derecho del ratón en URL y pulse Nuevo. Los campos siguientes aparecen en el separador General.

Este proveedor, formado por clases, amplía las clases java.net.URLStreamHandler y java.net.URLConnection.

Nombre

Nombre administrativo del URL.

Descripción

Descripción opcional del URL, para los registros de administración.

Nombre JNDI

El programa de tiempo de ejecución del cliente de aplicaciones utiliza este campo para recuperar información de configuración.

URL

Nombre de URL (Uniform Resource Locator) que apunta a un recurso de Internet o Intranet. Por ejemplo: http://www.ibm.com

Propiedades personalizadas

Especifica los pares nombre-valor para establecer propiedades adicionales en el objeto creado durante el tiempo de ejecución de este recurso.

Especifique un nombre que sea una propiedad pública en el objeto y un valor que se pueda convertir desde una serie en el tipo necesario para el método set de la propiedad. Las propiedades y valores aceptables dependen del objeto creado. Consulte la documentación del objeto para obtener una lista de propiedades válidas.



Los enlaces marcados (en línea) requieren acceso a Internet.
