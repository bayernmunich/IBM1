Configuración de nuevos proveedores de entorno de recursos para clientes de aplicaciones

Puede crear nuevas configuraciones de proveedores de entorno de recursos para el cliente de aplicación utilizando la herramienta ACRCT (Application Client Resource Configuration Tool).

Antes de empezar

Durante esta tarea, podrá crear nuevas configuraciones de proveedores de entorno de recursos para el cliente de aplicaciones.

Acerca de esta tarea

Para configurar un nuevo proveedor de entorno de recursos, siga los pasos siguientes:

Procedimiento

  1. Inicie la herramienta Application Configuration Resource Tool y abra el archivo EAR para el que desea configurar el nuevo proveedor de JMS (Java Message Service). El contenido del archivo EAR se muestra en una vista de árbol.
  2. En esta vista de árbol, seleccione el archivo JAR en el que desea configurar el nuevo proveedor de JMS.
  3. Expanda el archivo JAR para ver su contenido.
  4. Pulse la carpeta Proveedores de entornos de recursos. Realice una de las acciones siguientes:
    • Pulse con el botón derecho del ratón en la carpeta del proveedor y pulse Nuevo.
    • Pulse Editar > Nuevo en la barra de menús.
  5. Configure las propiedades del proveedor de JMS en los campos visualizados.
  6. Pulse Aceptar cuando haya terminado.
  7. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
