Configuración de nuevas sesiones de correo para clientes de aplicaciones

Puede utilizar la herramienta ACRCT (Application Client Resource Configuration Tool) para configurar nuevas sesiones de correo para el cliente de aplicaciones.

Antes de empezar

Durante esta tarea, se configurarán nuevas sesiones de correo para el cliente de aplicaciones. Las sesiones de correo se asocian con el proveedor de correo configurado previamente predeterminado que suministra el producto.

Procedimiento

  1. Inicie ACRCT (Application Client Resource Configuration Tool) y abra el archivo EAR.. El contenido del archivo EAR se muestra en una vista de árbol.
  2. Seleccione el archivo JAR en el que desea configurar la nueva sesión de JavaMail.
  3. Expanda el archivo JAR para ver su contenido.
  4. Pulse Proveedores de correo > Proveedor de correo > Sesiones de correo. Realice una de las acciones siguientes:
    • Pulse con el botón derecho del ratón en la carpeta Sesiones de correo y seleccione Nueva fábrica.
    • Pulse Editar > Nuevo en la barra de menús.
  5. Configure las propiedades de la sesión de correo en los campos visualizados.
  6. Pulse Aceptar.
  7. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
