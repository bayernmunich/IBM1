Configuración de sesiones y proveedores de correo para clientes de aplicaciones

Puede editar las configuraciones de todas las sesiones y proveedores de correo para la herramienta ACRT (Application Client Resource Configuration Tool).

Acerca de esta tarea

Utilice la herramienta (Application Client Resource Configuration Tool) para editar las configuraciones de las sesiones y proveedores de correo para que los clientes de aplicaciones las utilicen.

Procedimiento

  1. Inicie la herramienta ACRCT.
  2. Abra un archivo EAR.
  3. Localice los objetos de correo en el árbol que se visualiza en el archivo EAR. Por ejemplo, si el archivo contiene sesiones de correo, expanda Recursos > aplicación.jar > Proveedores de correo > instancia_proveedor_correo_java > Sesiones de correo.

    En este ejemplo, instancia_proveedor_correo_java es un proveedor de correo determinado.

Resultados

Las instancias de la sesión de correo se encuentran en la carpeta Sesiones de JavaMail.


Los enlaces marcados (en línea) requieren acceso a Internet.
