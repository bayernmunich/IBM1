Valores de entrada de entorno de recursos para clientes de aplicaciones

Utilice esta página para especificar las propiedades de entrada de entorno de recursos.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Proveedores de entorno de recursos > instancia_entorno_recursos. Pulse con el botón derecho del ratón en Entradas de entorno de recursos y pulse Nueva. Los campos siguientes aparecen en el separador General:

Nombre

Especifica el nombre administrativo de la entrada de entorno de recursos.

Descripción

Especifica una descripción del URL para los registros de administración.

Nombre JNDI

Especifica el nombre de la interfaz JNDI (Java Naming and Directory Interface) del recurso, incluidos los subcontextos de denominación.

Utilice este nombre para enlazar con la información de enlace de la plataforma. El enlace asocia los recursos definidos en el descriptor de despliegue del módulo con los recursos reales (o físicos) enlazados con JNDI mediante la plataforma.

Propiedades personalizadas

Especifica los pares nombre-valor para establecer propiedades adicionales en el objeto creado durante el tiempo de ejecución de este recurso.

Especifique un nombre que sea una propiedad pública en el objeto y un valor que se pueda convertir desde una serie en el tipo necesario para el método set de la propiedad. Las propiedades y valores aceptables dependen del objeto creado. Consulte la documentación del objeto para obtener una lista de propiedades válidas.
