Valores de destino de tema del proveedor predeterminado

Utilice este panel para ver o modificar las propiedades de configuración del destino de tema JMS seleccionado para su uso con el proveedor de JMS (Java Message Service) del producto interno que está instalado con WebSphere Application Server.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Proveedores de mensajería > Proveedor predeterminado. Pulse con el botón derecho del ratón en Destinos de tema y pulse Nuevo. Los campos siguientes aparecen en el separador General.

Nombre

Nombre de la entrada de destino del tema.

Tipo de datos String
Descripción

Descripción de la entrada.

Tipo de datos String
Nombre JNDI

Nombre JNDI utilizado para hacer coincidir esta definición con un nombre resource-env-ref de descriptor de despliegue.

Tipo de datos String
Espacio de tema

Nombre del espacio de tema. Este campo es necesario.

Tipo de datos String
Valor predeterminado DEFAULT_TOPIC_SPACE
Nombre de tema

Nombre del tema. Este campo es necesario.

Tipo de datos String
Modalidad de entrega

Modalidad predeterminada para los mensajes enviados a este destino.

Tipo de datos String
Rango Aplicación, Persistente o No Persistente
Valor predeterminado Aplicación
Tiempo de duración

La cantidad de tiempo predeterminado desde la hora de asignación durante la cual el sistema debe conservar un mensaje enviado a este destino, donde 0 indica que el valor de tiempo de duración no caduca. Valor del generador que se utiliza, si este campo no se rellena.

Tipo de datos Largo
Unidades Milisegundos
Prioridad

Prioridad de los mensajes enviados a este destino. Valor del generador que se utiliza, si este campo no se rellena.

Tipo de datos Integer
Rango De 0 a 9 con 0 como la prioridad inferior y 9 como la prioridad superior
Lectura hacia adelante

Se utiliza para controlar la optimización de lectura hacia adelante durante el envió de mensajes.

Tipo de datos String
Rango AsConnection, AlwaysOn y AlwaysOff
Valor predeterminado AsConnection
