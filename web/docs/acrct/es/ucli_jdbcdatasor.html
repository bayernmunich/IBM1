Propiedades del origen de datos para clientes de aplicaciones

Utilice esta página para crear o modificar los orígenes de datos.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Proveedores de orígenes de datos > Instancia de proveedor de origen de datos. Pulse con el botón derecho del ratón en Orígenes de datos y pulse Nuevo. Los campos siguientes se muestran en el separador General:

Nombre

Especifica el nombre de visualización de este origen de datos.

Tipo de datos String
Descripción

Especifica una descripción de texto del origen de datos.

Tipo de datos String
Nombre JNDI

El programa de tiempo de ejecución del cliente de aplicaciones utiliza este campo para recuperar información de configuración.

Nombre de la base de datos

Nombre de la base de datos a la que desea conectarse.

Usuario

Utilice el ID de usuario junto con la propiedad Contraseña en la operación de autenticación si la aplicación que llama no proporciona explícitamente un ID de usuario ni contraseña.

Si especifica un valor para la propiedad ID de usuario, también debe especificar un valor para la propiedad Contraseña. Las propiedades ID de usuario y Contraseña de la fábrica de conexiones se utilizan si la aplicación que llama no proporciona explícitamente un ID de usuario ni una contraseña.

Contraseña

Utilice la contraseña junto con la propiedad ID de usuario en la operación de autenticación si la aplicación que llama no proporciona explícitamente un ID de usuario ni contraseña.

Si especifica un valor para la propiedad Contraseña, también debe especificar un valor para la propiedad ID de usuario.

Vuelva a entrar la contraseña

Confirma la contraseña.

Propiedades personalizadas

Especifica los pares nombre-valor para establecer propiedades adicionales en el objeto creado durante el tiempo de ejecución de este recurso.

Especifique un nombre que sea una propiedad pública en el objeto y un valor que se pueda convertir desde una serie en el tipo necesario para el método set de la propiedad. Las propiedades y valores aceptables dependen del objeto creado. Consulte la documentación del objeto para obtener una lista de propiedades válidas.



Los enlaces marcados (en línea) requieren acceso a Internet.
