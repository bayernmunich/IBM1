Configuración de nuevas entradas de entorno de recursos para clientes de aplicaciones

Puede crear nuevas entradas de recursos para la aplicación de cliente utilizando la herramienta ACRCT (Application Client Resource Configuration Tool).

Antes de empezar

Durante esta tarea, podrá crear nuevas entradas de entorno de recursos de la aplicación cliente.

Acerca de esta tarea

Procedimiento

  1. Inicie ACRCT (Application Client Resource Configuration Tool).
  2. Abra el archivo EAR para el que desea configurar la nueva entrada de entorno de recursos. El contenido del archivo EAR se muestra en una vista de árbol.
  3. Pulse el proveedor de entorno de recursos deseado y realice la siguiente acción para configurar nuevos proveedores:
    • Configure un nuevo proveedor de entorno de recursos.
  4. Expanda el proveedor de entorno de recursos para ver la carpeta de entradas de entorno de recursos.
  5. Pulse la carpeta de entradas de entorno de recursos y realice una de las siguientes acciones:
    • Pulse con el botón derecho del ratón en la carpeta y seleccione Nuevo.
    • Pulse Editar > Nuevo en la barra de menús.
  6. Configure las propiedades de entradas de entorno de recursos en los campos visualizados.
  7. Pulse Aceptar.
  8. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
