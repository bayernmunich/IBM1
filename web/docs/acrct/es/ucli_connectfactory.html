Configuración de nuevas fábricas de conexiones para adaptadores de recursos para el cliente

Utilice ACRC (Application Client Resource Configuration Tool) para configurar nuevas fábricas de conexiones para adaptadores de recursos para el cliente.

Acerca de esta tarea

Lleve a cabo esta tarea para configurar nuevas fábricas de conexiones para los adaptadores de recursos.

Procedimiento

  1. Inicie ACRCT (Application Client Resource Configuration Tool).
  2. Abra el archivo EAR para el que desea configurar las nuevas fábricas de conexiones. El contenido del archivo EAR se muestra en una vista de árbol.
  3. En esta vista de árbol, seleccione el archivo JAR donde desea configurar las nuevas fábricas de conexiones.
  4. Expanda el archivo JAR para ver su contenido.
  5. Pulse la carpeta Adaptadores de recursos.
  6. Expanda el adaptador de recursos para el que desea crear las fábricas de conexiones.
  7. Pulse con el botón derecho del ratón en la carpeta Fábrica de conexiones y pulse Nueva.
  8. Configure las propiedades de la fábrica de conexiones en el diálogo de propiedades resultante.
  9. Pulse Aceptar.
  10. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
