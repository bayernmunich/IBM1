Valores del proveedor de entorno de recursos para clientes de aplicaciones

Utilice esta página para especificar las propiedades de entrada de entorno de recursos.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR (Java Archive). Pulse con el botón derecho del ratón en Proveedores de entorno de recursos y pulse Nuevo. Los campos siguientes se muestran en el separador General:

Nombre

Especifica el nombre administrativo del proveedor de entorno de recursos.

Descripción

Especifica una descripción del proveedor de entorno de recursos para los registros de administración.

Classpath

Especifica la vía de acceso al archivo JAR que contiene las clases de implementación del proveedor de entorno de recursos.

Propiedades personalizadas

Especifica los pares nombre-valor para establecer propiedades adicionales en el objeto creado durante el tiempo de ejecución de este recurso.

Especifique un nombre que sea una propiedad pública en el objeto y un valor que se pueda convertir desde una serie en el tipo necesario para el método set de la propiedad. Las propiedades y valores aceptables dependen del objeto creado. Consulte la documentación del objeto para obtener una lista de propiedades válidas.



Los enlaces marcados (en línea) requieren acceso a Internet.
