Valores de fábrica de conexiones del adaptador de recursos

Utilice este panel para ver o modificar las propiedades de configuración de la fábrica de conexiones del adaptador de recursos.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Adaptadores de recursos. Pulse con el botón derecho del ratón en la carpeta Fábrica de conexiones y pulse Nueva. Los campos siguientes aparecen en el separador General.

Nombre

Nombre por el que se conoce esta fábrica de conexiones con fines administrativos en WebSphere Application Server. El nombre debe ser exclusivo dentro de las fábricas de conexiones del adaptador de recursos en el dominio administrativo del producto.

Tipo de datos String
Descripción

Descripción opcional de esta fábrica de conexiones con fines administrativos dentro de IBM WebSphere Application Server.

Tipo de datos String
Nombre JNDI

Nombre JNDI que se utiliza para hacer coincidir esta definición de la fábrica de conexiones de adaptador de recursos con el descriptor de despliegue. Esta entrada debería ser un nombre resource-ref.

Tipo de datos String
Nombre de usuario

Nombre de usuario utilizado, junto con la propiedad Contraseña, en la operación de autenticación si la aplicación que llama no proporciona explícitamente un ID de usuario ni contraseña cuando se obtiene la conexión. Si se utiliza este campo, se ignorará el campo Nombre de usuario de Propiedades.

Si especifica un valor para la propiedad Nombre de usuario, también debe especificar un valor para la propiedad Contraseña.

Las propiedades Nombre de usuario y Contraseña de la fábrica de conexiones se utilizan si la aplicación que llama no proporciona explícitamente un ID de usuario ni una contraseña cuando se obtiene la conexión.

Tipo de datos String
Contraseña

Especifica una contraseña cifrada. Si rellena este campo, se ignora el campo Contraseña del recuadro Propiedades.

Si especifica un valor para la propiedad Nombre de usuario, también debe especificar un valor para la propiedad Contraseña.

Tipo de datos String
Vuelva a entrar la contraseña

Confirma la contraseña.

Tipo

Lista de todas las connectionFactoryInterfaces tal como se definen en el archivo RAR (Resource Adapter Archive).

Para cada Tipo, hay un conjunto de propiedades especificadas en el recuadro Propiedades. Para construir este conjunto de propiedades, recupere las propiedades de cada objeto de definición de conexión. En el caso de las fábricas de conexiones existentes que se visualizan para su actualización, esta lista de propiedades se sobrescriben con las propiedades especificadas para los objetos. Cuando el campo Tipo se modifica, las propiedades también se modifican para reflejar las propiedades correctas para este tipo.

Tipo de datos String

Nombre de archivo: ucli_connectfact.html
