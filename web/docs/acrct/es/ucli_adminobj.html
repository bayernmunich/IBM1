Valores de los objetos administrados

Utilice este panel para ver o modificar las propiedades de configuración de los objetos administrados seleccionados.

Para ver esta página de la herramienta ACRCT (Application Client Resource Configuration Tool), pulse Archivo > Abrir. Después de buscar un archivo EAR, pulse Abrir. Expanda el archivo JAR seleccionado > Adaptadores de recursos > instancia_adaptador_recursos. Pulse con el botón derecho del ratón en Objetos administrados y pulse Nuevo. Los campos siguientes aparecen en el separador General.

Los valores de los objetos administrados se manejan de modo similar a las fábricas de conexiones. Al actualizar los objetos administrados, utilice los mismos paneles utilizados para crearlos.

Nombre

Nombre por el que se conoce este objeto administrativo para fines administrativos dentro de IBM WebSphere Application Server. El nombre debe ser exclusivo dentro de los objetos administrados de adaptador de recursos en el dominio administrativo del producto.

Tipo de datos String
Descripción

Descripción opcional de esta fábrica de conexiones con fines administrativos dentro de IBM WebSphere Application Server.

Tipo de datos String
Nombre JNDI

Esta entrada es un nombre de resource-env-ref, un nombre de message-destination-ref (si el message-destination-ref no tiene enlace) o bien un enlace message-destination.

Tipo de datos String
Tipo

Lista desplegable de todas las parejas de interfaz-clase de los objetos administrados definidos para los objetos administrados en el archivo RAR (Resource Adapter Archive).

Para cada Tipo, hay un conjunto de propiedades especificadas en el recuadro Propiedades. Para construir este conjunto de propiedades, recupere las propiedades de cada definición de objeto administrado. En el caso de los objetos administrados existentes que se visualizan para su actualización, esta lista de propiedades se sobrescribe con las propiedades especificadas para los objetos. Cuando el campo Tipo se modifica, las propiedades también se modifican para reflejar las propiedades correctas para este tipo.

Tipo de datos String
