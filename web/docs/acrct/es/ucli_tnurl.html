Configuración de nuevos proveedores de URL para clientes de aplicaciones

Puede crear URL y proveedores de URL para la aplicación de cliente utilizando la herramienta ACRCT (Application Client Resource Configuration Tool).

Antes de empezar

Durante esta tarea, creará los URL y proveedores de URL para la aplicación cliente. En una tarea administrativa independiente, debe instalar el código Java para el proveedor de URL necesario en la máquina cliente en la que reside la aplicación cliente.

Acerca de esta tarea

Procedimiento

  1. Inicie ACRCT (Application Client Resource Configuration Tool).
  2. Abra el archivo EAR para el que desea configurar el nuevo proveedor de URL. El contenido del archivo EAR se muestra en una vista de árbol.
  3. En esta vista de árbol, seleccione el archivo JAR donde desea configurar el nuevo proveedor de URL.
  4. Expanda el archivo JAR para ver el contenido.
  5. Pulse la carpeta Proveedores de URL. Realice una de las acciones siguientes:
    • Pulse con el botón derecho del ratón en la carpeta y seleccione Nuevo.
    • Pulse Editar > Nuevo en la barra de menús.
  6. Configure las propiedades del proveedor de URL en el diálogo de propiedades resultante.
  7. Pulse Aceptar.
  8. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
