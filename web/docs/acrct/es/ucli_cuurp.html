Proveedores de URL para Application Client Resource Configuration Tool

Un proveedor URL implementa la función de un protocolo URL determinado, por ejemplo HTTP (HyperText Transfer Protocol). Este proveedor, que consta de un par de clases, amplía las clases java.net.URLStreamHandler y java.net.URLConnection.



Los enlaces marcados (en línea) requieren acceso a Internet.
