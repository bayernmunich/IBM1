Configuración de objetos administrados para adaptadores de recursos para el cliente

Esta sección le ayuda a configurar nuevos objetos administrados para el cliente.

Antes de empezar

Antes de configurar nuevos objetos administrados, debe completar los requisitos previos siguientes:
  1. Instale el archivo RAR (Resource Adapter Archive) utilizando la herramienta clientRAR.
  2. Configure el adaptador de recursos del archivo .ear, utilizando ACRCT (Application Client Resource Configuration Tool).

Acerca de esta tarea

Lleve a cabo esta tarea para configurar nuevos objetos administrados para los adaptadores de recursos instalados.

Procedimiento

  1. Inicie ACRCT (Application Client Resource Configuration Tool).
  2. Abra el archivo EAR para el que desea configurar los nuevos objetos administrados. El contenido del archivo EAR se muestra en una vista de árbol.
  3. En esta vista de árbol, seleccione el archivo JAR donde desea configurar los nuevos objetos administrados.
  4. Expanda el archivo JAR para ver su contenido.
  5. Pulse la carpeta Adaptadores de recursos.
  6. Expanda el adaptador de recursos para el que desea crear objetos administrados.
  7. Pulse con el botón derecho del ratón en la carpeta Objetos administrados y pulse Nuevo.
  8. Configure las propiedades de objeto administrado en el diálogo de propiedades resultante.
  9. Pulse Aceptar.
  10. Pulse Archivo > Guardar en la barra de menús para guardar los cambios.


Los enlaces marcados (en línea) requieren acceso a Internet.
