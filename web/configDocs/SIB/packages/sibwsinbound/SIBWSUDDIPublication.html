SIBWSUDDIPublication

This type is a class for model objects.

Describes the manner in which the WSDL describing an inbound service is published to UDDI registries.

Package: sibwsinbound

Classifier ID:  -1
Instance class name:  * Unspecified *

UDDIBusinessKey : EString The UDDI business key for this service.

WSDLServingHTTPURLRoot : EString The HTTPURLRoot for the server holding the WSDL. By default this value will be the URLRoot from the EndpointListener for the first Port in the first InboundService that has been configured (if any).
