Package portletapplication

Description: The configuration data of a portlet application. It is loaded from the portlet.xml file of the WAR-file. Consequently the structure is identical to what is defined in portlet-app_2_0.xsd. The data is used by the portlet container and made available to components that use the portlet container. Normally this would be a portal.
