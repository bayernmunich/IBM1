ActivationConfig

This type is a class for model objects.

@version J2EE1.4 The activation-configType defines information about the expected configuration properties of the message-driven bean in its operational environment. This may include information about message acknowledgement, message selector, expected destination type, etc. The configuration information is expressed in terms of name/value configuration properties. The properties that are recognized for a particular message-driven bean are determined by the messaging type.
