Attributes Summary
implementationClassName : EString The Java class name of the JDBC driver implementation.
xa : EBoolean This is an attribute added for handling templates within the admin console. It should not be examined by the binder or anything else. The idea being that some JDBC providers use the same classname for an XA or NON-XA driver (It's set via properties). This adds a consistant value to the config file for the templates that can be used to select the XA from the NON-XA template when the classnames are identical.
