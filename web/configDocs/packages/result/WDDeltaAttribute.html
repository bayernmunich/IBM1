WDDeltaAttribute

This type is a class for model objects.

The result of comparing two attributes.

When comparing two attributes, the attributes must match by name, type, and containment structure.

Comparison is according to the attribute typing; the contained/reference/string axis is checked first. This corresponds with the three immediate subtypes.
