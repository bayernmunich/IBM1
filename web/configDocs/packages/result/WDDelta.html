WDDelta

This type is a class for model objects.

Result from a node comparison.

The comparison acts on two model elements. There are three possible results:

  • The objects are both model elements;
  • The objects are both attributes;
  • The objects are not both of the same type, or do not have the same name, or do not have the same collection structure;

These three cases correspond to the three delta subtypes.
