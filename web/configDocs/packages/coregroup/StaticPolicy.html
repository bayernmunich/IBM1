StaticPolicy

This type is a class for model objects.

A concrete HAManagerPolicy. The policy specifies the fixed set of servers where the application will be activated by the HAManager. The policy does not provide any automatic fail over support. To switch the application from one server (possibly the failed one) to another, the administrator may have to manually intervene in the form of alteration of the 'servers' list.
