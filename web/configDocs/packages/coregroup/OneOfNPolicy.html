OneOfNPolicy

This type is a class for model objects.

A concrete type of PreferredServerPolicy containing no special attributes. The policy dictates that at any instance, the application can execute only on one server. HAManager may fail over the application to another server of the application group, if that server fails. The preferredServers list associated with an OneOfNPolicy objects specifies the server list ordered according to user preference. The preference can be based on server capacity, network topology, resource proximity, etc.
