NoOpPolicy

This type is a class for model objects.

A special concrete subtype of HAManagerPolicy with no additional attributes. There are no HAManager callbacks in this policy. The policy may be used by components who want to use only the Group Communication Systems, and not the complete HAManager functionalities.
