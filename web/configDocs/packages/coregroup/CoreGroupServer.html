CoreGroupServer

This type is a class for model objects.

Essentially a data-type for uniquely identifying WebSphere servers in a cell-scoped coregroup.xml document. Along with the String-valued server name, the pseudo-type also contains the String-valued node name for fully qualifying a WebSphere server.
