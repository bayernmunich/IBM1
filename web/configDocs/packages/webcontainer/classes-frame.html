Package webcontainer

Parent package: applicationserver

Description: Settings for Web Container components of an Application Server (server.xml) Application Servers can have various kinds of Containers configured within them to provide services to applications they are serving. Web Containers provide web-related services required for hosting Web Modules and their related components, such as Servlets and JSP's.
