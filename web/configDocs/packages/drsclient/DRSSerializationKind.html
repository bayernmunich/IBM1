DRSSerializationKind

This type is an enumeration data type (a type for attribute values).

This enumeration lists the kind of data transfers that can be performed for messages between Data Replication Service instances. The two options available are transfer as java objects and transfer as bytes. This attribute is relevant in v5.x only. @since v6.0 deprecated
