I18NContainerInternationalizationAttribute

This type is a class for model objects.

The base type for all container internationalization attributes; a I18NContainerInternationalizationAttribute is the internationalization policy setting that indicates which internationalization context a container will scope to an invocation of a J2EE application entity.
