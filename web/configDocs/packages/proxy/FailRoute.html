FailRoute

This type is a class for model objects.

This action indicates that the incoming request that this rule matched should be failed.

Package: proxy
Supertype: Route

Classifier ID:  -1
Instance class name:  * Unspecified *
Instance class:  * Unspecified *


Attributes Summary
failStatusCode : EInt This attribute specifies the HTTP status code that should be used to indicate the reason for failing the request. This status code is used in the response if there is no custom error page generation application defined. If there is a custom error page generation application that can be routed to using a valid proxy or local route rule, then this status code is passed to the application as an argument to generated the error body.
