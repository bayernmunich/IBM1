Performing a quick search

You can search a topic or a set of topics in the navigation by using the Quick Search feature.

In the Navigation view, take the following steps to perform a quick search:

  1. Click Search Topic () on the tool bar of the navigation view to display the menu.
    1. To search only within the topic that you open, click Search selected topic.
    2. To search through the topic and all its subtopics, click Search selected topic and all its subtopics.
  2. In the popup search dialog box, enter your query in the Search field and click OK to start the search.

If your search terms are matched, the search result view will display the returned topics and a message stating the number of the search results.

If your search terms are not matched, you will get the message that 0 results find in topics for your query.
