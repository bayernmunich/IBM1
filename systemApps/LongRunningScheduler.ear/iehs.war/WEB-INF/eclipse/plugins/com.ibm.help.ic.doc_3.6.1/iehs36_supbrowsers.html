Supported browsers for the information center

To use this information center, you must have a supported web browser installed on your system.

The following web browsers are supported:

  • Microsoft Internet Explorer Version 7.0 or Version 8.0
  • Mozilla Firefox Version 3.5 or Version 3.6.

If you use a different version of the supported web browsers or another web browser, some functionality might not be available or work as expected.

To ensure that all functions of the information center are usable, you must enable cookies and JavaScript in the web browser and you must allow the web browser to display pop-up windows from this information center.
