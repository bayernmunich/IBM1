檢視工作

請利用這個頁面來檢視所有已提交給工作排程器的工作清單，以及檢視工作的相關資訊。

如果要檢視這個工作管理主控台頁面，請按一下工作管理 > 檢視工作。

如果要在工作上套用動作，請選取工作和動作，然後按一下套用。

請利用過濾器功能，進行工作的過濾和排序。 請按一下「顯示過濾器功能」圖示 ，輸入過濾準則，然後按一下執行。

如果要重新整理這個畫面，請按一下導覽中的檢視工作，或按一下「狀態」直欄中的圖示。

工作管理主控台中的專用權會隨著指派的角色而不同。 針對這個主控台頁面，使用者可以執行下列動作：

  • 管理身為 lrsubmitter 角色之使用者所擁有的工作。
  • 管理所有身為 lradmin 角色之使用者的工作。

可以採取多種動作，視工作類型和狀態而定。 從這個主控台頁面中，具備 lrsubmitter 或 lradmin 角色的使用者可以完成取消、清除、重新啟動、移除、暫停或回復工作之類的動作。

工作 ID

指定工作鏈結。如果要檢視工作日誌，請按一下工作 ID 的鏈結。

提交者

指定提交工作之使用者的 ID。

前次更新

指定工作前次更新的日期。

狀態

顯示工作的狀態。

節點

指定工作所在的節點。

應用程式伺服器

指定工作所在的應用程式伺服器。

群組

指定在啟用群組型安全的情況下，或是在啟用群組型與角色型安全的情況下，指派給工作的群組名稱。

標示（線上）的鏈結表示需要存取網際網路。
