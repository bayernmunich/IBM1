Actualizar planificación

Utilice este panel para ver o actualizar un trabajo y su planificación.

Para ver esta página, pulse Gestión de planificaciones > Ver planificaciones > Seleccionar_planificación.

Los privilegios en la consola de gestión de trabajos varían en función de la seguridad de grupo y los roles asignados.
  • Un usuario con el rol lradmin puede actualizar una planificación.
  • Un usuario con el rol lrsubmitter o el rol no puede lrmonitor no puede actualizar una planificación.
  • Cuando la seguridad de grupo está habilitada, un usuario del grupo puede actualizar una planificación para un trabajo en el grupo.
  • Cuando la opción para la seguridad del grupo y la seguridad de roles está habilitada, un usuario del grupo puede actualizar una planificación para un trabajo en el grupo. El usuario debe estar en el rol lradmin.

Pulse Aplicar o Aceptar para guardar los cambios. Pulse Cancelar para cancelar los cambios. Pulse Restablecer para borrar los cambios en el panel de la consola y restaurar los últimos valores guardados.

Planificación

Se cambia la fecha de inicio, la hora de inicio y el intervalo de la planificación.

Trabajo

Visualiza el contenido de un trabajo o sustituye la definición del trabajo en este panel.

  • Pulse el enlace junto a Trabajo a ejecutar para ver el contenido del trabajo asociado con la planificación que desea cambiar.

    Opcionalmente, puede actualizar las propiedades de sustitución.

  • Pulse el recuadro de selección, Sustituir definición de trabajo, para cambiar la definición del trabajo. Puede sustituir la definición del trabajo desde un sistema de archivos local o del depósito de trabajos.

Los enlaces marcados (en línea) requieren acceso a Internet.
