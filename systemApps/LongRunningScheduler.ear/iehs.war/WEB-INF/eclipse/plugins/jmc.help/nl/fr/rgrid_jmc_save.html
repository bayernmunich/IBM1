Sauvegarde d'un travail

Cette page permet de sauvegarder une définition de travail dans le référentiel des travaux.

Pour afficher cette page, cliquez sur Référentiel des travaux > Enregistrer un travail.

Les privilèges de la console de gestion des travaux varient selon la sécurité des groupes et les rôles affectés.
  • Un utilisateur ayant le rôle lradmin peut sauvegarder un travail, contrairement à un utilisateur ayant le rôle lrsubmitter ou lrmonitor.
  • Lorsque la sécurité des groupes est activée, un utilisateur du groupe peut sauvegarder un travail dans le groupe.
  • Lorsque l'option relative à la sécurité des groupes et des rôles est activée, un utilisateur du groupe peut sauvegarder un travail dans le groupe. Cet utilisateur doit avoir le rôle lradmin.

Nom du travail

Indique le nom du travail.

Cliquez sur Sauvegarder pour sauvegarder la nouvelle définition de travail. Cliquez sur Réinitialiser pour recommencer.

Remplacer le travail si le nom de travail défini existe

Indique que la définition de travail présente dans le référentiel doit être remplacée, le cas échéant. La définition est remplacée lorsque vous cliquez sur Sauvegarder.

Si cette case n'est pas cochée, la définition n'est pas remplacée lorsque vous cliquez sur Sauvegarder.
