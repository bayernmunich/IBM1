Bienvenue dans la console de gestion des travaux

La console de gestion des travaux est une interface Web autonome permettant aux utilisateurs d'exécuter des opérations sur les travaux.

Pour afficher cette page de la console de gestion de travaux, cliquez sur Aide sur la page Bienvenue de la console de gestion des travaux.

Vous devez avoir le rôle lrsubmitter, lradmin ou les deux dans la console d'administration pour accéder à console de gestion des travaux.

Selon leurs privilèges, les utilisateurs peuvent exécuter diverses opérations sur les travaux. La console de gestion des travaux fournit un accès contrôlé lorsque la sécurité est active.

Les liens marqués (en ligne) requièrent un accès à Internet.
