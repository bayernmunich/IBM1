Просмотр сохраненного содержимого задания

На этой странице можно просмотреть содержимое xJCL сохраненного задания. Просмотр сохраненного содержимого может потребоваться для определения его правильности либо необходимости внесения корректировок.

Для просмотра этой страницы выберите Хранилище заданий > Показать сохраненные задания > Выберите-задание.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли. Пользователь с ролью lrsubmitter или lradmin может просматривать сохраненное содержимое задания. При включенной защите группы пользователь в группе может просматривать сохраненное содержимое для задания в группе.

При включенной защите группы и защите роли пользователь в группе может просматривать сохраненное содержимое для заданий в группе. Пользователю может быть назначена роль lrsubmitter или lradmin.

Назад

Для возврата к списку сохраненных заданий нажмите кнопку Назад.
