Plan erstellen

Verwenden Sie den Assistenten, um einen Plan der Ereignisse zu stellen, die für einen Job eintreten.

Zum Anzeigen dieser Seite klicken Sie auf Planverwaltung > Plan erstellen.

Verwenden Sie den Assistenten, um Informationen anzugeben.

Die Berechtigungen in der Jobverwaltungskonsole variieren in Abhängigkeit von der Gruppenzugehörigkeit und den zugeordneten Rollen.
  • Benutzer in der Rolle "lradmin" können Pläne erstellen.
  • Benutzer in der Rolle "lrmonitor" oder "lrsubmitter" können keine Pläne erstellen.
  • Wenn die Gruppensicherheit aktiviert ist, kann jeder Benutzer in der Gruppe Pläne für die Jobs in der Gruppe erstellen.
  • Wenn die Option für Gruppensicherheit und Rollensicherheit aktiviert ist, kann ein Benutzer in der Gruppe einen Plan für einen Job in der Gruppe erstellen. Der Benutzer muss die Rolle "lradmin" haben.

Schritt 1: Plan erstellen

Alle Felder in dieser Anzeige sind erforderlich.

Name
Geben Sie den Namen des Plans an, den Sie erstellen möchten.
Startdatum
Geben Sie das Startdatum für den Plan an. Das Startdatum muss das Format jjjj-MM-tt haben.
Startzeit
Geben Sie die Startzeit für den Plan an. Die Startzeit muss das Format HH:mm:ss haben.
Intervall
Geben Sie das Intervall an, in dem der Plan ausgeführt werden soll: täglich, wöchentlich oder monatlich.

Klicken Sie auf Weiter.

Schritt 2: Job angeben

Gibt den Pfad der Jobdefinition an, die als neuer Job übergeben werden soll.

Lokales Dateisystem
Suchen Sie im lokalen Dateisystem nach der Jobdefinition.
Fernes Dateisystem
Suchen Sie auf dem Jobrepositorysystem nach der Jobdefinition.

Geben Sie den Pfad an, und klicken Sie auf Weiter.

Schritt 2.1: Substitutionseigenschaften angeben

Gibt Werte für die Substitutionseigenschaften für diesen Job an.

Dieser Schritt wird geöffnet, wenn die Jobdefinition Substitutionseigenschaften enthält.

Geben Sie Werte an, und klicken Sie auf Weiter.

Schritt 3: Plan bestätigen

Listet eine Zusammenfassung der von Ihnen ausgewählten Optionen auf.

Klicken Sie auf Fertig stellen, um die Jobübergabe abzuschließen.

Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
