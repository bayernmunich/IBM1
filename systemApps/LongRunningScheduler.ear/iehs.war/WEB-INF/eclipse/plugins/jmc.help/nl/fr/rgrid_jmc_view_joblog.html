Afficher le journal d'un travail

Cette page permet d'afficher les journaux de tous les travaux envoyés au planificateur des travaux et de télécharger un journal sur votre système local.

Pour afficher cette page de la console de gestion des travaux, cliquez sur Gestion des travaux > Afficher les travaux > nom du travail.

Utilisez ce panneau pour afficher les informations d'un journal de travail ou télécharger le fichier journal vers votre système local.

Les privilèges de la console de gestion des travaux varient selon les rôles assignés. Pour cette page de la console, un utilisateur peut exécuter les actions suivantes :

  • Gérer les travaux dont l'utilisateur est propriétaire à l'aide du rôle lrsubmitter.
  • Gérer les travaux de tous les utilisateurs à l'aide du rôle lradmin.

Régénérer

Cliquez pour actualiser.

Télécharger

Cliquez pour télécharger ce journal de travail sur votre système local.

Retour

Cliquez pour revenir au panneau d'affichage des travaux.

Les liens marqués (en ligne) requièrent un accès à Internet.
