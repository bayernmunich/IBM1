Creación de una planificación

Utilice el asistente para crear un planificación de sucesos que se producen para un trabajo.

Para ver esta página, pulse Gestión de planificación > Crear una planificación.

Utilice el asistente para especificar información.

Los privilegios en la consola de gestión de trabajos varían, dependiendo de la seguridad del grupo y de los roles asignados.
  • Los usuarios con el rol lradmin pueden crear planificaciones.
  • Los usuarios con el rol de lrmonitor o el rol lrsubmitter no pueden crear planificaciones.
  • Cuando la seguridad de grupo está habilitada, cualquier usuario del grupo puede crear planificaciones para trabajos en el grupo.
  • Cuando la opción para la seguridad de grupo y la seguridad de roles está habilitada, un usuario del grupo podrá crear una planificación para un trabajo en el grupo. El usuario debe estar en el rol lradmin.

Paso 1: cree una planificación

Todos los campos de este panel son necesarios.

Name
Especifica el nombre de la planificación que desea crear.
Fecha de inicio
Especifique la fecha de inicio para que comience la planificación. La fecha de inicio debe estar en formato aaaa-MM-dd.
Hora de inicio
Especifique la hora de inicio para que comience la planificación. La hora de inicio debe estar en el formato HH:mm:ss.
Intervalo
Especifique el período del intervalo como o diario, semanal o mensual para ejecutar la planificación.

Pulse Next (Siguiente).

Paso 2: especifique un trabajo

Especifica la vía de acceso a la definición del trabajo para someterlo como un nuevo trabajo.

Sistema de archivos local
Busque la definición de trabajo del sistema de archivos local.
Sistema de archivos remoto
Examine la definición de trabajo del sistema de repositorio de trabajos.

Especifique la vía de acceso y pulse Siguiente.

Paso 2.1: especifique las propiedades de sustitución

Especifica los valores de las propiedades de sustitución de este trabajo.

Este paso se abre si la definición de trabajo tiene propiedades de sustitución.

Especifique los valores y pulse Siguiente.

Paso 3: confirme la creación de la planificación

Lista un resumen según las selecciones.

Para completar el envío de esta planificación, pulse Finalizar.

Los enlaces marcados (en línea) requieren acceso a Internet.
