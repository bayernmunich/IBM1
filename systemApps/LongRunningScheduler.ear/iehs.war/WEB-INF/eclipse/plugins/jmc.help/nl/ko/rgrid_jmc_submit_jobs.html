작업 제출

이 페이지에서 작업 정의를 지정하여 작업을 제출할 수 있습니다. 작업 정의는 로컬 파일 시스템에서 생성할 수 있습니다.

이 작업 관리 콘솔 페이지를 보려면 작업 관리 > 작업 제출을 클릭하십시오.

작업 관리 콘솔의 권한은 지정된 역할에 따라 다릅니다. 이 콘솔 페이지의 경우, lrsubmitter 역할 또는 lradmin 역할의 사용자가 작업을 제출할 수 있습니다.

로컬 파일 시스템

새 작업으로 제출할 작업 정의의 경로를 지정합니다.

찾아보기 단추를 사용하여 로컬 파일의 전체 경로를 찾아 지정합니다.

작업 저장소

작업의 이름을 지정합니다.

작업 정의가 작업 저장소에 저장되어 있으면 작업 저장소를 선택하십시오.

대체 특성

이 옵션을 선택하여 작업의 대체 특성 값을 업데이트합니다.

작업에 값이 없는 대체 특성이 있을 경우 값을 지정해야 합니다.

제출 지연

작업을 실행할 때 시작 날짜 및 시간을 지연하려면 선택하십시오.

작업을 스케줄러에 제출하려면 제출을 클릭하고 제출을 취소하려면 취소를 클릭하십시오.

(온라인)으로 표시된 링크는 인터넷에 액세스해야 합니다.
