Enviar uma Tarefa

Utilize essa página para enviar uma tarefa especificando a definição da tarefa. A definição de tarefa pode se originar do sistema de arquivos local.

Para visualizar esta página do console de gerenciamento de tarefa, clique em Gerenciamento de Tarefa > Enviar uma Tarefa.

Os privilégios no console de gerenciamento de tarefas variam, dependendo das funções designadas. Para esta página do console, os usuários com a função lrsubmitter ou lradmin podem enviar tarefas.

Sistema de Arquivos Local

Especifique o caminho da definição de tarefa a ser enviada como uma nova tarefa.

Utilize o botão Procurar para localizar e especificar o caminho completo de um arquivo local.

Repositório de Tarefa

Especifique o nome da tarefa.

Selecione Repositório de Tarefa se a definição de tarefa estiver armazenada no repositório de tarefas.

Propriedades de Substituição

Selecione para atualizar os valores das propriedades de substituição para a tarefa.

Se uma tarefa tiver propriedades de substituição sem valores, você deve especificá-las.

Envio com Atraso

Selecione para atrasar a data e o horário de início da execução da tarefa.

Clique em Enviar para enviar a tarefa para o planejador ou clique em Cancelar para cancelar o envio.

Links marcados (on-line) requerem acesso à Internet.
