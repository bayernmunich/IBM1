Visualización de planificaciones

Utilice esta página para ver la información de planificación o para cancelar planificaciones de trabajo.

Para ver esta página, pulse Gestión de planificador > Visualización de planificaciones.

Esta página muestra la lista de las planificaciones.

Los privilegios en la consola de gestión de trabajos varían, dependiendo de la seguridad del grupo y de los roles asignados.
  • Los usuarios con el rol lradmin pueden ver planificaciones y puede cancelar planificaciones de trabajos.
  • Cuando se habilita la seguridad de grupo, cualquier usuario del grupo podrá ver las planificaciones del grupo y podrá cancelar planificaciones de trabajos en el grupo.
  • Cuando la opción de seguridad del grupo y la seguridad de roles está habilitada, un usuario puede completar las siguientes acciones:
    • Ver planificaciones y cancelar planificaciones de trabajos si el usuario está en el rol lradmin.

Utilice la función de filtrado para filtrar y clasificar planificaciones. Pulse el icono Mostrar función de filtro , entre un criterio de filtro y, a continuación, pulse Ir. Para renovar este panel, pulse Renovar.

Name

Especifica el nombre de la planificación.

Para ver una definición de planificación, pulse el enlace Nombre. Para cancelar una planificación, seleccione la planificación y pulse Cancelar.

Sometedor

Especifica el emisor de la planificación.

Fecha y hora de inicio

Especifica la fecha y hora de inicio de la planificación.

Intervalo

Especifica el intervalo de tiempo en que se ejecuta el programa.

Los enlaces marcados (en línea) requieren acceso a Internet.
