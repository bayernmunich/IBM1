Jobprotokoll anzeigen

Verwenden Sie diese Seite, um Jobprotokollinformationen aller an den Job-Scheduler übergebenen Jobs anzuzeigen und um ein Jobprotokoll auf das lokale System herunterzuladen.

Klicken Sie zum Anzeigen dieser Seite der Jobverwaltungskonsole auf Jobverwaltung > Jobs anzeigen > Jobname.

Verwenden Sie diese Anzeige, um Jobprotokollinformationen anzuzeigen oder um die Protokolldatei auf Ihr lokales System herunterzuladen.

Die Berechtigungen in der Jobverwaltungskonsole variieren in Abhängigkeit von den zugeordneten Rollen. Für diese Konsolseite kann ein Benutzer die folgenden Aktionen ausführen:

  • Jobs verwalten, deren Eigner der Benutzer mit der Rolle "lrsubmitter" ist
  • Jobs aller Benutzer mit der Rolle "lradmin" verwalten

Aktualisieren

Klicken Sie auf diese Schaltfläche, um die Anzeige zu aktualisieren.

Herunterladen

Klicken Sie auf diese Schaltfläche, um das Jobprotokoll auf Ihr lokales System herunterzuladen.

Zurück

Klicken Sie auf diese Schaltfläche, um zur Anzeige "Job anzeigen" zurückzukehren.

Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
