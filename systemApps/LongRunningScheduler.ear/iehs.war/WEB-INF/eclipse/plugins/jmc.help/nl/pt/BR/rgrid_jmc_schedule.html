Visualizar Planejamentos

Use esta página para visualizar informações de planejamento ou cancelar planejamentos de tarefa.

Para visualizar esta página, clique em Gerenciamento de Planejamento > Visualizar Planejamentos.

A página exibe a lista de planejamentos.

Privilégios no console de gerenciamento de tarefa variam, dependendo da segurança do grupo e das atribuições designadas.
  • Usuários na função lradmin podem visualizar planejamentos e cancelar planejamentos para tarefas.
  • Quando a segurança do grupo está ativada, qualquer usuário no grupo pode visualizar planejamentos no grupo e cancelar planejamentos para tarefas no grupo.
  • Quando a opção para segurança do grupo e segurança da função está ativada, um usuário pode concluir as seguintes ações.
    • Visualizar planejamentos e cancelar planejamentos para tarefas se o usuário estiver na função lradmin.

Use a função de filtro para filtrar e classificar planejamentos. Clique no ícone Mostrar função de filtro , insira os critérios de filtragem e, depois, clique em Ir. Para atualizar esse painel, clique em Atualizar.

Nome

Especifica o nome do planejamento.

Para visualizar uma definição de planejamento, clique no link Nome. Para cancelar um planejamento, selecione-o e clique em Cancelar.

Requisitante

Especifica o requisitante do planejamento.

Data e Horário de Início

Especifica a data e o horário de início para o planejamento.

Intervalo

Especifica o intervalo de tempo para execução do planejamento.

Links marcados (on-line) requerem acesso à Internet.
