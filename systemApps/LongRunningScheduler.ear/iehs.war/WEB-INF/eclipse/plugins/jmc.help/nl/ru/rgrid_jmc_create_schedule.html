Создание расписания

Мастер используется для создания расписания событий, выполняемых для задания.

Для просмотра этой страницы выберите Управление расписанием > Создать расписание.

Для указания информации используйте мастер.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли.
  • Пользователи с ролью lradmin могут создавать расписания.
  • Пользователи с ролью lrmonitor или lrsubmitter не могут создавать расписания.
  • При включенной защите группы любой пользователь в группе может создать расписание для задания в этой группе.
  • При включенной защите группы и защите роли пользователь в группе может создать расписание для задания в группе. Пользователю должна быть назначена роль lradmin.

Этап 1: Создать расписание

Все поля на этой панели являются обязательными.

Имя
Укажите имя создаваемого расписания.
Начальная дата
Укажите начальную дату для расписания. Формат начальной даты: гггг-ММ-дд.
Начальное время
Укажите начальное время для расписания. Формат начального времени: ЧЧ:мм:сс.
Интервал
Укажите интервал запуска расписания: ежедневно, еженедельно или ежемесячно.

Нажмите Далее.

Этап 2: Указать задание

Задает путь определения задания для выполнения в качестве нового задания.

Локальная файловая система
Просматривает определение задания из локальной файловой системы.
Удаленная файловая система
Просматривает определение задания из системы хранилища заданий.

Укажите путь и нажмите кнопку Далее.

Этап 2.1: Указать свойства подстановки

Задает значения свойств подстановки для этого задания.

Этот этап доступен, если определение задания содержит свойства подстановки.

Укажите значения и нажмите кнопку Далее.

Этап 3: Подтвердить создание расписания

Выводит общий список всех выбранных значений.

Для завершения запуска этого задания нажмите кнопку Готово.

Ссылки, помеченные как (в сети), требуют подключения к Internet.
