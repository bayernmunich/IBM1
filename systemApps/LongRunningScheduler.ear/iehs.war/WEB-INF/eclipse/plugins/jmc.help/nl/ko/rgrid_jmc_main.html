작업 관리 콘솔 시작

작업 관리 콘솔은 사용자가 작업을 수행할 때 사용하는 독립형 웹 인터페이스입니다.

이 작업 관리 콘솔 페이지를 보려면 작업 관리 콘솔 시작 페이지에서 도움말을 클릭하십시오.

작업 관리 콘솔에 액세스하려면 관리 콘솔을 통해 lrsubmitter 역할, lradmin 역할 또는 두 역할 권한을 부여 받아야 합니다.

역할 권한에 따라 사용자가 다양한 작업을 수행할 수 있습니다. 작업 관리 콘솔은 보안이 사용될 때 제어된 액세스를 제공합니다.

(온라인)으로 표시된 링크는 인터넷에 액세스해야 합니다.
