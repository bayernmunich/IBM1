Отправка задания

На этой странице можно запустить задание, указав его определение. Определение задания может быть создано в локальной файловой системе.

Для просмотра страницы консоли управления заданиями выберите Управление заданиями > Отправить задание.

Права доступа на консоли управления заданиями различаются в зависимости от назначенных ролей. На этой странице консоли запускать задания могут пользователи, имеющими роль lrsubmitter или lradmin.

Локальная файловая система

Укажите путь определения задания для выполнения в качестве нового задания.

Нажмите кнопку Обзор, чтобы найти и указать полный путь локального файла.

Хранилище заданий

Указать имя задания.

Если определение задания хранится в хранилище заданий, выберите Хранилище заданий.

Свойства подстановки

Выберите обновление значений свойств подстановки для задания.

Если задание имеет свойства подстановки без значений, необходимо указать их.

Отложенное выполнение

Позволяет указать задержку начальной даты и времени запуска задания.

Для отправки задания в планировщик нажмите кнопку Отправить, для отмены подстановки нажмите кнопку Отмена.

Ссылки, помеченные как (в сети), требуют подключения к Internet.
