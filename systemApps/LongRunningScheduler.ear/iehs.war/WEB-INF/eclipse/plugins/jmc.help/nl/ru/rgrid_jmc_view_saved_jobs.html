Просмотр сохраненных заданий

На этой странице можно просмотреть список всех заданий, сохраненных в хранилище заданий.

Для просмотра этой страницы выберите Хранилище заданий > Показать сохраненные задания.

Будет показан список сохраненных определений заданий.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли. Пользователь с ролью lrsubmitter или lradmin может просматривать сохраненные задания. При включенной защите группы пользователь в группе может просматривать сохраненные задания в этой же группе.

При включенной защите группы и защите роли пользователь в группе может просматривать сохраненные задания в группе. Пользователю может быть назначена роль lrsubmitter или lradmin.

Имя

Задает имя задания. Для просмотра определения задания щелкните на ссылке имени.

Для удаления определения задания из хранилища нажмите кнопку Удалить.

Ссылки, помеченные как (в сети), требуют подключения к Internet.
