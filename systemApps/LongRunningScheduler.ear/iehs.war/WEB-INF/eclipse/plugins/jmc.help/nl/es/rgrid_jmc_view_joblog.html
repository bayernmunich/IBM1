Visualizar registro de trabajo

Utilice esta página para ver información de el registro de trabajo de todos los trabajos sometidos al planificador de trabajos y para descargar un registro de trabajo en el sistema local.

Para ver esta página de la consola de gestión de trabajos, pulse Gestión de trabajos > Ver trabajos > nombre del trabajo.

Utilice este panel para ver información de el registro de trabajo o para descargar el archivo de registro en el sistema local.

Los privilegios en la consola de gestión de trabajos varían según los roles asignados. Para esta página de la consola, un usuario puede realizar las siguientes acciones:

  • Gestionar trabajos que el usuario posee con el rol lrsubmitter.
  • Gestionar trabajos de todos los usuarios con el rol lradmin.

Renovar

Pulse este botón para renovar la pantalla.

Descargar

Pulse este botón para descargar este registro de trabajo en el sistema local.

Anterior

Pulse este botón para volver al panel de visualización del trabajo.

Los enlaces marcados (en línea) requieren acceso a Internet.
