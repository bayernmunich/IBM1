Показать расписания

На этой странице можно просмотреть информацию о расписании или отменить расписания заданий.

Для просмотра этой страницы выберите Управление расписаниями > Показать расписания.

На странице будет показан список расписаний.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли.
  • Пользователи с ролью lradmin могут просматривать расписания и отменять их для заданий.
  • При включенной защите группы любой пользователь группы может просматривать расписания в этой группе, а также отменять расписания для заданий в группе.
  • При включенной защите группы и защите роли пользователь может выполнить следующие действия:
    • Просмотреть расписания и отменить их для заданий (если пользователю назначена роль lradmin).

Для фильтрации и сортировки расписаний воспользуйтесь функцией фильтра. Щелкните на значке Показать фильтр , введите критерии фильтра и нажмите Применить. Для обновления информации на панели нажмите кнопку Обновить.

Имя

Задает имя расписания.

Для просмотра определения расписания щелкните на ссылке Имя. Для отмены расписания выберите его и нажмите кнопку Отмена.

Инициатор

Указывает инициатора расписания.

Начальные дата и время

Задает начальные дату и время для расписания.

Интервал

Задает временной интервал для выполнения расписания.
