Bem-vindo ao Console do Gerenciamento de Tarefas

O console de gerenciamento de tarefa é uma interface da Web independente para usuários que executam operações de tarefa.

Para visualizar esta página de console de gerenciamento de tarefa, clique em Ajuda na página de boas-vindas do console de gerenciamento de tarefa.

A função lrsubmitter, lradmin ou ambas devem ser concedidas a você por meio do console administrativo para acessar o console de gerenciamento de tarefa.

Dependendo dos privilégios de função, os usuários podem executar várias operações de tarefa. O console de gerenciamento de tarefas fornece acesso controlado quando a segurança está ativada.

Links marcados (on-line) requerem acesso à Internet.
