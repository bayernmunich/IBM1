Обновление расписания

Данная панель позволяет просмотреть и обновить задание и его расписание.

Для просмотра страницы выберите Управление расписанием > Показать расписания > Выберите-расписание.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли.
  • Для обновления расписания требуется пользователь с ролью lradmin.
  • Пользователь с ролью lrsubmitter или lrmonitor не может обновить расписание.
  • При включенной защите группы пользователь в группе может обновить расписание для задания в группе.
  • При включенной защите группы и защите роли пользователь в группе может обновить расписание для задания в группе. Пользователю должна быть назначена роль lradmin.

Для сохранения изменений нажмите кнопку Применить или OK. Для отмены изменений нажмите кнопку Отмена. Нажмите кнопку Сброс для удаления изменений данных и восстановления последних сохраненных значений.

Расписание

Изменение начальной даты, начального времени и интервала действия расписания.

Задание

Показать содержимое задания или заменить определение задания на этой панели.

  • Щелкните на ссылке рядом с полем Задание для выполнения для просмотра задания, связанного с расписанием, которое требуется изменение.

    Необязательно, можно обновить свойства подстановки.

  • Для изменения определения задания включите переключатель Заменить определение задания. Заменить определение задания из локальной файловой системы или из хранилища заданий.

Ссылки, помеченные как (в сети), требуют подключения к Internet.
