작업 보기

이 페이지에서 작업 스케줄러에 제출된 모든 작업의 목록을 보고 작업에 대한 정보를 볼 수 있습니다.

이 작업 관리 콘솔 페이지를 보려면 작업 관리 > 작업 보기를 클릭하십시오.

작업에서 조치를 적용하려면 작업과 조치를 선택한 다음 적용을 클릭하십시오.

필터 기능을 사용하여 작업을 필터링 및 정렬하십시오. 필터 기능 표시 아이콘을 클릭하고 필터 기준을 입력한 후 이동을 클릭하십시오.

이 패널을 새로 고치려면 탐색에서 작업 보기를 클릭하거나 상태 열에서 해당 아이콘을 클릭하십시오.

작업 관리 콘솔의 권한은 지정된 역할에 따라 다릅니다. 이 콘솔 페이지의 경우, 사용자는 다음 조치를 수행할 수 있습니다.

  • lrsubmitter 역할을 사용하여 사용자가 소유한 작업을 관리할 수 있습니다.
  • lradmin 역할을 사용하여 모든 사용자의 작업을 관리할 수 있습니다.

작업 유형 및 상태에 따라 여러 가지 유형의 수행할 조치가 있습니다. 이 콘솔 페이지에서 lrsubmitter 역할 또는 lradmin 역할을 가진 사용자가 작업을 취소, 삭제, 다시 시작, 제거, 일시중단 또는 재개 등과 같은 조치를 수행할 수 있습니다.

작업 ID

작업에 대한 링크를 지정합니다. 작업 로그를 보려면 작업 ID의 링크를 클릭하십시오.

제출자

작업을 제출한 사용자의 ID를 지정합니다.

마지막 업데이트

작업이 마지막으로 업데이트된 날짜를 지정합니다.

상태

작업의 상태를 표시합니다.

노드

작업이 있는 노드를 지정합니다.

애플리케이션 서버

작업이 있는 애플리케이션 서버를 지정합니다.

그룹

그룹 기반의 보안을 사용하거나 그룹 기반 및 역할 기반의 보안을 사용하는 경우에는 작업에 할당된 그룹 이름을 지정합니다.

(온라인)으로 표시된 링크는 인터넷에 액세스해야 합니다.
