Visualizar Tarefas

Utilize a página para visualizar a lista de todas as tarefas enviadas ao planejador de tarefas e para visualizar informações sobre as tarefas.

Para visualizar esta página do console de gerenciamento de tarefa, clique em Gerenciamento de Tarefas > Visualizar tarefas.

Para aplicar a ação nas tarefas, selecione as tarefas e a ação, depois, clique em Aplicar.

Use a função de filtro para filtrar e classificar tarefas. Clique no ícone Mostrar função de filtro , insira os critérios de filtragem e, depois, clique em Ir.

Para atualizar este painel, clique em Visualizar Tarefas na navegação ou clique no ícone na coluna Status.

Os privilégios no console de gerenciamento de tarefas variam, dependendo das funções designadas. Para essa página do console, um usuário pode executar as seguintes ações:

  • Gerenciar tarefas que o usuário possui com a função lrsubmitter.
  • Gerenciar tarefas de todos os usuários com a função lradmin.

Há vários tipos de ações a serem executadas, dependendo do tipo e do estado da tarefa. Nesta página de console, os usuários com as funções lrsubmitter ou lradmin podem concluir ações como cancelar, limpar, reiniciar, remover, suspender ou continuar uma tarefa.

ID da Tarefa

Especifica o link para a tarefa. Para visualizar o log de tarefas, clique no link do ID da tarefa.

Requisitante

Especifica o ID do usuário que enviou a tarefa.

Última Atualização

Especifica a data em que a tarefa foi atualizada pela última vez.

Estado

Mostra o estado da tarefa.

Nó

Especifica o nó em que a tarefa está localizada.

Servidor de Aplicativos

Especifica o servidor de aplicativos no qual a tarefa está localizada.

Grupo

Especifica o nome do grupo designado a uma tarefa quando a segurança baseada em grupo é ativada ou quando a segurança baseada em grupo e baseada em função é ativada.

Links marcados (on-line) requerem acesso à Internet.
