Atualizar Planejamento

Use este painel para visualizar ou atualizar uma tarefa e seu planejamento.

Para visualizar esta página, clique em Gerenciamento de Planejamento > Visualizar Planejamentos > Select_schedule.

Privilégios no console de gerenciamento de tarefa variam, dependendo da segurança do grupo e das atribuições designadas.
  • Um usuário na função lradmin pode atualizar um planejamento.
  • Um usuário na função lrsubmitter ou lrmonitor não pode atualizar um planejamento.
  • Quando a segurança do grupo está ativada, um usuário no grupo pode atualizar um planejamento para a tarefa no grupo.
  • Quando a opção para segurança do grupo e segurança da função está ativada, um usuário no grupo pode atualizar um planejamento para uma tarefa no grupo. O usuário deve estar na função lradmin.

Clique em Aplicar ou OK para salvar suas mudanças. Clique em Cancelar para cancelar suas mudanças. Clique em Reconfigurar para limpar suas mudanças no painel do console e restaurar os valores salvos mais recentemente.

Planejamento

Altere a data de início, o horário de início e o intervalo do planejamento.

Tarefa

Visualize o conteúdo de uma tarefa ou substitua a definição de tarefa neste painel.

  • Clique no link próximo de Tarefa para Executar para visualizar o conteúdo da tarefa associada ao planejamento que deseja alterar.

    Opcionalmente, é possível atualizar as propriedades de substituição.

  • Clique na caixa de seleção Substituir Definição de Tarefa para alterar a definição da tarefa. É possível substituir a definição de tarefa de um sistema de arquivos local ou do repositório de tarefa.

Links marcados (on-line) requerem acesso à Internet.
