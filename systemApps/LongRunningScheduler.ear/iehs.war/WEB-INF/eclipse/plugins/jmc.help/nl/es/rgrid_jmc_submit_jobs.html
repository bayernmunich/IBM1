Someter un trabajo

Utilice esta página para someter un trabajo especificando la definición del trabajo. La definición del trabajo puede provenir del sistema de archivos local.

Para ver esta página de la consola de gestión de trabajos, pulse Gestión de trabajos > Someter un trabajo.

Los privilegios en la consola de gestión de trabajos varían según los roles asignados. Para esta página de la consola, los usuarios con el rol lrsubmitter o el rol lradmin pueden someter trabajos.

Sistema de archivos local

Especifique la vía de acceso de la definición del trabajo para someterlo como un nuevo trabajo.

Utilice el botón Examinar para localizar y especificar la vía de acceso completa de un archivo local.

Repositorio de trabajos

Especifique el nombre del trabajo.

Seleccione Repositorio de trabajos si la definición del trabajo se almacena en el repositorio de trabajos.

Propiedades de sustitución

Seleccione esta opción para actualizar los valores de las propiedades de sustitución del trabajo.

Si un trabajo tiene propiedades de sustitución sin valores, debe especificarlas.

Retardar envío

Seleccione esta opción para retardar la fecha y hora de inicio de la ejecución del trabajo.

Pulse Someter para someter el trabajo al planificador o pulse Cancelar para cancelar el envío.

Los enlaces marcados (en línea) requieren acceso a Internet.
