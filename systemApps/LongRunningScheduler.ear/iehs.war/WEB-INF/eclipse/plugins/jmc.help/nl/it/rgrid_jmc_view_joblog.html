Visualizza log del lavoro

Utilizzare questa pagina per visualizzare i log di tutti i lavori inoltrati allo scheduler lavori e per scaricare un log del lavoro sul proprio file system locale.

Per visualizzare questa pagina della console di gestione lavori, fare clic su Gestione lavori > Visualizza lavori > nome lavoro.

Utilizzare questo pannello per visualizzare le informazioni del log del lavoro oppure per scaricare il file di log sul proprio file system locale.

I privilegi nella console di gestione dei lavori variano in base ai ruoli assegnati. In questa pagina della console, un utente può effettuare le azioni riportate di seguito:

  • Gestire lavori che appartengono all'utente a cui è stato assegnato il ruolo lrsubmitter.
  • Gestire lavori di tutti gli utenti con il ruolo lradmin.

Aggiornamento

Fare clic per aggiornare il pannello.

Scarica

Fare clic per scaricare questo log del lavoro sul proprio file system locale.

Indietro

Fare clic per tornare al pannello Visualizza lavoro.

I link contrassegnati (on-line) richiedono accesso a Internet.
