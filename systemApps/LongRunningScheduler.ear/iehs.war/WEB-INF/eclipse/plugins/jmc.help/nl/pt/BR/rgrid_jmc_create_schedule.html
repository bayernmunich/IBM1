Criar um Planejamento

Use o assistente para criar um planejamento de eventos que ocorrem para uma tarefa.

Para visualizar esta página, clique em Gerenciamento de Planejamento > Criar um Planejamento.

Use o assistente para especificar informações.

Privilégios no console de gerenciamento de tarefa variam, dependendo da segurança do grupo e das atribuições designadas.
  • Usuários na função lradmin podem criar planejamentos.
  • Usuários na função lrmonitor ou função lrsubmitter não podem criar planejamentos.
  • Quando a segurança do grupo está ativada, qualquer usuário no grupo pode criar planejamentos para tarefas no grupo.
  • Quando a opção para segurança do grupo e segurança da função está ativada, um usuário no grupo pode criar um planejamento para uma tarefa no grupo. O usuário deve estar na função lradmin.

Etapa 1: Criar Planejamento

Todos os campos deste painel são obrigatórios.

Nome
Especifique o nome do planejamento que deseja criar.
Data de início
Especifique a data de início para o planejamento começar. A data de início deve estar no formato dd-MM-aaaa.
Horário de Início
Especifique o horário de início para o planejamento começar. O horário de início deve estar no formato HH:mm:ss.
Intervalo
Especifique o período do intervalo como diário, semanal ou mensal para execução do planejamento.

Clique em Avançar.

Etapa 2: Especificar Tarefa

Especifica o caminho da definição de tarefa para enviar como uma nova tarefa.

Sistema de Arquivos Local
Navega para a definição de tarefa a partir do sistema de arquivos local.
Sistema de Arquivos Remoto
Navegue para a definição de tarefa a partir do sistema de repositório.

Especifique o caminho e clique em Avançar.

Etapa 2.1: Especificar Propriedades de Substituição

Especifica valores para propriedades de substituição para essa tarefa.

Esta etapa será aberta se a definição de tarefa contiver propriedades de substituição.

Especifique valores e clique em Avançar.

Etapa 3: Confirmar Criação de Planejamento

Lista um resumo de suas seleções.

Para concluir o envio do planejamento, clique em Concluir.

Links marcados (on-line) requerem acesso à Internet.
