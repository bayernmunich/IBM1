Afficher les travaux

Utilisez cette page pour afficher la liste des travaux envoyés au planificateur des travaux et les informations relatives aux travaux.

Pour afficher cette page de la console de gestion des travaux, cliquez sur Gestion des travaux > Afficher les travaux.

Pour appliquer une action aux travaux, sélectionnez les travaux et l'action, puis cliquez sur Appliquer.

Utilisez la fonction de filtrage pour filtrer et trier les travaux. Cliquez sur l'icône Afficher la fonction de filtrage , entrez les critères de filtrage, puis cliquez sur Go.

Pour actualiser ce panneau, cliquez sur Afficher les travaux dans la zone de navigation ou cliquez sur l'icône dans la colonne d'état.

Les privilèges de la console de gestion des travaux varient selon les rôles assignés. Pour cette page de la console, un utilisateur peut exécuter les actions suivantes :

  • Gérer les travaux dont l'utilisateur est propriétaire à l'aide du rôle lrsubmitter.
  • Gérer les travaux de tous les utilisateurs à l'aide du rôle lradmin.

Il existe plusieurs types d'actions à exécuter, selon l'état et le type du travail. A partir de cette page de la console, les utilisateurs avec le rôle lrsubmitter ou lradmin peuvent effectuer des actions telles qu'annuler, purger, redémarrer, supprimer, suspendre ou reprendre un travail.

ID travail

Indique le lien vers le travail. Pour afficher le journal du travail, cliquez sur le lien de l'ID travail.

Emetteur

Indique l'ID de l'utilisateur qui a soumis le travail.

Dernière mise à jour

Indique la date de la dernière mise à jour du travail.

Etat

Indique l'état du travail.

Noeud

Indique le noeud sur lequel le travail est situé.

Serveur d'applications

Indique le serveur d'applications sur lequel le travail est situé.

Groupe

Définit le nom de groupe affecté à un travail lorsque la sécurité basée sur les groupes est activée ou lorsque la sécurité basée sur les groupes et sur les rôles est activée.

Les liens marqués (en ligne) requièrent un accès à Internet.
