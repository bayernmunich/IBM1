Inoltro di un lavoro

Utilizzare questa pagina per inoltrare un lavoro specificando la definizione lavoro. La definizione lavoro può derivare dal file system locale.

Per visualizzare questa pagina della console di gestione lavori, fare clic su Gestione lavori > Inoltra lavoro.

I privilegi nella console di gestione dei lavori variano in base ai ruoli assegnati. Per questa pagina della console, gli utenti a cui è assegnato il ruolo ù lrsubmitter oppure il ruolo lradmin possono inoltrare i lavori.

File system locale

Specificare il percorso della definizione lavoro da inoltrare come nuovo lavoro.

Utilizzare il pulsante Sfoglia per individuare e specificare il percorso completo di un file locale.

Repository lavori

Specificare il nome del lavoro.

Selezionare Repository lavori se la definizione del lavoro è memorizzata nel repository lavori.

Proprietà di sostituzione

Selezionare questa opzione per aggiornare i valori delle proprietà di sostituzione per il lavoro.

Se un lavoro dispone di proprietà di sostituzione senza valori, è necessario specificarli.

Inoltro del ritardo

Selezionare per ritardare la data e l'ora di inizio dell'esecuzione del lavoro.

Fare clic su Inoltra per inviare il lavoro allo scheduler, oppure fare clic su Annulla per annullare l'inoltro.

I link contrassegnati (on-line) richiedono accesso a Internet.
