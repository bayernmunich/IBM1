Ver trabajos

Utilice esta página para ver la lista de todos los trabajos sometidos al planificador de trabajos y para ver información sobre los trabajos.

Para ver esta página de la consola de gestión de trabajos, pulse Gestión de trabajos > Ver trabajos.

Para aplicar una acción en los trabajos, seleccione los trabajos y la acción y, a continuación, pulse Aplicar.

La función de filtro sirve para filtrar y ordenar trabajos. Pulse el icono Mostrar función de filtro , especifique los criterios de filtro y, a continuación, pulse Ir.

Para renovar este panel, pulse Ver trabajos en la navegación o pulse el icono en la columna Estado.

Los privilegios en la consola de gestión de trabajos varían según los roles asignados. Para esta página de la consola, un usuario puede realizar las siguientes acciones:

  • Gestionar trabajos que el usuario posee con el rol lrsubmitter.
  • Gestionar trabajos de todos los usuarios con el rol lradmin.

Hay varios tipos de acciones a realizar según el tipo y el estado del trabajo. Desde esta página de la consola, los usuarios con los roles lrsubmitter o lradmin pueden realizar acciones como cancelar, depurar, reiniciar, eliminar, suspender o reanudar un trabajo.

ID de trabajo

Especifica el enlace al trabajo. Para visualizar el registro de trabajo, pulse el enlace del ID de trabajo.

Sometedor

Especifica el ID del usuario que ha sometido el trabajo.

Última actualización

Especifica la fecha en que el trabajo se actualizó por última vez.

State

Muestra el estado del trabajo.

Node

Especifica el nodo en que se encuentra el trabajo.

Servidor de aplicaciones

Especifica el servidor de aplicaciones en que se encuentra el trabajo.

Grupo

Especifica el nombre de grupo asignado a un trabajo cuando está habilitada la seguridad basada en grupo o la seguridad basada en rol.

Los enlaces marcados (en línea) requieren acceso a Internet.
