Просмотр заданий

На этой странице можно просмотреть список всех заданий, переданных на выполнение планировщику заданий, а также просмотреть информацию о заданиях.

Для просмотра этой страницы консоли управления заданиями выберите Управление заданиями > Показать задания.

Для того чтобы выполнить действие для заданий, выберите задания и действие, а затем нажмите Применить.

С помощью фильтра можно отфильтровать и отсортировать задания. Щелкните на значке Показать фильтр , введите критерии фильтра и нажмите Применить.

Для того чтобы обновить эту панель, нажмите Просмотреть задания на панели навигации или щелкните на значке в столбце Состояние.

Права доступа на консоли управления заданиями различаются в зависимости от назначенных ролей. На этой странице консоли пользователи могут выполнять следующие действия:

  • Управлять заданиями, владельцами которых являются пользователи с ролью lrsubmitter.
  • Управлять заданиями от всех пользователей с ролью lradmin.

Существует несколько типов действий в зависимости от типа и состояния задания. На этой странице консоли пользователи, имеющие роль lrsubmitter или lradmin, могут выполнить такие действия как отмена, очистка, перезапуск, удаление, приостановка или возобновление задания.

ИД задания

Указывает ссылку на задание. Для просмотра протокола задания щелкните на ссылке ИД задания.

Инициатор

Указывает ИД пользователя, который передал задание на выполнение.

Последнее обновление

Задает дату последнего обновления задания.

Состояние

Показывает состояние задания.

Node

Указывает узел, на котором расположено задание.

Сервер приложений

Указывает сервер приложений, на котором расположено задание.

Группа

Задает имя группы, связанное с заданием при включенной защите на основе групп и на основе ролей.
