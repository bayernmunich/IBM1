Сохранение задания

На этой странице можно сохранить определение задания в хранилище заданий.

Для просмотра этой страницы выберите Хранилище заданий > Сохранить задание.

Права доступа в консоли управления заданиями зависят от защиты группы и назначенной роли.
  • Пользователь с ролью lradmin может сохранить задание, в пользователь с ролью lrsubmitter или lrmonitor - нет.
  • При включенной защите группы пользователь в группе может сохранить задание в группе.
  • При включенной защите группы и защите роли пользователь в группе может сохранить задание в группе. Пользователю должна быть назначена роль lradmin.

Имя задания

Задает имя задания.

Для сохранения определения задания нажмите кнопку Сохранить. Для возврата к началу нажмите кнопку Сброс.

Путь xJCL

Описывает путь xJCL.

Заменить задание, если указанное имя задания уже существует

Задает замену определения задания в хранилище, если задание существует. Определение задания будет заменено после нажатия кнопки Сохранить.

Выключенный переключатель означает, что при нажатии кнопки Сохранить определение задания заменено не будет.
