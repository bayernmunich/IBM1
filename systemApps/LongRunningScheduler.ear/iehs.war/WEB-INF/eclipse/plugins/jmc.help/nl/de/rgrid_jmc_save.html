Job speichern

Verwenden Sie diese Seite, ,um eine Jobdefinition im Jobrepository zu speichern.

Klicken Sie zum Anzeigen dieser Seite auf Jobrepository > Job speichern.

Die Berechtigungen in der Jobverwaltungskonsole variieren in Abhängigkeit von der Gruppenzugehörigkeit und den zugeordneten Rollen.
  • Ein Benutzer in der Rolle "lradmin" kann einen Job speichern, aber ein Benutzer in der Rolle "lrsubmitter" oder "lrmonitor" nicht.
  • Wenn die Gruppensicherheit aktiviert ist, kann ein Benutzer in der Gruppe einen Job in der Gruppe speichern.
  • Wenn die Option für die Gruppensicherheit und Rollensicherheit aktiviert ist, kann ein Benutzer in der Gruppe einen Job in der Gruppe speichern. Der Benutzer muss die Rolle "lradmin" haben.

Jobname

Gibt den Jobnamen an.

Klicken Sie auf Speichern, um die neue Jobdefinition zu speichern. Klicken Sie auf Zurücksetzen, um von vorn anzufangen.

xJCL-Pfad

Beschreibt den xJCL-Pfad.

Job ersetzen, wenn der angegebene Jobname bereits vorhanden ist

Gibt an, dass die Jobdefinition im Repository ersetzt werden soll, falls bereits eine Jobdefinition mit dem angegebenen Namen vorhanden ist. Die Jobdefinition wird ersetzt, wenn Sie auf Speichern klicken.

Wenn Sie das Kontrollkästchen abgewählt lassen, wird die Jobdefinition nicht ersetzt, wenn Sie auf Speichern klicken.

Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
