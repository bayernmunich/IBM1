Gespeicherte Jobs anzeigen

Verwenden Sie diese Seite, um eine Liste aller Jobs anzuzeigen, die im Jobrepository gespeichert sind.

Klicken Sie zum Anzeigen dieser Seite auf Jobrepository > Gespeicherte Jobs anzeigen.

Die Liste gespeicherter Jobdefinitionen wird angezeigt.

Die Berechtigungen in der Jobverwaltungskonsole variieren in Abhängigkeit von der Gruppenzugehörigkeit und den zugeordneten Rollen. Ein Benutzer in der Rolle "lrsubmitter" oder "lradmin" kann gespeicherte Jobs anzeigen. Wenn die Gruppensicherheit aktiviert ist, kann ein Benutzer in der Gruppe gespeicherte Jobs in derselben Gruppe anzeigen.

Wenn die Option für Gruppensicherheit und Rollensicherheit aktiviert ist, kann ein Benutzer in der Gruppe gespeicherte Jobs in der Gruppe anzeigen. Ein Benutzer kann die Rolle "lrsubmitter" oder "lradmin" haben.

Name

Gibt den Jobnamen an. Zum Anzeigen der Jobdefinition klicken Sie auf den Namenslink.

Klicken Sie auf Löschen, um eine Jobdefinition aus dem Repository zu löschen.

Mit (online) gekennzeichnete Links setzen einen Internet-Zugang voraus.
