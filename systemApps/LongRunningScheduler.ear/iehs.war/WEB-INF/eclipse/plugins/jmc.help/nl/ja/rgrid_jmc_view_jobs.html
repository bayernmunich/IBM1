ジョブの表示

このページを使用して、ジョブ・スケジューラーにサブミットされたすべてのジョブのリストを表示し、ジョブに関する情報を表示します。

このジョブ管理コンソール・ページを表示するには、「ジョブ管理 (Job Management)」 > 「ジョブの表示 (View jobs)」をクリックします。

ジョブに対してアクションを適用するには、ジョブとアクションを選択して、「適用」をクリックします。

ジョブをフィルタリングおよびソートする場合は、フィルター機能を使用します。フィルター機能の表示アイコン をクリックし、フィルター基準を入力して、「実行」をクリックします。

このパネルをリフレッシュするには、ナビゲーションで「ジョブの表示」をクリックするか、「状況」列のアイコンをクリックします。

ジョブ管理コンソールでの特権は、 割り当てられているロールによって異なります。このコンソール・ページで、ユーザーは以下のアクションを実行できます。

  • lrsubmitter ロールでのユーザーが所有するジョブの管理。
  • lradmin ロールでの全ユーザーのジョブの管理。

ジョブ・タイプと状態に応じて複数のアクション・タイプがあります。lrsubmitter または lradmin ロールを持つユーザーは、このコンソール・ページから、ジョブのキャンセル、パージ、再始動、削除、中断、または再開などのアクションを実行できます。

ジョブ ID

ジョブへのリンクを指定します。ジョブ・ログを表示するには、ジョブ ID のリンクをクリックします。

実行依頼者

ジョブをサブミットしたユーザーの ID を指定します。

最終更新

ジョブが最後に更新された日付を指定します。

状態

ジョブの状態を表示します。

ノード

ジョブが置かれているノードを指定します。

アプリケーション・サーバー

ジョブが置かれているアプリケーション・サーバーを指定します。

グループ

グループ・ベースのセキュリティーが有効化されたとき、またはグループ・ベースとロール・ベースのセキュリティーが有効化されたときにジョブに割り当てられたグループ名を指定します。

マーク付きのリンク (オンライン) では、インターネットにアクセスする必要があります。
