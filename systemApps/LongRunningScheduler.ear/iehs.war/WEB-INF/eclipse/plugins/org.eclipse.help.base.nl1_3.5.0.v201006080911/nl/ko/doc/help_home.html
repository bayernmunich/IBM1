도움말 시스템 사용

왼쪽에 있는 컨텐츠 프레임()에서 주제를 찾습니다. 주제를 클릭하여 표시합니다. 이전 및 다음 단추를 사용하여 표시된 주제의 히스토리 내에서 탐색합니다.

검색

문서의 특정 주제에 대한 주제를 빠르게 찾으려면 검색 필드에서 조회할 사항을 입력하십시오. 검색 프레임()을 사용하여 검색 보기를 표시하십시오. 관심이 있는 절만 선택하여 검색 범위를 좁힐 수 있습니다.

동기화

목차에 표시 단추()를 클릭하면 탐색 트리에서 해당 주제를 선택할 수 있습니다. 컨텐츠에 대한 링크 단추()는 탐색 트리를 현재 주제에 동기화시킵니다.

기능

애플리케이션에서 사용하지 않는 기능에 대한 문서를 표시하려면 모든 주제 표시 단추()를 선택하십시오. 이는 목차 및 검색 결과에 영향을 줍니다.

키워드 색인

사전순으로 정렬된 키워드의 색인을 표시하려면 색인 프레임 ()을 선택하십시오. 모든 제품에 색인이 포함된 것은 아닙니다.
