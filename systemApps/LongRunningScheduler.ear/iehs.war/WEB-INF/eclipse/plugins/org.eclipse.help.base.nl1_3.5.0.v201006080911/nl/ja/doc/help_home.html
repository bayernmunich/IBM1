ヘルプ・システムの使用

左側の「目次」フレーム () でトピックを見つけてください。 トピックを表示するには、それをクリックします。既に表示したトピックの履歴を調べるには、「戻る」および「進む」ボタンを使用してください。

検索

文書内で特定の件名のトピックを短時間で見つけるには、 照会を「検索」フィールドに入力してください。「検索」ビューを表示するには、 「検索」フレーム () を使用します。目的のセクションのみを選択することによって、検索の範囲を限定できます。

同期化

「目次に表示」ボタン () をクリックすると、ナビゲーション・ツリーでそのトピックが選択されます。The 「コンテンツとリンク」ボタン () はナビゲーション・ツリーと現在のトピックの同期をとります。

機能

アプリケーションで使用不可にされている機能に関する文書を表示するには、 「すべてのトピックを表示」ボタン () を選択します。これは、目次および検索結果の表に影響を与えます。

キーワード索引

アルファベット順にソートされたキーワード索引を表示するには、the 「索引フレーム」 () を選択します。 すべての製品に索引があるわけではありません。
