Utilizar o sistema de ajuda

Pesquise tópicos na moldura Índice () do lado esquerdo. Faça clique num tópico para que seja apresentado. Utilize os botões Retroceder e Avançar para navegar no histórico de tópicos visualizados.

Procurar

Para localizar rapidamente tópicos sobre um assunto específico na documentação, introduza uma consulta no campo Procurar. Utilize a moldura Procura () para que apresentar a vista Procura. Pode limitar o âmbito da procura, seleccionando apenas as secções relevantes.

Sincronizar

Ao fazer clique no botão Mostrar no índice (), irá seleccionar esse tópico na árvore de navegação. O botão Ligar ao índice () mantém a árvore de navegação sincronizada com o tópico actual.

Funções

Para que seja apresentada documentação relativa a funcionalidades que estão desactivadas na aplicação, seleccione o botão Mostrar todos os tópicos () . Este procedimento afecta o índice e os resultados da procura.

Índice remissivo de palavras-chave

Para que seja apresentado um índice de palavras-chave ordenado alfabeticamente, seleccione Moldura do índice remissivo (). Nem todos os produtos contêm um índice remissivo.
