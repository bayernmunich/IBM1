Использование справочной системы

Разделы показываются во фрейме Оглавление () слева. Нажмите раздел для его просмотра. Кнопки Назад и Вперед позволяют перемещаться по хронологии просмотра разделов.

Поиск

Для быстрого поиска в документации разделов по конкретной теме введите запрос в поле Поиск. Панель Поиск открывается с помощью фрейма Поиск (). Область поиска можно сузить, выбрав только интересующие подразделы.

Синхронизация

Кнопка Показать в оглавлении () позволяет выбрать раздел в дереве навигации. Кнопка Связать с оглавлением () включает режим синхронизации дерева навигации с текущим разделом.

Группы функций

Для просмотра документации по отключенным группам функций приложения нажмите кнопку Показать все разделы (). Это влияет на оглавление и результаты поиска.

Индекс

Для просмотра индекса, упорядоченного по алфавиту, выберите фрейм Индекс (). Не все продукты содержат индекс.
