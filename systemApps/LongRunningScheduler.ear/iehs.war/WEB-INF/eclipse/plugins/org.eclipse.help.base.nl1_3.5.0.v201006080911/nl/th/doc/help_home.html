การใช้ระบบวิธีใช้

เรียกดูหัวข้อในกรอบ เนื้อหา () ทางด้านซ้าย คลิกบนหัวข้อเพื่อแสดงเนื้อหาของหัวข้อนั้น ใช้ปุ่ม ย้อนกลับ และ เดินหน้า เพื่อนำทางภายในหัวข้อที่ดูผ่านมา

การค้นหา

เมื่อต้องการค้นหาหัวข้อบนหัวเรื่องเฉพาะอย่างรวดเร็วในเอกสารคู่มือ ให้พิมพ์เคียวรีในฟิลด์ ค้นหา ใช้กรอบ ค้นหา () เพื่อแสดงผลมุมมองค้นหา คุณสามารถจำกัด ขอบเขต ของการค้นหาได้โดยเลือกเฉพาะส่วนที่คุณสนใจ

การซิงโครไนซ์

คลิกปุ่ม แสดงในสารบัญ () จะเลือกหัวข้อในแผนผังนำทาง ปุ่ม ลิงก์พร้อมข้อคิดเห็น () ทำให้แผนผังนำทางซิงโครไนซ์กับหัวข้อปัจจุบัน

คุณลักษณะต่างๆ

เมื่อต้องการแสดงเอกสารเกี่ยวกับคุณลักษณะที่ถูกปิดใช้งานในแอ็พพลิเคชัน ให้เลือกปุ่ม แสดงหัวข้อทั้งหมด () ซึ่งส่งผลกับสารบัญและผลการค้นหา
