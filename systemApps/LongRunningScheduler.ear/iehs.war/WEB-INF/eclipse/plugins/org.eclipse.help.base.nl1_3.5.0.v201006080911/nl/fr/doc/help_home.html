Utilisation du système d'aide

Parcourez les rubriques dans le cadre Sommaire () situé sur la gauche. Pour afficher une rubrique, cliquez dessus. Les boutons Précédent et Suivant permettent de naviguer dans l'historique des rubriques consultées.

Recherche

Pour trouver rapidement dans la documentation des rubriques traitant d'un sujet particulier, entrez une requête dans la zone Rechercher. Utilisez l'icône Rechercher () pour afficher la vue de recherche. Vous pouvez réduire la portée de la recherche en sélectionnant uniquement les sections qui vous intéressent.

Synchronisation

Le bouton Afficher dans la table des matières () sélectionne la rubrique dans l'arborescence de navigation. Le bouton Lier au sommaire () permet de maintenir l'arborescence synchronisée sur la rubrique en cours.

Fonctions

Pour afficher la documentation concernant des fonctions désactivées dans l'application, cliquez sur le bouton Afficher toutes les rubriques (). Cette opération affecte la table des matières et les résultats de recherche.

Index des mots-clés

Vous pouvez afficher l'index des mots-clés trié par ordre alphabétique en sélectionnant le cadre Index (). Tous les produits n'ont pas nécessairement d'index.
