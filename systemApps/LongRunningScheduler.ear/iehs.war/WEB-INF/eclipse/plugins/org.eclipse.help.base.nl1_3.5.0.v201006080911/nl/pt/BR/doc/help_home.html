Utilizando o sistema de ajuda

Navegue nos tópicos no quadro Conteúdo () à esquerda. Clique em um tópico para exibi-lo. Utilize os botões Voltar e Avançar para navegar dentro do histórico de tópicos visualizados.

Procurando

Para localizar rapidamente tópicos sobre um assunto em particular na documentação, digite uma consulte no campo Procurar. Utilize o quadro Procurar () para exibir a visualização Procura. É possível limitar o escopo de sua procura selecionando apenas as seções nas quais você está interessado.

Sincronizando

Clique no botão Mostrar no Índice () para selecionar esse tópico na árvore de navegação. O botão Link para Conteúdos () mantém a árvore de navegação sincronizada com o tópico atual.

Recursos

Para mostrar a documentação sobre os recursos que estão desativados no aplicativo, selecione o botão Mostrar Todos os Tópicos () . Isso afeta o índice e os resultados da procura.

Índice de Palavras-chaves

Para mostrar um índice de palavras-chaves classificadas alfabeticamente, selecione Quadro de Índice (). Nem todos os produtos contêm um Índice.
