使用說明系統

在左側的內容頁框 （） 中瀏覽主題。按一下主題可顯示該主題。使用上一頁和下一頁按鈕導覽檢視過的主題歷程。

搜尋

如果要在說明文件中快速找出有關特定主旨的主題，請在搜尋欄位中輸入查詢。使用搜尋頁框（）以顯示「搜尋」視圖。您可以僅選取您有興趣的章節，縮小搜尋範圍。

同步

按一下顯示在目錄中按鈕（）將會在導覽樹狀結構中選取該主題。與內容鏈結按鈕（）會讓導覽樹狀結構與目前的主題保持同步。

功能

如果要顯示已於應用程式中停用之功能的相關說明文件，請選取顯示所有主題按鈕（） 。這將會影響目錄和搜尋結果。

關鍵字索引

如果要顯示以字母順序排序的關鍵字索引，請選取索引頁框 （）。 並非所有產品都包含索引。
