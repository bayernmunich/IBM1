Hilfefunktion verwenden

Blättern Sie durch die Themen im Inhaltsfenster (), das sich im linken Seitenbereich befindet. Klicken Sie auf ein Thema, um es anzuzeigen. Mithilfe der Schaltflächen Zurück und Weiter können Sie im Verlauf der angezeigten Themen blättern.

Suchen

Um schnell die Themen zu einem bestimmten Suchbegriff zu finden, geben Sie eine entsprechende Abfrage im Feld Suchen ein. Die Suchergebnisse werden im Frame Suchen () angezeigt. Sie können den Suchbereich eingrenzen, indem Sie nur die für Sie relevanten Abschnitte auswählen.

Synchronisieren

Wenn Sie auf die Schaltfläche Im Inhaltsverzeichnis anzeigen () klicken, wird das entsprechende Thema in der Navigationsstruktur angezeigt. Die Schaltfläche Mit Inhalt verlinken () bewirkt, dass die Navigationsstruktur stets mit dem aktuellen Thema synchron bleibt.

Funktionen

Klicken Sie auf die Schaltfläche Alle Themen einblenden (), damit die Dokumentation zu Funktionen angezeigt wird, die in der Anwendung inaktiviert sind. Diese Option betrifft das Inhaltsverzeichnis und die Suchergebnisse.

Suchbegriffindex

Sie können einen alphabetisch sortierten Index der Suchbegriffe anzeigen, indem Sie das Indexteilfenster () auswählen. Nicht für alle Produkte ist ein Index verfügbar.
