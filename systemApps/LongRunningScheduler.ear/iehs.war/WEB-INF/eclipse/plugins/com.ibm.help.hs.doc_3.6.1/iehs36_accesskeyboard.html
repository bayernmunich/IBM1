Keyboard shortcuts and accelerators

You can use keys or key combinations to perform operations that can also be done using a mouse.

The following tables list the keyboard shortcuts and accelerators that can be used in the help system for the Microsoft Internet Explorer and Mozilla Firefox web browsers.
Note: In Internet Explorer, use the accelerator keys to move focus to the control element and press Enter to activate. In Firefox, use the accelerator keys to move focus and activate the control element at the same time.
Table 1. Keyboard shortcuts and accelerators for navigating the help system using Internet Explorer
Action Shortcut for Internet Explorer
Go to the next link, button, or topic branch from inside a frame (page) Tab, then press Enter to follow the link
Move to the Contents View frame Alt+C, then press Enter and Shift+F6
Move to the Index View frame Alt+I, then press Enter and Shift+F6
Move to the Search Results View frame Alt+R, then press Enter and Shift+F6
Move to the Topic View frame Alt+K, then press Enter
Expand or collapse a topic branch Right Arrow or Left Arrow
Move to the next topic branch Down Arrow or Tab
Move to the previous topic branch Up Arrow or Shift+Tab
Scroll to the top Home
Scroll to the bottom End
Next frame F6
Previous frame Shift+F6
Print the current page or active frame Ctrl+P
Table 2. Keyboard shortcuts and accelerators for navigating the help system using Firefox
Action Shortcut for Firefox
Go to the next link, button, or topic branch from inside a frame (page) Tab, then press Enter to follow the link
Move to the Contents View frame Shift+Alt+C and Shift+F6
Move to the Index View frame Shift+Alt+I and Shift+F6
Move to the Search Results View frame Shift+Alt+R and Shift+F6
Move to the Topic View frame There is no direct shortcut to this action in Firefox. Use Tab or the navigation function in JAWS.
Expand or collapse a topic branch Right Arrow or Left Arrow
Move to the next topic branch Down Arrow or Tab
Move to the previous topic branch Up Arrow or Shift+Tab
Scroll to the top Home
Scroll to the bottom End
Next frame F6
Previous frame Shift+F6
Print the current page or active frame Ctrl+P
