Accessibility and keyboard shortcuts in the help system

Accessibility features help users with physical disabilities, such as restricted mobility or limited vision, to work with software products successfully.

IBM® Eclipse Help System (IEHS) is an IBM Common Component. IBM products adopt IEHS to present documentation online for external customers.

In the IBM Accessibility Center (AC) Product Accessibility Database, IEHS is categorized as a Reusable Component, because it is a component that is reused within IBM products, rather than a component sold on its own to external customers.

  • Keyboard input and navigation
    You can navigate the help system by using the keyboard.
