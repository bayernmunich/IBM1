Finding information with the Index view

This topic describes how to work with the Index view to find topics in the help system.

About this task

To work with the Index view for information you want, take the following steps:

Procedure

  1. Click the Index tab () to display the Index view.
  2. Type a keyword in the input field. The index list will automatically scroll to the best matched keyword.
    Note:
    • If a keyword associates with more than one topic, it will not be shown in the Index. You can find its topic headings displayed in the Index view instead.
  3. Click the keyword, or a sub keyword, or a topic heading, to display the topic in the Content view.
    Note: Indexes are not case-sensitive.

    To toggle between the Navigation view and the Index view, click the Contents tab () or the Index tab ().
