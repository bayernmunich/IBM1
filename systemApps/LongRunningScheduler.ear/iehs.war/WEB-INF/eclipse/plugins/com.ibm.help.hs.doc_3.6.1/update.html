Updating information in the help system

You can regularly update the documentation contents, if available, by downloading it from the help system. The contents will be periodically updated at the Web site.

About this task

Note: The update function is only available in the stand-alone mode.

You can check and download available updates by performing the following steps:

Procedure

  1. Click Update () on the toolbar. You will see a list of installed documentation sets.
    Note: The Update icon is not available if there are no information updates. And you can set the Update icon () either visible or invisible in the content view toolbar by specifying the -showUpdater parameter in the start script.
  2. Click Next at the bottom to see a list of Updates for existing information collections and New information collections. The available updates are stored at an update site.
  3. Select the documentation that you intend to install, and then click Install Updates.
  4. Click Finish when the installation is finished. You do not need to restart the help system because it will refresh automatically.
