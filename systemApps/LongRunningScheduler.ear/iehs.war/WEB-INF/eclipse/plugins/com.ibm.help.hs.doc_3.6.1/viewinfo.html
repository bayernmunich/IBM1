Viewing information in the help system

You can learn how to navigate and search in the help system by using the various buttons on the toolbar.

About this task

  • Prerequisite requirements for the help system
    When you set up the IBM® Eclipse Help System, make sure that your workstation, operation system, browser, and hardware meet the requirements described in this topic.
  • Navigating in the help system
    Use the graphical user interface provided by the help system to locate and view product documentation.
  • Searching the help system
    You can search all topics in the help system, search an individual topic or topic collection, or create a search scope to search only a subset of the information. The Search Results view can display up to 500 results, sorted by relevance.
  • Setting the appropriate scope
    You can narrow down your working scope by setting the appropriate filter criteria.
  • Finding information with the Index view
    This topic describes how to work with the Index view to find topics in the help system.
  • Setting bookmarks in the help system
    You can find a topic more easily by setting a bookmark and saving it to the Bookmarks view of the help system.
  • Viewing information in different languages
    To view the help system user interface in another language, you have to change the regional settings of your operating system to the locale and language of your choice.
  • Printing in the help system
    IBM Eclipse Help System provides you with different ways of printing the help contents within the system.
  • Accessibility features in the help system
    Accessibility features help people with a physical disability, such as restricted mobility or limited vision, or with other special needs, use information technology products successfully.
  • Updating information in the help system
    You can regularly update the documentation contents, if available, by downloading it from the help system. The contents will be periodically updated at the Web site.
