External configuration

Use this page to edit a configuration file for a middleware server.

To view this administrative console page, click Servers > Server types > middleware_server_type > middleware_server > External configuration tab, or Servers > All servers > middleware_server > External configuration tab.

External configuration

Displays the editable configuration files.

To change the configuration of a PHP server template, edit the httpd.conf and php.ini files. The httpd.conf file is the Apache server configuration file, and you can define variables in this file for use in the WebSphere Application Server administrative console. The php.ini file contains data regarding the PHP module.

Retrieve

Retrieves the selected file from the server.

Apply

Applies the edits that you have made to the file.

Reset

Resets the file to its original format before you made changes, or before you clicked Apply.
