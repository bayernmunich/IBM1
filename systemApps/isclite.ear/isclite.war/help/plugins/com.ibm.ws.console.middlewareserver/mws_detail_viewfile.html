Log viewer

Use this page to view the log files for your middleware servers.

To view this administrative console page, click Servers > All servers > middleware_server > Log viewer tab, or Servers > Server types > middleware_server_type > middleware_server > Log viewer tab.

View log files

Choose a log file from the list that you want to view.

Retrieve

Click Retrieve to retrieve the file from the server.
