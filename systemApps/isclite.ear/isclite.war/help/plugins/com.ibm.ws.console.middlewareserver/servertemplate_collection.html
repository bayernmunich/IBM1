Middleware server template collection

Use this page to create a new server template. You can use server templates to define the configuration settings when you are creating a middleware server.

To view this administrative console page, click Servers > All servers > Templates, or select Servers, choose a specific middleware server under Server types, and click Templates.

Button Resulting action
New Opens a wizard that helps you create a new Middleware server template. Server types from which you can create a template include application server, generic server, on demand router, proxy server, or Web server. You can also create or add server template for middleware servers that already exist in your environment such as PHP, Apache Tomcat, BEA WebLogic Server, JBoss, Custom HTTP server, external Websphere Application Server, Liberty profile server, or WebSphere Application Server Community Edition server.
Delete Removes the server template.

Name

Specifies the name of the server template.

Platform

Specifies the type of platform on which the server is running.

Version

Specifies the version of WebSphere Application Server Network Deployment that is installed and running on your server.

Description

Displays the description of the server template.
