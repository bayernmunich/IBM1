Middleware servers

Navigate to help topics within this section.

  • Create a middleware server
    Use this page to create a middleware server. Middleware servers are servers in the middleware tier that host applications or their data.
  • External configuration
    Use this page to edit a configuration file for a middleware server.
  • External configuration service
    Use this page to enable or disable configuration files associated with a middleware server.
  • External log service
    Use this page to access log files for your middleware servers.
  • Log viewer
    Use this page to view the log files for your middleware servers.
  • Middleware server collection
    Use this page to view and manage your middleware servers. Middleware servers are generic servers in the middleware tier that host applications or their data.
  • Middleware server operations settings
    Use this page to configure operations for middleware servers. You can use server operations to run executable files on your middleware servers from the administrative console.
  • Middleware server template collection
    Use this page to create a new server template. You can use server templates to define the configuration settings when you are creating a middleware server.
  • Middleware server template settings
    Use this page to configure the settings for a newly created server template. Existing server types from which to create a template include application, generic, on demand router, proxy, or Web server. You can also create or add server templates for a middleware server that already exists in your environment such as PHP, Custom HTTP server, or WebSphere Application Server Community Edition.
  • Middleware server detail settings
    Use this page to view the configuration for middleware servers.
  • Server operations
    Use this page to add, edit, and delete server operations.
