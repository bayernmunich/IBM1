External log service

Use this page to access log files for your middleware servers.

To view this administrative console page, click Servers > All servers > middleware_server > External log service, or Servers > Server types > middleware_server_type > middleware_server > External log service.

You must have administrative privileges to configure the service and view log files. Users with configurator privileges can use the service to view files only.

Configure the external log viewer service to provide view the log files for your middleware servers in the administrative console. You configure directories, allowing all files in the directory to be viewed through the log viewing service. The server templates provide a default configuration of the list of log directories for servers of that type. You can add or remove directories from the list if you have an the administrator role.

Enable remote log viewing service

Enables the log viewing service.

Check the box to enable the log viewing service. In the field provided, type the name of the directory to enable for viewing in the external log viewing service. Click Add to add the directory to the list of directories in the following table.

File path

Identifies the path and directory of the log files.
