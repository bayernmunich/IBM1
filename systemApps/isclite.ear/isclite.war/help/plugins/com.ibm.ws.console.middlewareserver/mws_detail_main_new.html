Create a middleware server

Use this page to create a middleware server. Middleware servers are servers in the middleware tier that host applications or their data.

To view this administrative console page, click Servers > All servers > New.

You must have administrator privileges to create middleware servers. For other administrative roles and privileges information, see the Reference section in the information center.

Add a server

Create a new server instance or add an existing middleware server. These servers have complete life cycle management.

If you created the middleware server outside of the administrative console, you can install the node agent on that server and add the existing server to the cell. This action creates a representation of the server in the administrative console that you can use to manage the externally created server. These servers have assisted life cycle management.

Create a new server instance:
  • Application server
  • Generic server
  • On demand router
  • PHP server
  • Proxy server
  • Web server
  • WebSphere Application Server Community Edition server
Add an existing server in the cell:
  • Custom HTTP server
  • BEA WebLogic server
  • JBoss server
  • PHP server
  • Apache Tomcat server
  • External WebSphere Application Server
  • WebSphere Application Server Community Edition server
  • Liberty profile server

Select a node

Select a node that corresponds to the server you want to create, and define a server name.

Select a runtime

For WebSphere Application Community Edition servers, choose a runtime for your server.

Select a server template

Select the template that best specifies the attributes of the server that you want to create.

Server templates are exist for the server types. You can also create your own server templates. To create your own server template, click Servers > All servers > Templates.

Confirm the new middleware server

The summary of actions field describes the choices you have made in the wizard. Click Finish to create your new middleware server.
