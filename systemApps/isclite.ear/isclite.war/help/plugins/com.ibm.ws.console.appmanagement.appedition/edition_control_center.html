Edition control center

The edition control center provides an overview of all enterprise applications and their current editions.

To view this administrative console page, click Applications > Edition control center.

If you are a user with either a monitor or an operator role, you can only view the edition manager information. If you have the role of configurator or administrator, you have all the configuration privileges for the application edition manager.

Click on an application name to view detailed information about the editions of that application.

Applications

Displays the application name.

Type

Displays the type of application.

Editions

Shows the total number of editions of the application installed in the cell.

Active

Shows the total number of editions of the application that are in the active state.

Validation

Shows the total number of editions of the application that are in the validation state.
