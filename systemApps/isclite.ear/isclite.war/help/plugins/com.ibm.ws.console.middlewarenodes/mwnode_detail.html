Middleware node settings

Use this page to define or change the configuration for the middleware node.

To view this administrative console page, click System administration > Middleware nodes > middleware_node_name.

Name

Specifies a name for the middleware node. The name must be unique among all the nodes in the cell. On this page, you cannot edit the node name value.

A node name usually is identical to the host name for the computer. However, you choose the node name. You can make the node name some name other than the host name.

Host name

Specifies the host name for the middleware node. On this page, you cannot edit the host name value.

Platform type

Specifies the operating system on which the middleware node runs.

Choose from the following platform types:
  • Windows
  • AIX®
  • HP-UX
  • Solaris
  • Linux
  • OS/400®
  • z/OS®
