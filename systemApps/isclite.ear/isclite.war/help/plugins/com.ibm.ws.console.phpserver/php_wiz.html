Create a new PHP server

Use this page to create PHP Hypertext Preprocessor (PHP) servers. PHP servers are defined as an Apache server with the mod_php module. By configuring PHP servers, you can manage your PHP applications in the same administrative environment as your Java Platform, Enterprise Edition (Java EE) applications.

You can access this administrative console page in several ways:
  • Click Servers > All servers > New. Select Create new server instance and choose the PHP server type.
  • Click Servers > New server. Select Create new server instance and choose the PHP server type.
  • Click Servers > Server types > PHP servers > New.

Step 1: Select a node

Select node
Choose the node on which you want the PHP server to run.
Server name
Specifies a name for the PHP server that is unique on the node.

The name field cannot contain the following characters: # \ / , : ; " * ? < > | = + & % ' . The name that is defined must be unique among servers and cannot begin with a period (.) or a space. A space does not generate an error, but leading and trailing spaces are automatically deleted.

Step 2: Select an Apache runtime

Choose an Apache runtime for your PHP server.

The page displays the valid Apache runtimes on the selected node, including the version and installation path.

Step 3: Select a PHP runtime

Choose a PHP runtime version for your PHP server.

The page displays the valid PHP runtimes on the selected node, including the version and installation path.

Step 4: Select a PHP template

Select a server template on which to model the servers for the new PHP server. You can select a template, or map an existing server.

  • Select Create the PHP server using a PHP server template to model the PHP server after a defined PHP server template. Choose a server template from the list.
  • Select Create the PHP server using an existing PHP server to model the PHP server after an existing PHP server. Choose a server from the list.

Step 5: Confirm new PHP server

A summary of the actions to create the PHP server displays. Click Finish to create the PHP server.
