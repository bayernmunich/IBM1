Middleware descriptors details

Middleware platform descriptors provide information about managed middleware platforms to the Intelligent Management runtime and discovery sensors.

To view this administrative console page, click System administration > Middleware descriptors > middleware_descriptor.

Name

Specifies the name of the middleware descriptor.

Automated discovery support

Specifies if automatic discovery is supported for this middleware platform type. PHP and Apache servers are the only middleware platform types that support automatic discovery.

Discovery interval

Specifies how often the node agent runs discovery for this platform type with the following values:

  • -1: no auto discovery
  • 0 : at agent startup
  • > 0: agent startup and the interval at which to run

Discovery interval units

Specifies the units of the discovery interval. The interval can be in seconds, minutes, hours, or days.

Versions

Specifies all the versions supported by the middleware descriptor. To view details about a specific version, click the link in the table.
