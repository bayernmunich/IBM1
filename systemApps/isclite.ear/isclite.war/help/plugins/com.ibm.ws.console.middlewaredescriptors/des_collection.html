Middleware descriptors collection

Middleware platform descriptors provide information about managed middleware platforms to the Intelligent Management runtime and discovery sensors.

To view this administrative console page, click System administration > Middleware descriptors.

Middleware Platform

Specifies all of the middleware platforms in your environment. To edit a middleware platform, click the middleware platform now.

Versions

Specifies all the versions that are supported by the middleware descriptor.
