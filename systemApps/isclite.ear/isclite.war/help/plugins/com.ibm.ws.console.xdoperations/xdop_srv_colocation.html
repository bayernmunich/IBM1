Server collocation

Use this page to view the processes that are running on the same node as the specified server.

To view this administrative page, click Servers > All servers > server_name > Operations tab > Collocation tab.

Server

Specifies the name of the server that is running on the node.

Cluster

Specifies the name of the cluster that is associated with the server.

Stability

Specifies the operational status of the server.

Table 1. Server stability
Icon Status Description
Stable The server is operational, without issues.
Questionable The server has some problems, but is still operational.
Unstable The server has severe issues.
Unknown The server stability cannot be reported.

Weight

Specifies the currently assigned routing weight of the server.

The weight determines the amount of work that is directed to the server. If the weight value for the server is greater than the weight values that are assigned to other servers in the cluster, the server receives a larger share of the cluster workload.

CPU utilization

Specifies the CPU utilization of the server. The value is indicated as a percentage of the total CPU capacity.
