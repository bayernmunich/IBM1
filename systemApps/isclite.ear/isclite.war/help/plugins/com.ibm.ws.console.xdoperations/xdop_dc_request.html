Dynamic cluster request management

Use this page to view the current status of clusters in your environment.

To view this administrative console page, click Servers > Clusters > Dynamic clusters > dynamic _cluster > Operations tab > Request management tab.

This view shows the on demand routers (ODR) that are currently routing work to the cluster, which applications are receiving work, and which service policies have had work classified to them since the ODR startup.

On demand routers

Specifies the ODRs that are associated with the cluster.

Applications

Specifies the applications that are associated with the cluster.

Service policies

Specifies the service policies that associated with the cluster after the ODR startup.
