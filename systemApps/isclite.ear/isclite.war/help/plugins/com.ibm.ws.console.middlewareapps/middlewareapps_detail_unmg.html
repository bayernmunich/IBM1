Install unmanaged Web applications

Use this wizard to install unmanaged Web applications.

To view this administrative console page, click Applications > Select Unmanaged Web application > Install new middleware application > Next.

Step 1: Define the general properties

Used to define the general properties of the unmanaged Web application.

Name

Specifies a name for the application. The name for a middleware unmanaged application must be unique among all the applications in the cell.

The name field cannot contain the following characters: # \ / , : ; " * ? < > | = + & % ' . The name cannot begin with a period (.) or a space. A space does not generate an error, but leading and trailing spaces are automatically deleted.

The name cannot be the same as any other defined application name in the cell. For example, an unmanaged Web application cannot have the same name as a Java Platform, Enterprise Edition (Java EE) application that is deployed in the same cell.

Edition

Specifies the current edition of the application.

Description

Optional field for specifying a description of the application.

Step 2: Define the deployment properties

Module name

Specifies the full remote or local path where the application is located.

Context root

Used for standalone Web modules (.war files) and SIP modules (.sar files).

Virtual host

This is required for all applications.

Deployment targets

Specifies the target where the unmanaged application will be installed.

Table 1. Deployment target action buttons
Button Resulting action
Add Select the deployment targets that you want added to the module list, then click Add.
Remove Select the deployment targets that you want deleted from the application or representation of the application from the configuration repository, then click Remove.
Apply After you add the deployment targets, click Apply to add the selected deployment targets to the module list.
New Allows you to create a new Web module within this panel.

Step 3: Confirm the new application representation

Confirms the application name, edition, and description of the application you want to add.

Click Finish to confirm the middleware application installation.
