On demand router routing policies

Use this page to configure routing policies for the on demand router.

To view this administrative console page, click Servers > Server types > On demand routers > ODR > On demand router properties > Generic server cluster routing policies.

To modify rules with the subexpression builder, you must have administrator administrative privileges.

From this tab you can expand HTTP work classes to view or manage the work classes for HTTP requests, and take the following actions:

Work classes for HTTP requests

Expands to view or manage the work classes for Hypertext Transfer Protocol (HTTP) requests.

New

Creates a new work class.

Use the wizard to create a new work class and specify general properties.

Delete

Deletes a work class.

Edit HTTP pattern

Use to add or remove a destination type to the work class members list.

Add rule

Builds a new rule.

See the Subexpression builder for more information.

Delete rule

Removes an existing rule.

Move up, move down

Prioritizes the rules in the case of multiple rules. The order of rules is key in determining matches. If a match occurs on the first rule, subsequent rules are not pursued.

If URI matches

Sets a Uniform Resource Identifier (URI) match condition for an existing rule. Type the name of the URI in this field.

For virtual host

Sets a URI match condition for an existing rule. Select a host name. You can use an asterisk wildcard ( * ) in the If URI matches field

Select action

Assigns the default routing action to one of the following options:
Permit routing to
From the Select generic server cluster name list, select the appropriate generic server cluster.
Reject routing with return code
In the Enter in return code field, type the HTTP return code.
Redirect routing to
In the Enter URI to redirect to field, type the URI.
Permit routing with affinity to
From the Select generic server cluster name list, select the appropriate generic server cluster.

Apply

Commits your new rule settings.

Click Apply to commit your rule setting.
