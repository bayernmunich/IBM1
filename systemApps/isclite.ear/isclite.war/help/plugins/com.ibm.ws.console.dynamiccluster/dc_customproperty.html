Dynamic cluster custom properties

Use this page to create or delete custom properties. For custom properties to function properly, the name and value pairs must be relevant. Custom properties set internal system configuration properties, so setting custom properties requires significant product knowledge.

To view this administrative console page, click Servers > Clusters > Dynamic clusters > dynamic_cluster_name > Custom properties.

You must have configurator privileges to create or delete dynamic clusters. For other administrative roles and privileges for dynamic clusters, see the Reference section of the Infocenter.

Name

Specifies a unique name for the custom property.

Each property name must be unique. If the same name is used for multiple properties, the value that is specified for the first property with that name is used. Do not start your property names with was. because this prefix is reserved for properties that are predefined in WebSphere® Application Server.

Value

Specifies the value that is paired with the specified name.

Description

Provides information about the name and value pair.
