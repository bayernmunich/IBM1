Add and remove setting

Use this page to define actions for the add and remove elasticity operation. Ensure that elasticity custom actions are created before you define actions for add and remove elasticity operations.

To view this administrative console page for the add operation, click Operational policies > Autonomic managers > Application placement controller > Elasticity operation > Add.

To view this administrative console page for the remove operation, click Operational policies > Autonomic managers > Application placement controller > Elasticity operation > Remove.

Name

Specifies the name of the elasticity operation

Description

Optionally specifies the description of the elasticity operation.

Elasticity operation

Specifies the type of elasticity operation. The value is Add for the add operation and Remove for the remove operation.

Add action, remove action

Specifies the actions for the elasticity operation that you can add to or remove from the table.

Move up, move down

Prioritizes the actions in the order that they run in the case of multiple actions.

Step

Specifies an integer value that is automatically assigned to the action, depending on the order that the action is listed in the table.

Action

Specifies the name of the action that you add when you click Add action.

Apply

Commits your new add settings.

Click Apply to commit your changes.
