Create a new on demand router

Use this page to create an on demand router.

To view this administrative console page, click Servers > All servers > New. Select Create new server instance and choose the on demand router type. You can also view this administrative console page by clicking Servers > Server types > On demand routers > New.

Step 1: Select a node

Select node
Choose the node on which you want the on demand router to run.
Server name
Specifies a name for the on demand router server that is unique in the cell.

The name field cannot contain the following characters: # \ / , : ; " * ? < > | = + & % ' . The name that is defined must be unique among servers and cannot begin with a period (.) or a space. A space does not generate an error, but leading and trailing spaces are automatically deleted.

Step 2: Specify server specific properties

Specify server specific properties.

Supported protocols
Supported protocols HTTP and SIP are checked by default. Clear the checkbox if you want to disable it.
Generate unique ports
The option to generate unique ports is selected by default. Clear the checkbox if you want to disable it.

Step 3: Select a server template

Choose an on demand router template from the list that best specifies the attributes of the on demand router you want to create.

The template type and description are displayed.

Step 4: Confirm new server

A summary of the actions to create the on demand router displays. Click Finish to create the on demand router
