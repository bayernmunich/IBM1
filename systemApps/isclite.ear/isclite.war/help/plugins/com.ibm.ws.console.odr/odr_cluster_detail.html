On demand router clusters configuration

Use this page to modify the configuration of an on demand router (ODR) cluster. An ODR cluster consists of a group of ODRs that share a common configuration.

To view this administrative console page, click Servers > Clusters > On demand router clusters > odr_cluster_name > Configuration.

To change the ODR cluster settings, you must have administrator or configurator administrative privileges.

Cluster name

Indicates a logical name for the ODR cluster.

This is a read-only field.

Bounding node group name

Indicates the name of the node group with which the ODR cluster is associated. The DefaultNodeGroup node group is displayed by default.
