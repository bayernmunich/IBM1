On demand router static cache rules

This topic lists the on demand router (ODR) static cache rules. On this page you can create, delete, or modify an on demand router static cache rule.

To view this administrative console page, click Servers > Server types > On demand routers > odr_name > On demand router properties > Static cache rules.

To change the ODR settings, you must have administrator or configurator administrative privileges.

Select

Select a rule to view its details.

URI groups

The unique name for groups of URIs that form a collection used as a basis for routing requests.

Disable caching for this URI group

Indicates whether caching is enabled or disabled for this URI group.

Default expiration

The default expiration value, in seconds, is used to determine the validity of a cached response when all other HTTP 1.1 caching-related response headers do not give guidance.

Last modified factor

The percentage of a Last-Modified header for a response that determines the validity of a cached response when the response does not have explicit HTTP expiration headers.

Name of virtual host

Name of the virtual host, along with the URI group, defines the address space to have cache customization performed.
