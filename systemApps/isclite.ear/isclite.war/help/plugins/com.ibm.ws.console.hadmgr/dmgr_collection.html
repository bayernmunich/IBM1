All deployment managers

This page displays the multiple deployment managers that comprise a highly available deployment manager configuration. A highly available deployment manager configuration ensures that if the active deployment manager fails, a deployment manager that is in standby mode can take over the administrative function of the cell.

To view this administrative console page, click System administration > Deployment manager > All Deployment managers.

Privileges for the high availability (HA) deployment manager differ, depending on the various roles. Roles include monitor, operator, configurator, and administrator. If you are a user with either a monitor or a configurator role, you can only view the HA deployment manager information. If you have the role of operator or administrator, you have all the privileges for the HA deployment manager.

Node

Specifies the node name of the deployment manager. All deployment managers have the name dmgr, so only the node name is meaningful for telling them apart.

Mode

Specifies if the deployment manager is in active or standby mode. The active deployment manager is currently hosting the administrative function for the cell. All the other listed deployment managers are backups for the active deployment manager.

Status

Indicates if the deployment manager is running.
