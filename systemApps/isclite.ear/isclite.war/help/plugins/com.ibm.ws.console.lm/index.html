Service mapping administrative console panels

You can use these administrative console panels to manage your service mapping resources.

Links marked (online) require access to the Internet. Each link starts a search for the topic in the online information center. When search results are listed, select the topic that corresponds best to your configuration.

Check for updates to this topic (online)

  • Local mapping services collection
    A local mapping service defines the interception point between a service client and a service provider.
  • Local mapping service settings
    A local mapping service defines the interception point between a service client and a service provider. Use this panel to set properties for a specific local mapping service.
  • Local mapping services - Attach
    Use this panel to attach a service map to a local mapping service.
  • Service maps collection
    A service map defines how a request sent from a service client to a service provider is transformed or rerouted.
  • Service map settings
    A service map defines how a request sent from a service client to a service provider is transformed or rerouted. Use this panel to view properties for a specific service map.
  • Service maps - Install
    Use this panel to specify the service map library that contains the service maps that you want to install.
  • Service maps - Uninstall
    Use this panel to uninstall one or more service maps.
