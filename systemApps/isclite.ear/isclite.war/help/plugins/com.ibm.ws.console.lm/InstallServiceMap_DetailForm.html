Service maps - Install

Use this panel to specify the service map library that contains the service maps that you want to install.

To view this panel in the administrative console, click Service integration > Service mapping > Service maps > Install.

This panel shows the following properties:

Path

Local file system
The location on your local file system from which you want to upload your service map library.
Remote file system
The location of a service map library on another file system or application.

Buttons

Label Action
Browse Search for a file in either your local or a remote file system.
Next Upload the service map library file and proceed to the wizard to install service maps contained within the service map library.
Cancel Cancel any changes made, then return to the previous panel.

Links marked (online) require access to the Internet. Each link starts a search for the topic in the online information center. When search results are listed, select the topic that corresponds best to your configuration.
