Local mapping services collection

A local mapping service defines the interception point between a service client and a service provider.

To view this panel in the administrative console, click Service integration > Service mapping > Local mapping services

  • To browse the properties of a local mapping service, select its name in the list.
  • To act on one or more of the local mapping services, select the check boxes next to the names of the items on which you want to act, then use the button provided.
  • To change which local mapping services are listed, or to change the level of detail that is displayed for those entries, use the Filter settings.
  • To create a new local mapping service, click New.

This panel shows the following properties:

Name
The unique name for this local mapping service.
Description
The description of this local mapping service.
Attached Service Map
The service map that is being used by the local mapping service.
Status
The status of the local mapping service: Started, represented by a green arrow (), Stopped, represented by a red cross (), or Unknown, represented by a question mark (?).
Table 1. Status and behavior of a local mapping service
Local mapping service status Attached to a service map? Yes/No Behavior
Started No The request is routed to the original target service specified by the service client.
Stopped No The service is not available. An error is returned.
Started Yes The service map is applied.
Stopped Yes The service is not available. An error is returned.
Unknown Yes The request is routed to the original target service specified by the service client.

Buttons

Label Action
New Create a local mapping service.
Delete Delete one or more local mapping services. You can delete a local mapping service only if it is not attached to a service map.
Attach Attach the selected local mapping service to a service map.
Detach Detach one or more local mapping services from a service map.
Start Start one or more local mapping services.
Stop Stop one or more local mapping services.
