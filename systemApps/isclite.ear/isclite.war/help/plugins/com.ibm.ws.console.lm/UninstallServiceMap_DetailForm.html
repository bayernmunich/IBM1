Service maps - Uninstall

Use this panel to uninstall one or more service maps.

To view this panel in the administrative console, click Service integration > Service mapping > Service maps > Uninstall

This panel shows the following properties:

Name
The name of a service map or names of the service maps that you have selected to uninstall. Before you uninstall a service map you must ensure that that service map is not attached to a local mapping service.
Note: If you uninstall a service map, that service map is deleted and is no longer available to attach to a local mapping service.

Buttons

Label Action
OK Commit your changes and return to the previous panel.
Cancel Cancel any changes made, then return to the previous panel.

Links marked (online) require access to the Internet. Each link starts a search for the topic in the online information center. When search results are listed, select the topic that corresponds best to your configuration.
