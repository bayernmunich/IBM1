Service maps - Export

Use this panel to export one or more service maps.

To view this panel in the administrative console, click Service integration > Service mapping > Service maps > Export.

This panel shows the following properties:

Name
Shows the name of the service map or service maps that you have selected to export.
Service map library name
A library is a container for the service maps that you have selected to export. Enter a name for the library you want to create.

Buttons

Label Action
Export Confirms your selection and opens a Save dialog box.
Cancel Cancel any changes made, then return to the previous panel.

Links marked (online) require access to the Internet. Each link starts a search for the topic in the online information center. When search results are listed, select the topic that corresponds best to your configuration.
