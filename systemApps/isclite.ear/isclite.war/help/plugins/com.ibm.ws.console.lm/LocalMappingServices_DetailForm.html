Local mapping service settings

A local mapping service defines the interception point between a service client and a service provider. Use this panel to set properties for a specific local mapping service.

To view this panel in the administrative console, click Service integration > Service mapping > Local mapping services > local_mapping_service_definition_name

This panel shows the following properties:

General Properties

Name
The unique name for this local mapping service.
Note: Names must be unique within a cell and cannot contain unsupported characters. The following characters are not supported:
  • /\,#$@:;"*?<>|=+&%'
  • (space)
  • the character combination ]]>
Description
The description of this local mapping service. This field is optional.

Service Properties

Endpoint address
The endpoint address of the service provider that is used by the web service request that the local mapping service intercepts.
Port type
The port type associated with the service request that is intercepted by the local mapping service. This field is optional.
Namespace
The namespace for the service and port.
Service name
The name of the web service request that is intercepted by the local mapping service.
Port name
The port name associated with the web service request that is intercepted by the local mapping service.
Note: The values for the fields on this panel must match the values of the service map to which the local mapping service is attached.

Service Maps

Attached service map
The service map that is attached to this local mapping service, shown in a drop-down menu. The menu can be used to select a different installed service map to attach to this local mapping service. The menu shows only if at least one service map is installed. If no service maps are installed, then an information message is displayed.

If a service map is attached to the local mapping service you selected, and you select a different service map from this menu from the one that was attached, then the original service map is detached when the new service map is attached.

Note: Only one service map can be attached to a local mapping service. However, a service map can be attached to multiple local mapping services.

Buttons

Label Action
Apply Commit the changes and see the results in this panel.
OK Commit the changes and return to the previous panel.
Reset Cancel any changes made. The panel reverts back to the values when you last saved the panel.
Cancel Cancel any changes made, then return to the previous panel.
