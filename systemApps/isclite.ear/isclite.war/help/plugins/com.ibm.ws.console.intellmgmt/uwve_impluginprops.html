Intelligent Management plug-in property

Use this page to configure an Intelligent Management property for your web server plug-in.

To view this administrative console page click Servers > Server types > Web servers > web_server_name > Intelligent Management > Intelligent Management plug-in properties > New.

You can also view this console page if you click Servers > Server types > Web servers > web_server_name > Intelligent Management > Intelligent Management plug-in properties > plug_in_property.

Name

Specify the Intelligent Management property name. The name is a property key used for setting the internal system configuration properties.

Value

Specify the Intelligent Management property value. The value is a string used for setting the internal system configuration properties.
