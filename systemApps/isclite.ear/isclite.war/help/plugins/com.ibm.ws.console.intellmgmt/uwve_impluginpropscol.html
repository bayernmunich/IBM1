Intelligent Management properties collection

Use this page to configure a remote cell for the Intelligent Management web server plug-in.

To view this administrative console page click Servers > Server types > Web servers > web_server_name > Intelligent Management > Intelligent Management plug-in properties.

Intelligent Management properties operations

You can add or delete properties, or you can modify existing cell properties.

Button Resulting Action
New Launches the wizard to add a new Intelligent Management property.
Delete Deletes selected properties.
Information Definition
Name Property key to be used for internal system configuration properties.
Value String value to be used for internal system configuration properties.
