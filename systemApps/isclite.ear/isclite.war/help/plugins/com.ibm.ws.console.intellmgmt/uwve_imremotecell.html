Web server Intelligent Management remote cell

Use this page to configure a remote cell for the Intelligent Management web server plug-in.

To view this administrative console page click Servers > Server types > Web servers > web_server_name > Intelligent Management > New.

Enable

Select the check box to enable Intelligent Management capabilities for the remote cell.

Host

Specifies the host name of the remote cell.

Port

Specifies the SOAP port of the remote cell.

Cell identifier

Specifies a unique remote cell identifier.

User ID

Specifies the administrator user ID for the remote cell.

Password

Specifies the administrator user password for the remote cell.

Import certificates

Select the check box to import remote cell certificates. This check box is selected by default.
