Intelligent Management for web server plug-ins

Use this page to enable and configure Intelligent Management for Apache or IBM HTTP web servers.

To view this administrative console page click Servers > Server types > Web servers > web_server_name > Intelligent Management.

Enable

Select this check box to enable Intelligent Management capabilities for your web server.

Cell identifier

Specifies the unique cell identifier for the local cell.

Intelligent Management service

Specifies Intelligent Management service connection parameters.

Information Definition
Retry interval Specifies the time interval (in seconds) between connection attempts.
Maximum retries Specifies the maximum number of retries for enabling the Intelligent Management service.

Remote cells

You can add or delete remote cells, or you can edit or refresh remote cell properties. Some actions have options.

Button Resulting Action
Add Launches the wizard to add a remote cell.
Delete Deletes selected remote cells. Optionally, select the Delete certificates checkbox below to also remove certificates.
Edit Launches the wizard to edit the remote cell properties in the Intelligent Management section of the plugin-cfg.xml file.
Refresh Refreshes the remote cell properties for the selected cell. If administrative security is enabled, provide user name and password in the User ID (Refresh) and Password (Refresh) field.
Information Definition
Host Specifies the host name of the remote cell.
Cell ID Specifies a unique cell identifier.
Enabled Indicates if Intelligent Management capabilities are enabled for the specified remote cell.
Connectors Specifies the connectors for the remote cell.

Import Certificates (Import)

Select the check box to import remote cell certificates. This check box is selected by default.

Note: To complete certificate exchange to enable the Intelligent Management plug-in to securely connect to a remote cell, it might be necessary to import the certificate of the cell into the remote cells keystore. You can use the AdminTask.retrieveSignerFromPort script command or the administrative console by clicking SSL certificate and key management > Key stores and certificates > CellDefaultTrustStore > Signer certificates > Retrieve from port.

Delete certificates (Delete)

Select the check box to delete certificates for cells to be deleted.

User ID (Refresh)

Enter the user ID for the cell selected for refresh.

Password (Refresh)

Enter the password for the cell selected for refresh.
