Printing in the help system

IBM® Eclipse Help System provides you with different ways of printing the help contents within the system.

About this task

You can either print the topic that you are currently viewing in the Content view or print one or more topics in the navigation tree quickly.

  • Printing the content view
    This topic illustrates how to print the topic that you are currently viewing in the help system.
  • Performing a quick print
    This topic shows how to print one or multiple topics in the help system.
