Printing the content view

This topic illustrates how to print the topic that you are currently viewing in the help system.

About this task

You can print the content view that you are currently viewing by performing the following steps:

Procedure

  1. Click Print () on the toolbar of the content view. The Print window opens.
  2. When you are ready to send the printing job to the printer, click Print in the Print window to start printing.
