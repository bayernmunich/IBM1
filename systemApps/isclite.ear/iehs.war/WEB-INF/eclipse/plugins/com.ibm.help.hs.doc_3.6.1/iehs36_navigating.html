Navigating in the help system

Use the graphical user interface provided by the help system to locate and view product documentation.

The Contents tab () at the bottom of the view displays the titles of the topics in the help system. The titles are displayed in a table of contents or navigation tree structure. You can select the topic title in the navigation tree to display the topic content.

Click Go Back () and Go Forward () to navigate within the history of viewed topics.

Click Home () to open the help system home page.

Click the Contents tab () at the bottom of the view to return to the Contents view after a search task is performed.

When you click a link within the topic that you are currently viewing, a new topic opens, the navigation tree will automatically changes to reflect the location of the new topic by default settings.

If the change does not automatically take place, you can do one of the following steps.
  • Locate the current topic in the navigation tree, click Show in Table of Contents () on the toolbar of the Contents view.
  • Keep the navigation tree and the topic pane synchronized at all times, click Link with Contents () on the toolbar of the Contents view. Click the button again to disable this function.

For information about navigating by keyboard, see Keyboard input and navigation.
