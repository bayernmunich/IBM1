Använda hjälpsystemet

Om du vill bläddra bland avsnitten använder du rutan Innehåll () till vänster. Om du vill visa ett avsnitt klickar du på det. Om du vill förflytta dig i historiken med visade avsnitt använder du knapparna Föregående och Nästa.

Söka

Om du vill snabbt vill leta rätt på avsnitt om ett visst ämne i dokumentationen anger du ett sökvillkor i fältet Sök. Om du vill visa sökvyn använder du rutan Sökresultat (). Om du vill begränsa omfånget för sökningen markerar du endast de avsnitt du är intresserad av.

Synkronisera

När du klickar på Visa i innehållsförteckningen () markeras det avsnittet i navigeringsträdet. Du kan använda knappen Länka till innehåll () till att synkronisera navigeringsträdet med det aktuella avsnittet.

Funktioner

Om du även vill visa dokumentationen för funktioner som är avaktiverade i programmet klickar du på Visa alla avsnitt (). Om du gör det påverkas både innehållsförteckningen och sökresultat.

Nyckelordsindex

Om du vill visa ett sorterat nyckelordsindex visar du rutan Index (). Det finns inte index för alla produkter.
