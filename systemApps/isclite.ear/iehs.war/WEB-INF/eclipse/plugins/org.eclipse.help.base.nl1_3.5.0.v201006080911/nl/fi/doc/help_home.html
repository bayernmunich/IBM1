Ohjetoiminnon käyttö

Selaa vasemmalla olevassa Sisältö-kehyksessä () näkyviä aiheita. Tuo aihe esiin napsauttamalla sitä. Voit selata aiemmin tarkasteltuja aiheita Taaksepäin- ja Eteenpäin-painikkeilla.

Haku

Kun haluat etsiä dokumentaatiosta johonkin tiettyyn aihealueeseen kuuluvia aiheita, määritä kysely Hae-kenttään. Haku-kehyksen () avulla saat esiin hakunäkymän. Voit rajata hakualuetta valitsemalla vain sinua kiinnostavat osat.

Synkronointi

Kun napsautat Näytä sisällysluettelossa -painiketta (), kyseinen aihe tulee valituksi rakenne-esityksessä. Voit synkronoida rakenne-esityksen ja valitun aiheen napsauttamalla Linkitä sisältöön -painiketta ().

Toiminnot

Kun haluat saada esiin sovelluksessa käytöstä poistettuja toimintoja koskevan dokumentaation, valitse Näytä kaikki aiheet -painike (). Tämä vaikuttaa sisällysluetteloon ja hakutuloksiin.

Avainsanahakemisto

Voit tuoda näkyviin avainsanojen aakkosjärjestykseen lajitellun hakemiston valitsemalla Indeksikehys-painikkeen (). Kaikilla tuotteilla ei ole hakemistoa.
