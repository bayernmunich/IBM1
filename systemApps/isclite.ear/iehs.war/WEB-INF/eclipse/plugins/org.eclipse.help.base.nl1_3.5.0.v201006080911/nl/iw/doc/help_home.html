שימוש במערכת העזרה

עיינו בנושאים במסגרת תוכן () משמאל. לחצו על נושא כדי להציג אותו. השתמשו בלחצנים אחורה וקדימה כדי לנווט בהיסטוריה של הנושאים המוצגים.

חיפוש

כדי לאתר במהירות נושאים בעניין מיוחד בתיעוד, ציינו שאילתה בשדה חיפוש. השתמשו במסגרת חיפוש () להצגת תצוגת החיפוש. ניתן לצמצם את היקף החיפוש באמצעות בחירה רק במקטעים המעניינים אתכם.

סינכרון

יכולות

כדי להציג תיעוד לגבי יכולות המושבתות ביישום, בחרו בלחצן הצגת כל הנושאים () . פעולה זו משפיעה על תוכן העניינים ועל תוצאות החיפוש.
