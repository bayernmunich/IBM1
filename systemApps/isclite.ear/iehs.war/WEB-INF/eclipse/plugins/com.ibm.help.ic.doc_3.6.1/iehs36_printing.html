Printing topics in the information center

You can print the topic that you are currently viewing or print one or more topics from the Contents view.

Procedure

  • To print the currently displayed topic:
    1. Click Print Page () on the toolbar. The Print window for the browser opens.
    2. To start printing, click Print or OK in the Print window.
  • To print a topic or set of topics from the Contents view:
    1. Select the topic in the navigation.
    2. Click Print topics () on the toolbar.
      • To print only the selected topic, select Print selected topic.
      • To print the selected topic and all of its subtopics, select Print selected topic and all subtopics, and then you have to choose whether to have the page break between each topic in the print output or not.
      The Print window for the browser opens.
    3. To start printing, click Print or OK in the Print window.
