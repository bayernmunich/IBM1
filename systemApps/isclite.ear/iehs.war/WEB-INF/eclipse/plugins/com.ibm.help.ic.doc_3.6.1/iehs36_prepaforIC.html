Preparing to use information center

Performing the following steps to get the best results from using the information center.

  • Make sure your browser is supported.

    To use this information center, you must have a supported web browser installed on your system. You can find the supported browsers for the information center at Supported browsers for the information center

  • Set the web browser to your preferred language.

    You can set the language option of your web browser to display information center content in your preferred language. See Displaying information in your preferred language for more information.
