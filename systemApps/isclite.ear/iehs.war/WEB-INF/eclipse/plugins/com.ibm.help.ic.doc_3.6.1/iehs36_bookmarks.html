Setting bookmarks in the information center

You can find a topic more easily by setting a bookmark. Set bookmarks by using the features in a web browser or using the Bookmarks view.

Parent topic: Viewing information in the information center

Setting bookmarks by using a web browser

For an information center deployed on a server and viewed over an intranet or the Internet, you can save bookmarks for topics to the Bookmarks or Favorites menu of your web browser.

Procedure

To set a bookmark for the current topic in your web browser:
  • If you are using Internet Explorer, click Bookmark () to save the topic in your Favorites folder.
  • If you are not using Internet Explorer, right-click any blank area on the current page to open the menu. Choose the option Add to favorites or Add to bookmarks to save the topic in one of your Favorites folders.
To view your bookmarks, open the Favorites menu of your web browser.

To delete a bookmark, open the Favorites menu of your web browser to find the bookmark. Right-click and click Delete in the menu.
